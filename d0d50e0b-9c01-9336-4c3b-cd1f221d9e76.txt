　　武斗大会，除非是在上一届中，已经是拿到了前十的名次，已经是进入过归墟天的武者，不许参加。
<br/>　　剩下的武者，都足以参加，所以许多武者前来，也都不是为了冲击前十而来的，而是在这武斗大会上，磨练实力。
<br/>　　武斗大会上的每一场战斗，都是真正的生死之战，能够参加一次，对自身实力也有着极大的锻炼。
<br/>　　像是这些北邙学院的核心弟子，一直以来参加这武斗大会，没有争取过前十的名次，这一次也依然来参加。
<br/>　　除了这些北邙学院的核心弟子，楚风眠也注意到了，就在另一片区域之中，也有着一群武者，聚集在一起。
<br/>　　这些人，皆是一身血红衣衫，一身魔意，楚风眠便是一眼认得出来。
<br/>　　这些武者，都是来自地魔门的弟子。
<br/>　　作为在北域之中，唯一足以跟北邙学院正面抗衡的势力，地魔门的实力，也极为强大。
<br/>　　甚至是因为地魔门中的争斗，比起北邙学院，都还要残酷的多，所以弟子的实力，甚至是还要再强上一份。
<br/>　　上一届的最大黑马，毒手魔君，便是地魔门的弟子。
<br/>　　这些地魔门弟子之中，其中的一尊最强者，便是引起了楚风眠的注意。
<br/>　　这是一位看起来极为年轻的少年，只是他的脸色，都是一直极为冰冷，让人不敢靠近。
<br/>　　虽然看似年轻，但是这少年身上蕴含的灵力，可绝对不弱，楚风眠甚至感觉到，此人的实力，都不在那司空家老祖，司空景胜之下。
<br/>　　这种实力，放在北邙学院，都是绝顶的高手。
<br/>　　如今没想到这地魔门中，居然还有这样一尊人物，尤其是此人的境界，还是神力境极限的境界，还没有度过生死大劫。
<br/>　　度过生死大劫之后，此人的实力，将会变得更加可怕许多。
<br/>　　“这是地魔门如今弟子一代的第一人，冷面魔君，在北域百绝榜上，排名第二位。”
<br/>　　寒月漓看着那年轻少年，开口道。
<br/>　　“此人就是冷面魔君？”
<br/>　　楚风眠听到寒月漓的话后，目光也是看向了这年轻男子。
<br/>　　冷面魔君这个名字，楚风眠还是听说过的，在北域之中，可以算是一尊叱咤风云的人物了。
<br/>　　曾经北域的武道家族之一的苏家，便是被这冷面魔君一人毁灭。
<br/>　　苏家可是足以跟司空世家，公羊世家等武道世家平起平坐的家族，就被这冷面魔君一人给毁灭。
<br/>　　就连苏家的老祖，实力达到生死境巅峰的人物，都被他所轰杀。
<br/>　　这一战，使得冷面魔君一战成名，令他在北域百绝榜上，都是排到了第二位。
<br/>　　“你们看，那就是冷面魔君！”
<br/>　　“果然如同传闻中的一样，冷面无情，此人的实力，简直称得上是妖孽。”
<br/>　　“这一届武斗大会，他可是最有机会争夺头名的人物之一，这一次的前十名，他几乎就是锁定了一个位置。”
<br/>　　一尊尊武者，也都是看到了这冷面魔君的存在，几乎每一名参加这次武斗大会的武者，看到冷面魔君，都是如同大敌。
<br/>　　毕竟在武斗大会上，谁若是运气不好，碰到冷面魔君的话，几乎便是已经必输无疑了。
<br/>　　除了地魔门一方的武者，楚风眠也看到了不少聚集的人群，有的是一些武道家族的弟子。
<br/>　　像是沧澜世家，风家等等的弟子。
<br/>　　也有一些二流势力中的弟子。
<br/>　　这些二流势力，虽然比不上北邙学院跟地魔门这两大庞然大物的势力，但是拼劲一宗之力，还是足以培养出几尊强者的。
<br/>　　在北域百绝榜上，大部分的武者，其实都并非是来自于这两大势力，来自其他势力宗门也不少。
<br/>　　“夜天君，居然没有来？”
<br/>　　楚风眠扫视了周围一圈，微微有些诧异的说道。
<br/>　　夜天君，楚风眠的确还没有见到过，不过夜天君，可是先天神体之一的混乱神体。
<br/>　　一身的混乱之力，一眼就可以认得出来。
<br/>　　楚风眠扫视了周围一圈，几乎便是没有碰到这种混乱之力，甚至就连一个圣党弟子，都没有看见。
<br/>　　这也让楚风眠有些奇怪的。
<br/>　　圣党可是北邙学院之中最为庞大的一方势力了，其中更是强者如云，这武斗大会，圣党弟子没有理由不来参加才对。
<br/>　　“夜天君，一定会来的。”
<br/>　　寒月漓在一旁，平静的开口道。
<br/>　　夜天君现在的境界，已经是在神力境极限，停留了很久的位置。
<br/>　　并非是夜天君不能度过生死大劫，以他现在的实力，引来生死大劫，都是轻而易举的一件事。
<br/>　　就跟楚风眠一样，现在的楚风眠想要引来生死大劫，几乎瞬息之间都可以做到。
<br/>　　只是楚风眠是在积累，他要在得到奇遇，积累足够的底蕴，在准备度过生死大劫。
<br/>　　夜天君的想法，便是跟楚风眠一样。
<br/>　　这归墟天，正是拥有这无数奇遇的地方，这一次，夜天君没有理由不来。
<br/>　　“夜天君的实力，排在北域百绝榜什么位置？”
<br/>　　楚风眠出声询问道。
<br/>　　对着北域百绝榜，楚风眠倒是没有多少了解，夜天君在北邙圣域之中，是绝对的第一人。
<br/>　　只是放在这整个北域上，他到底是什么位置，楚风眠还不是很清楚。
<br/>　　“北域百绝榜，夜天君排在第三位。”
<br/>　　寒月漓开口道。
<br/>　　“不过这还是他得到了奇遇之前的排名，如今夜天君得到了如意君的洞府传承，他的实力，只怕又要更强。”
<br/>　　第三名……
<br/>　　这个名次，让楚风眠微微有些出乎意料，本来看到这冷面魔君排到第一位的时刻。
<br/>　　楚风眠本以为，夜天君应该便是北域百绝榜的头名位置。
<br/>　　没想到这头名，居然另有人选。
<br/>　　“轰隆！”
<br/>　　就在这交谈之际，远方，就在这武王城南门前，一道巨大无比的裂缝，突然显现出来。
<br/>　　只见这周围的空间，都被撕碎，从中，一连走出了无数人影。
<br/>　　这些人，每个人的身上，都蕴含有着一种极为精妙的混乱之力。
<br/>　　“圣党的人！”
<br/>　　“圣党的人来了！”
<br/>　　无数人的眼神，也都是随之看了过去。
<br/>　　这一群人，正是圣党的人，一眼望去，足足是有着数百人。
