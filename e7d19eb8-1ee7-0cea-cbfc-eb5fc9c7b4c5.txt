　　一下子爆发的视觉冲击不断刺激众人的眼球，不仅狼牙防守队的众人震惊当下，在场的所有死士也惊恐无比，尤其白家派来的那些精英死士，全体被惊吓到了失去思考能力。
<br/>　　这名被瞬间秒杀的死士，是他们之中的最强者！
<br/>　　众人的目光不约而同的再次望向那处巨大深坑，只见白允儿的身躯微侧，站位有了一些变化，显然在事发的一瞬间就反应过来。
<br/>　　而站在深坑内的那个神秘人，此时正微微偏头看着后方，露出的半边侧脸让很多人愣住，没有太突出的五官，除了与他魁梧身躯相配的凶煞长相外，便是面孔上的皮肤有些偏灰暗。
<br/>　　在那名精英死士身躯炸开之后，他便会转过头，仿佛刚才的那一瞥只是对身后死去之人的蔑视。
<br/>　　看到这样的反应和这样的动作，在场谁还不明白出手的人是谁？
<br/>　　一颗微小的石子而已，就能杀人于千米之外！
<br/>　　甚至很多人，都根本没注意到凶器是一颗石子，也压根看不清那一瞬间秒杀的移动轨迹。
<br/>　　一下子，所有人都心头恐惧滋长，过分的强大让他们不敢有任何动作，生怕下一秒就会身亡，在绝对的碾压战力面前，谁都没有主动权。
<br/>　　场面就这么寂静无声，在一开始就不顾一切拼命靠近深坑的尚九娣，则一下子发现了一丝不同寻常，她的心脏狂跳不止，冷汗大颗大颗的滴下，脑中更是一下子想到了无数方面的问题。
<br/>　　因为距离近的她，看到了这个神秘人的双眼。
<br/>　　那是一片没有眼白的极致黑色！
<br/>　　丧尸瞳孔压根没有让整个眼球都一片森白，异种的瞳孔则是腥红，一眼便能看出区分。
<br/>　　可是这样全部黑色的眼球，是什么生物？
<br/>　　尚九娣不知道该用什么来形容这个人，但她的第一反应就是，他不是人类！
<br/>　　同样发现这一细节的人，还有距离尚九娣不远的顾凉晨，他的第一反应便是阴阳谷内的某门家族成员，毕竟在当下的末世纪元除了已知的四大神秘家族，和那个中间人的制裁者，便只有阴阳谷内的那另外三门如此特别。
<br/>　　只是忽的，顾凉晨又冒出另一想法，莫不是来自另外星球？
<br/>　　这时候在深坑处。
<br/>　　似乎是思考了一会儿白允儿刚才的话，这个黑色眼球的神秘人又一次开口，这次直接无比：“我要木屋里的人，要活的。”
<br/>　　白允儿的双眼一眯，眼底一抹绚丽的蓝紫色一闪而过，只说了两个字：“放肆。”
<br/>　　哗！
<br/>　　在场的所有人顿时大惊失色，先是被白允儿的这两个字吓到冷汗直冒，再是被白允儿这种强硬和高冷的态度所折服。
<br/>　　换成他们，或许能说出拒绝的话，但绝对说不出如此高姿态的‘放肆’二字。
<br/>　　远处的白家死士们一个个面色苍白如纸，前去拦住白允儿的下场就会和那名死的连尸骨都不留的人死士一样，可若是白允儿继续这么强硬下去，万一有个三长两短，他们的下场也依旧是死。
<br/>　　两难的抉择，顿时让这些死士们踌躇不前。
<br/>　　这时候在最后方的那片树林中，一名罗家死士缓慢的朝着后方靠近，用了九牛二虎之力将移动隐匿到极致，来到了一名埋伏在这里已经半小时之久的白家死士身旁。
<br/>　　白家死士没有任何动作，除了紧张之外也没有其他情绪，这时候罗家的死士再如何，他也管不着。
<br/>　　“怎么说？”罗家的死士率先开口，声音细小如蚊子叫，显然小心谨慎到了极限。
<br/>　　白家的那名死士皱了皱眉，同样小声的回复：“少添乱，任务我们放弃。”
<br/>　　罗家的那名死士则咬了咬牙，道：“那个家伙的目标也是楚涵，单靠我们一队人可抢不到，要不要联手？”
<br/>　　白家的死士一愣，扭头露出惊讶的目光。
<br/>　　“很简单的道理。”罗家的死士解释道：“你们同样敌不过这个人，或许白允儿有能力抵挡，但她受了伤，应当没有能力倾尽一战，说不定还会战败然后死去，她死了你们恐怕也都活不成吧？”
<br/>　　“但如果我们联手，由白允儿拖住此人，我们一边稳住这里朝此人的后背攻击，一边派出另一队人绕后，如何？”
<br/>　　白家的死士瞬间明白过来，当即发问：“谁绕后？”
<br/>　　“当然是我们罗家的队伍。”罗家死士大言不惭，也直截了当：“你们好好想想，是任务失败重要，还是白允儿的命重要？”
<br/>　　白家死士顿时哑口无言，只能点头与罗家的死士达成协议。
<br/>　　在这种情况下，他们只能先以白允儿的安危为先，而罗家去杀楚涵，那名高战力者也一定会发现，目标也将改变，毕竟他可是说过，要活的。
<br/>　　到时候他们白家的死士就立即带着白允儿撤退，罗家那队人会如何他们可管不了。
<br/>　　楚涵的命，他们不要了！
<br/>　　刽子手交给罗家人来当！
<br/>　　达成协议之后，罗家的一部分死士便一点点的向后移动，一副要撤离放弃的样子，样子装的很足。
<br/>　　白家的死士则缓慢朝前靠近，不断紧张的盯着白允儿的动作。
<br/>　　这两队的行动不仅那名高战力者有所察觉，不远处的狼牙防守队成员也一清二楚，只是这时候谁也不能去深想，他们的心思都放在了那间被白允儿拦下的木屋里。
<br/>　　“罗家的队伍要走，白家的队伍想救你，当然这两批人马无论哪一方在我看来都不堪一击。”这时候那名神秘的黑瞳者淡淡开口，表情没有因为白允儿的话有丝毫变化。
<br/>　　说道这里他眼神带着一丝疑惑，仿佛真诚无比的在请教：“其实我刚才没有想通，你能以什么身份，对身为丧尸王的我说‘放肆’二字呢？”
<br/>　　此话一出。
<br/>　　轰！
<br/>　　在场所有人集体炸开的情绪，好似暴风卷席！
<br/>　　丧尸王！
<br/>　　这个黑瞳者，不是人类，也不是什么神秘种族，更不是顾凉晨所想的来自哪个外星。
<br/>　　而是那个他们一直知道会出现，并且从头至尾都忌惮防备的丧尸王！
<br/>　　竟然就这么出现，竟然就来到了狼牙，并且直言要楚涵，还要活的！
