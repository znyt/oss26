　　第259章
<br/>　　长草野道，往前行视线中出现一座小镇。
<br/>　　一座破落的小镇，算是个小型市场，草原上的人把兽皮，羊羔，马奶带到这里，交换盐巴，麻布等生活必需品。
<br/>　　在这个既不属于戎族，也不被帝国控制的地带，这座小镇也是马贼，杀人犯，走私犯的乐土。
<br/>　　这里有赌场，有妓院，这里既可以买到来自帝国的好酒，也能买到粗长辫子的草原姑娘。
<br/>　　一辆马车来到这里，在镇中心的水井前停下，男人从马车上跃下来，去井中取水，过程中他停下来，扫视着周围。
<br/>　　诡异来自于安静，周围太安静些，像是被生化危机席卷过的城市，人都死了，城市变做坟墓。
<br/>　　“嘿，赌两把，赌两把。”
<br/>　　突然有个人从屋顶跳出来，凑到程大雷面前，手中是一颗足球大小的白铁骰子。
<br/>　　程大雷怔了怔，这个人的速度好快，快到他来不及握剑。
<br/>　　“不赌行么？”程大雷笑笑问。
<br/>　　“不好的，不好的。”这人笑着：“你不赌我就得直接杀死你，多没意思。”
<br/>　　“喔，那看来我是不得不赌了。”程大雷道：“赌注是什么？”
<br/>　　“你的命。”
<br/>　　程大雷表情没有变：“如果我赢了，就可以走了？”
<br/>　　“那也走不了。”
<br/>　　“哦……”
<br/>　　“你赢了，我就不杀你了，但是有个人要杀你。”
<br/>　　随着他的声音，一人从路旁的酒馆中走出，他身材很瘦，表情还有些怯懦，就像那种谁都可以踹两脚的乖学生，但他手中拎着一柄很大的杀猪刀。
<br/>　　鱼龙卫，赌鬼和屠夫。
<br/>　　程大雷看着二人，肩上的伤已经令他无一战之力，而眼前二人的实力却都不弱于他。
<br/>　　他看着赌鬼，道：“你说话算数么？”
<br/>　　“愿赌服输，绝不赖账。”赌鬼昂着脑袋。
<br/>　　“我直接杀了他不久完了么？”屠夫拎着杀猪刀：“用不用这么费力。”
<br/>　　“你先别动手，我要和他赌一把。”赌鬼道。
<br/>　　“那就赌一把。”程大雷耸耸肩：“怎么赌？”
<br/>　　“赌大小，一局定胜负，你若输了，我就要你的命。”
<br/>　　“赌大小太没意思，我有个好玩的赌法，不知道你感不感兴趣？”
<br/>　　“有趣的赌法，怎么玩，快说，快说。”赌鬼迫不及待道。
<br/>　　“大富翁听过没？”程大雷一指小镇入口的槐树：“从这里到槐树有一百步，每人摇骰子，摇到几点就走几步，谁先摸着树谁赢。”
<br/>　　“好好，太有趣了，我先来。”
<br/>　　赌鬼举着手中的硕大骰子，突然丢到半空中，咚地一声落在地上，显出一个六点。
<br/>　　“哈哈，小子，你输了。”
<br/>　　赌鬼向前迈步，一步跨出就是普通人四五步的距离，看这情况，若再有一两次，他便能到大槐树下。
<br/>　　轮到程大雷掷骰子，他抱起地上的白铁骰子，入手颇为沉重，至少有五六十斤，刚才赌鬼可以将骰子丢到四五米高度，可见其实力的恐怖。
<br/>　　程大雷将骰子丢向赌鬼，骨碌碌滚到他脚下时，显出一个两点。
<br/>　　程大雷走到马车前。
<br/>　　赌鬼哈哈大笑，道：“小子，你虽然力气不错，但你还是要输了。”
<br/>　　他把骰子丢到程大雷脚边，又滚出一个六点。向前跨出六步，已经更接近槐树，程大雷似乎已经没有翻盘的希望。
<br/>　　程大雷没有丢骰子，而是操起马车上的黑伞，面向屠夫。
<br/>　　“认得这柄伞么？”
<br/>　　“这是行者的伞。”屠夫道：“我是行者的朋友。”
<br/>　　说到这里，程大雷发现屠夫脸上的表情忽然悲怆起来，他仰起头来，望着头顶的天空。
<br/>　　“在漫长的生命中，只有他是我的朋友，失去他之后，我只能一人跋涉这纷乱的世间，你不了解我的悲痛，或许我只有仰望天空，才能不让自己眼泪掉下来。”
<br/>　　程大雷愣住了，摸摸脑袋：莫非这屠夫真名姓郭？
<br/>　　“既然是故人遗物，那我就还给你吧。”程大雷握着伞柄将伞递还给屠夫。
<br/>　　“多谢，但我不会因此而不杀你。”
<br/>　　屠夫将刀交到左手，向前迈步。
<br/>　　蓬！
<br/>　　一声脆响，伞顶的钢针射出，屠夫竟丝毫没有防备，毒针准确的射在他脸上。他一张苍白的脸立刻变成黑色，临死前眼中除了惊恐，还有难以置信。
<br/>　　“很显然……”程大雷撇撇嘴：“行者没有把你当朋友。”
<br/>　　屠夫是行者的朋友，行者不是屠夫的朋友，当他以四十五度角仰望天空时，并不知道他朋友一直提防着他，就连伞中的毒针也没有告诉他。
<br/>　　赌鬼看到这一幕发愣，他当场就想发作，却已然来不及了。
<br/>　　“我想知道赌鬼是否真的愿赌服输，从不赖账？”
<br/>　　赌鬼脸上神色变幻，忽然咬着牙道：“我先胜了你，然后再杀了你。”
<br/>　　“那么该我掷骰子了。”
<br/>　　程大雷这才捡起地上的骰子，骨碌碌掷出一个五点。
<br/>　　只见他跃起，身体凌空飞起，一步跃出，便是三四丈的距离，当他落地之时，已经超过赌鬼。
<br/>　　赌鬼脸上阴晴不定，程大雷杀掉屠夫出乎他预料，而程大雷能超过自己，同样在自己意料之外。
<br/>　　“小子，你不要太得意，我还有一次掷骰子的机会，你必输无疑。”
<br/>　　“喔，不，我已经赢了。”程大雷道。
<br/>　　“赢……你那里赢了，你明明还没有摸到大树。”
<br/>　　“脚下，注意脚下。”程大雷跺了跺脚。
<br/>　　赌鬼注意到，他脚下是一截树根。
<br/>　　“树根也是大树的一部分么。”程大雷拔出了剑：“如果你愿意愿赌服输，我放你离开，如果你不愿意，我送你离开。”
<br/>　　赌鬼盯着程大雷，沉默良久，忽然一跺脚：“老子愿赌服输，你打听打听去，我什么时候赖过账，你休想诓我破规矩。”
<br/>　　说着，他长身而起，抱起地上的骰子，将屠夫的尸体扛在肩上，大步离去，几个转折声音便已经消失不见。
<br/>　　程大雷心松了下来，几步走到马车前，李婉儿拉他上车，入手发现程大雷的手掌十分滚烫，他几乎是爬到马车上。
<br/>　　“你……”
<br/>　　程大雷将脑袋搁在李婉儿肩上，声音越来越微弱。
<br/>　　“好像有点感冒。”
<br/>(本章完)
