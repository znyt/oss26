北城的游乐园，不如滨江城那边的热闹，特别是现在这种寒冷的入冬季节，游乐园里，来游玩的人寥寥无几。
<br/>
<br/>        傅晏琛带着连星儿在入口处买票的时候，女售票员见傅晏琛穿着军装，好心问道：“先生，您有没有带军人证，带了的话，可以拿出来，军人有优惠。”
<br/>
<br/>        “谢谢，我按原价购票即可。”傅晏琛礼貌性地颔了颔首，温文儒雅地对售票员微微一笑。
<br/>
<br/>        此刻，连星儿的两只眼睛还是红红的、微微浮肿，哭过的样子，楚楚可怜，却又不失可爱。
<br/>
<br/>        女售票员替他俩出了两张票后，无意间看到连星儿时，又忍不住多嘴了一句：“你女朋友长得这么可爱，咋就忍心惹她哭呢？”
<br/>
<br/>        “她是我妹妹。”傅晏琛接过票后，尴尬地解释道。
<br/>
<br/>        连星儿突然“戏精上身”，一边拍打着傅晏琛的手臂，一边装作很委屈地娇嗔道：“我明明就是你的女朋友，你非得在外人面前说我是你妹妹！呜呜——你太坏了！”
<br/>
<br/>        女售票员听后，忍俊不禁起来。
<br/>
<br/>        “……”傅晏琛扯了扯嘴角，赧然地对女售票员笑了笑，然后赶紧抓着连星儿的手腕，拉着她进了游乐园。
<br/>
<br/>        他们前脚刚进，后脚就有个戴着口罩和鸭舌帽的男生走过来买票。
<br/>
<br/>        就在明司寒准备掏钱的时候，突然一只手伸过来，抓住他的手腕，未等他反应过来，他人就已经被一道力度拉到了一旁。
<br/>
<br/>        明司寒打了个趔趄，站稳身子，准备还手制服对方的时候，对方敏捷地躲开了。
<br/>
<br/>        下一个瞬间，对方迅速出手，防不胜防地摘掉了他脸上的口罩。
<br/>
<br/>        “我就知道是你。”郎安手里拿着明司寒的口罩，不冷不热地说道。
<br/>
<br/>        明司寒伸手夺回自己的口罩，一脸不悦地皱起了眉头。
<br/>
<br/>        “你一路跟踪我和大小姐，到底有何目的？”郎安质问道。
<br/>
<br/>        明司寒舒展眉心，似笑非笑地回答道：“只是顺路罢了！”
<br/>
<br/>        “从我和大小姐离开比赛大厦起，你就一路跟到了军区，又一路跟到了这里。你说顺路，谁信？”郎安不以为然道。
<br/>
<br/>        明司寒不再解释，而是自顾自地望向游乐园内，待会他就要走了，只是想在走之前，再多看看星儿罢了。
<br/>
<br/>        郎安见明司寒不吭声，没再动手攻击明司寒，只是淡淡地接着问道：“我不明白。你看起来似乎很在意大小姐，又为什么要离开大小姐？”
<br/>
<br/>        “你想多了，我一点都不在意她。”明司寒收回目光，转过身去，准备离开时，又补充了一句，“你不要每天板着一副脸，她不喜欢。”
<br/>
<br/>        明司寒说完便直接走了。
<br/>
<br/>        郎安却不懂他最后那句话的意思。
<br/>
<br/>        游乐园里，连星儿拉着傅晏琛可谓是玩疯了。
<br/>
<br/>        哪个刺激，她就专挑哪个玩。
<br/>
<br/>        每个刺激的游乐设施项目都玩遍后，连星儿又盯上了那座大大的摩天轮。
<br/>
<br/>        正当连星儿想着要去坐的时候，却被傅晏琛制止了。
<br/>
<br/>        “我听说，那东西，只能情侣上去坐。”傅晏琛突然一本正经地说道。
<br/>
<br/>        连星儿眯起眼睛，一脸狐疑地打量着傅晏琛：“哥哥你真矫情！小时候，咱俩不知道坐过多少次了。”
<br/>
<br/>        “小时候是小时候，现在我两长大了。”傅晏琛汗颜着说。
<br/>
<br/>        连星儿嘴角微扬：“噢！我知道了，哥哥是想和宋小姐一起坐，对吧？那行，我给祖爷爷打个电话，让祖爷爷把宋小姐叫出来。”
<br/>
<br/>        “……”顿时，傅晏琛黑脸了，不冷不热地说道，“你能不能别插手？”
<br/>
<br/>        “哥哥，你就考虑一下宋小姐呗！我听祖爷爷说……”
<br/>
<br/>        “这样吧！你什么时候嫁到如意郎君了，我就什么时候考虑交个女朋友。”傅晏琛突然笑着打断了连星儿的话。
<br/>
<br/>        “那……要是我嫁不出去呢？”连星儿说着，瘪起嘴来。
<br/>
<br/>        傅晏琛抬手戳了戳连星儿瘪下去的嘴角，意味深长地笑了笑：“你若是嫁不出去，我就终身不娶了。”
<br/>
<br/>        “为什么啊？”连星儿抬头看着傅晏琛，不解地眨巴着眼睛。
<br/>
<br/>        傅晏琛摇头感叹道：“你嫁不出去，怪可怜的。我就给你做个伴，老了后咱俩还能相互有个照应。”
<br/>
<br/>        “哼！谁说我嫁不出去了！我美着了！有的是男孩子喜欢！”连星儿傲气道。
<br/>
<br/>        傅晏琛笑道：“这才对嘛！要早点从这段伤心中走出来。”
<br/>
<br/>        “那我要尽快找到下一个喜欢的男生，然后要他娶我！”连星儿咧嘴一笑，心里其实依旧在痛，只不过是在程程嘴皮子功夫罢了。
<br/>
<br/>        傅晏琛看着连星儿，抬起手来，揉了揉她的头顶，无奈地笑了笑：“傻丫头！”
