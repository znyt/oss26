        然后，让她安安分分地待在金笼里，等着他的“投喂”与“挑逗”，却不知道他心底的任何打算。
<br/>
<br/>        说白了，她还真的就像一只被他圈养在金笼里的金丝雀。
<br/>
<br/>        “你这个形容词，还真是用得不错。”傅夜沉忍俊不禁道。
<br/>
<br/>        童筱颖见他还有心情笑得出来，忍不住抬起手，懊恼地捶打着他的胸膛。
<br/>
<br/>        可是，她打着打着，又捂着脸，哭了起来。
<br/>
<br/>        他怎么可以这样让她又爱又恨？
<br/>
<br/>        这几天，她一直压抑着自己的悲痛，现在终于可以全部发泄出来了。
<br/>
<br/>        每一次，她一哭，他心口就会痛，这一痛，他的心就会软，恨不得自己能把天上的星星月亮摘下来哄她开心就好。
<br/>
<br/>        怪不得说女人是水做的，眼泪怎么擦也擦不尽。
<br/>
<br/>        傅夜沉拉开了童筱颖捂着脸的手，微微俯身，亲了亲她的唇，又去吻她颊畔的泪痕。
<br/>
<br/>        他每吻她一下，便抬头看向她那对水眸一会儿，耐着性子开始温柔地哄道：“乖，不哭了，你想要什么，我都给你。”
<br/>
<br/>        “我要你的吻！”
<br/>
<br/>        童筱颖长而卷翘的睫毛上全是晶莹的泪花，这刻却为面前这个心爱的男人，已经不顾自己任何形象了。
<br/>
<br/>        她踮起脚尖，一把抱住他的脖颈，献上自己软嫩的唇。
<br/>
<br/>        以前都是他主动攻击，这回终于算得上是风水轮流转了。
<br/>
<br/>        傅夜沉薄唇微微勾了勾，任由那细细软软的粉唇在自己唇上辗转来去。
<br/>
<br/>        他不但不闭上双眼，反而偏偏还睁大了黑眸凝视着这个小女人在自己面前着急。
<br/>
<br/>        童筱颖吻了一会儿，却没有得到傅夜沉的回应，只觉自己的心脏莫名其妙地像是被什么东西拧成了一团，这种感觉，让她闷得发慌，还很不是滋味。
<br/>
<br/>        她眉宇间微微蹙动，面前这个嘴角噙笑的男人像读懂了她所有心思般，在她弃械投降之际，突然张口将她就快要退缩的小嘴含进了嘴中。
<br/>
<br/>        明明她快要沉浸在他的回吻之中时，脑海里却突然想到了落以柔。
<br/>
<br/>        童筱颖下意识地推开了傅夜沉，推得他毫无准备地往后退了一步。
<br/>
<br/>        “落以柔为了你殉情了，你知道吗？”
<br/>
<br/>        她的话语一落，他的脸色就变了。
<br/>
<br/>        很显然，他不知道。
<br/>
<br/>        童筱颖凝视着傅夜沉的眼睛，隐隐地从他眼中读出了一丝惊愕、怜悯、感动、还有无奈……
<br/>
<br/>        “真是个蠢女人！”傅夜沉少有的心烦意躁。
<br/>
<br/>        童筱颖却听不出他这句话中的另外一层意思，只是问出了心中的疑惑：“大哥，你有没有对落以柔动过心？或者，在意过她？或者……”
<br/>
<br/>        “我曾在她的身上，看到过你的影子，仅此而已。”傅夜沉不假思索地回答。
<br/>
<br/>        这是他的实话。
<br/>
<br/>        童筱颖却将自己的目光，挪向了一旁，一脸不满地咬了咬自己的下唇。
<br/>
<br/>        傅夜沉抬起手来捏住她的下巴，迫使她仰起头来望着自己的眼睛：“筱颖，我的心里只有你，不会再对别的女人动心。你相信我。”
<br/>
<br/>        “落以柔比我更爱你，更了解你。”尽管他已经表态，但童筱颖的眉眼却仍旧有些飘忽不定，“还好，她殉情未遂，被郑忠威捡回来了一条命。”
<br/>
<br/>        “我怎么觉得，这世上最爱我的女人是你。”傅夜沉剑眉微扬。
<br/>
<br/>        童筱颖不明所以然地对上了傅夜沉的黑眸。
<br/>
<br/>        “你似乎在怀疑是那个八爷害的我，所以，才会穿成这样，来夜惑接近八爷。结果得知是四爷动的手，索性找机会替我报仇，嗯？你是不是还想着，报完仇之后就自杀？”傅夜沉慢条斯理地说。
<br/>
<br/>        心思被说穿，童筱颖瘪着嘴不吭声。
<br/>
<br/>        “你还是老老实实地当我的金丝雀比较好。”他忍俊不禁地接着说。
<br/>
<br/>        童筱颖这才义愤填膺地反驳：“你不就是想查夜惑里的其他几位爷的身份吗？我也可以帮你的！”
<br/>
<br/>        “你一不会开枪，二不会格斗术，三不会……”
<br/>
<br/>        “正因为我什么都不会，才是最好的卧底，不是吗？”童筱颖直接打断了傅夜沉的话。
<br/>
<br/>        “卧底？你从哪里学来的这词？电影里看的？”傅夜沉哑然失笑道。
<br/>
<br/>        童筱颖不满地傲气起来：“反正，我可以打听到，你们查不到的事情，就对了！”
<br/>
<br/>        “我不会让你以身犯险，所以，你现在给我回去乖乖呆着。”傅夜沉不以为然，双手扣住童筱颖的双肩，转动着她的身子，将她往门外推。
<br/>
<br/>        童筱颖下意识地攀住门框边缘的墙，努嘴道：“大哥！我不要被你关在笼子里。真的，我真的能帮到你。”
