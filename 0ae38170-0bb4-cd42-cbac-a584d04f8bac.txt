        连星儿抱着冷晏琛的脖子，微笑道：“当然知道！我就跟祖爷爷说，是那位机长小姐，看不上你。那我还要跟祖爷爷说理由吗？”
<br/>
<br/>        “不需要，祖爷爷是个精明的人，他会猜得八九不离十的。”冷晏琛会心一笑，背着连星儿慢慢地往前走。
<br/>
<br/>        连星儿不禁感慨道：“我两真的好悲催啊！一个被迫联姻，一个被迫相亲。”
<br/>
<br/>        “我听说，你那个联姻，其实是你自己选的吧！当初，明爷爷可是把他的所有未婚的孙子的资料送到你手中任你选择的。而且，我得知这个消息的时候，我还以为你会选择我的连长。”冷晏琛感叹道。
<br/>
<br/>        “哥哥为什么认为我会选择你的连长大人？”连星儿不解地问道。
<br/>
<br/>        冷晏琛笑道：“你小时候不是经常嚷着，长大以后，要嫁给爹地的吗？”
<br/>
<br/>        “可是，那时候我是觉得爹地长得帅啊！再说了，明司寒的大哥明启辉，比我大了整整十五岁，我才不会选个老男人！”连星儿噘嘴道。
<br/>
<br/>        冷晏琛顿时一脸严肃道：“哥哥我可不允许你这么说我的连长！”
<br/>
<br/>        “好吧！我错了！那哥哥，你告诉我，为什么你的连长大人，都三十三岁了，还未娶妻生子啊？”连星儿纳闷道。
<br/>
<br/>        毕竟，明启辉长得也挺不错的，论家世论样貌，不至于找不到女朋友。
<br/>
<br/>        冷晏琛忽然深沉又惆怅地回答道：“他女朋友，三年前在国外的支援战中，牺牲了。所以，他一直未娶。”
<br/>
<br/>        连星儿身心一怔，不禁沉默了。
<br/>
<br/>        冷晏琛听闻连星儿没吭声了，又接着问道：“怎么不说话了？”
<br/>
<br/>        “我以后再也不说他是老男人了！”连星儿信誓旦旦地说。
<br/>
<br/>        冷晏琛只是忍俊不禁。
<br/>
<br/>        “连长大人是打算一辈子不娶吗？”连星儿又问道。
<br/>
<br/>        冷晏琛感叹了一句：“应该是。”
<br/>
<br/>        “那当初明爷爷说要联姻的时候，他没反对吗？”
<br/>
<br/>        “他没反对，他说，你肯定看不上他。我还跟他打赌来着，结果，我输了。”冷晏琛笑道。
<br/>
<br/>        连星儿忍不住揪了一下冷晏琛的耳朵：“谁叫你自以为是地很了解我的！”
<br/>
<br/>        “我跟你说正经的！等你哥哥我有了女朋友后，我的后背、我的耳朵，都是属于我女朋友的。你可不能再乱来了！”冷晏琛一本正经地说。
<br/>
<br/>        连星儿微微一笑：“明白！”
<br/>
<br/>        就这样，冷晏琛一路背着连星儿，一直送她回到了酒店。
<br/>
<br/>        冷晏琛回部队里去了后，连星儿独自一个人在酒店的套房里，拿着手机给明司寒发信息。
<br/>
<br/>        明司寒回复她说要明天下午才会到，而且，他已经给她准备好了VIP入场券，还是观看比赛的最佳位置。
<br/>
<br/>        连星儿看着明司寒回给自己的信息，趴在大床上笑不拢嘴。
<br/>
<br/>        而此时，她的头顶，越过数层楼板，直达数千米的上空的一架民航飞机里。
<br/>
<br/>        飞机正处于巡航状态，宋云汐起身离开驾驶舱，走到茶水间给副驾驶也就是自己的师父泡咖啡。
<br/>
<br/>        这本该是空姐做的事情，但是师父说，她泡的咖啡好喝，所以她便主动来做了。
<br/>
<br/>        同机的乘务长，是个已婚又精致的女人，因为早就听说了她宋云汐在相亲，于是趁着宋云汐泡咖啡的空闲，和宋云汐八卦了起来。
<br/>
<br/>        “听说，你今天又去相亲了？怎么样？相中了吗？对方是个什么样的男人？”乘务长好奇地问道。
<br/>
<br/>        毕竟，宋云汐在他们这家民航公司里，可谓是坐头把交椅的“司花”。
<br/>
<br/>        “是个现役当兵的，相貌长得挺帅，人也很绅士，还不错。”宋云汐一边研磨着咖啡，一边云淡风轻地回答。
<br/>
<br/>        乘务长倒是一脸崇拜了起来：“原来是个兵哥哥啊！不错不错！想当初，我相亲的时候，怎么就没遇上兵哥哥呢？其实，我也好想嫁给兵哥哥。然后，大声对他说‘你保卫国家，我守护你’。你说，我这人，是不是特别豪情壮志！”
<br/>
<br/>        “有梦想不错，你可以离了婚再嫁。我想，兵哥哥不会嫌弃你二婚的。”宋云汐会心一笑，对于冷晏琛的事情，她并未过多地与自己的同僚谈及。
<br/>
<br/>        她不大喜欢把个人的私事带入到工作环境中来。
<br/>
<br/>        乘务长却啧啧嘴，摇头叹息道：“算了！其实，我家里那位人也不错。”
<br/>
<br/>        宋云汐听着，只笑不语。
<br/>
<br/>        乘务长又问道：“那你们有结果了吗？相亲成功了没？”
<br/>
<br/>        “今天交谈之后，发现性格不大合。”宋云汐拿起热水瓶，开始冲泡咖啡。
