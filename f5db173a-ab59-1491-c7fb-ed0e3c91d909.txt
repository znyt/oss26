        当连中凯准备按下制冷开关的时候，裴医生伸手制止道：“你可要想清楚了，童筱颖若是死了，这个世界上，就再也没有夏白薇的基因了。”
<br/>
<br/>        “有没有她的基因，我无所谓。我为她付出了那么多，而她，从始至终都没有爱过我。她跟龙半夏那傻女人一样，都爱着冷卫峰，为了冷卫峰，连自己的命都可以不要，真是可笑！”连中凯不以为然地说道，一双饱含沧桑的眼睛里，突然间充满了仇恨。

<br/>        连中凯挪开裴医生的手，想要再次按下制冷开关，以此来结束童筱颖的性命时，裴医生冷不丁地开口道：“你想杀了你自己的最后一个女儿，我也没意见。只是，你别后悔。”

<br/>        “你说什么？”连中凯一脸惊愕地瞪向一旁站着的这个女人。

<br/>        裴医生望向钢化玻璃墙那边的插满了各种白色管道的三副水晶棺材，不痛不痒地回答道：“当年，我从冷库里拿的是你的精子跟夏白薇的卵子结合后，才植入夏阳枭他老婆的子宫内的。”
<br/>
<br/>        “你……”连中凯顿时被裴医生的话给怔住。

<br/>        “没错，我当时是想，既然得不到白薇，那么就利用她的基因再造一个她出来对我唯命是从。这又有什么错？”连中凯愤愤不满地质问。
<br/>
<br/>        都说科学家是疯子，连中凯在爱情上也是个疯子。
<br/>
<br/>        裴医生深沉地接着解释道：“我去找白薇的时候，我还未表明自己的来意，白薇就主动跟我提出，愿意贡献卵子给我们做‘解药’研究。唯一的条件是，孩子另一半的基因必须是你连中凯的。”
<br/>
<br/>        连中凯顿时难以置信地支吾道：“你、你撒谎！”
<br/>
<br/>        夏白薇那么不待见他，怎么可能会愿意跟他结合再造个孩子出来。
<br/>
<br/>        那个时候，夏阳枭的妻子，其实在第一次生产后就患上了不育的后遗症，而她很想再要个女儿，才会拿着钱找上裴医生，希望裴医生能给她做一个健康漂亮的女儿。
<br/>
<br/>        刚巧，他们正在研究的人体‘解药’需要一个女人来作为孕体，所以，裴医生索性把‘解药’植入了夏阳枭的妻子的子宫里。
<br/>
<br/>        其实，在他们这群基因研究者的眼里，压根就没有什么人伦理论。
<br/>
<br/>        有的，只是实验结果的成功与失败。
<br/>
<br/>        裴医生见他身子都跟着在发抖，叹了口气，接着说道：“你若不信，可以取童筱颖的除了血液以外的细胞DNA跟你自己的DNA做亲子鉴定。”
<br/>
<br/>        连中凯抖动的手也在慢慢地从制冷开关上挪开。
<br/>
<br/>        裴医生本不想告诉连中凯这个真相，只是当她看到连中凯对童筱颖起了杀心，才会于心不忍。

<br/>        毕竟，虎毒不食子，更何况，童筱颖还是夏白薇的女儿。
