        不知道为什么，夏玄灵也打心底的希望童筱颖不是。
<br/>
<br/>        倘若她是的话，那么童筱颖真的集万千宠爱于一身了。
<br/>
<br/>        她的养父夏阳枭是爷爷夏海东最宠爱的小儿子，听说，当年爷爷的这个幺孙女出生的时候，爷爷高兴得大摆宴席。
<br/>
<br/>        当时，就连同辈的堂哥堂姐们，都十分喜欢这个小妹妹，养母更是对她爱不释手。
<br/>
<br/>        只是，事到如今，尘封了这么多年的往事，再次被家族的人兴师动众地去寻人，也不知道养父和爷爷心里到底是怎么想的。
<br/>
<br/>        夏玄灵也怀疑过养父和爷爷突然寻人的动机，毕竟作为一个只是夏家的养女，她在这种亲情上还是很敏感的。
<br/>
<br/>        “会不会因为你的那个亲妹妹是‘解药’的缘故？”夏玄灵忽然想到这里，试探性地问。
<br/>
<br/>        “你怎么知道这件事情？”夏凌焕顿时感到诧异。
<br/>
<br/>        夏玄灵嘴角扬了扬：“不告诉你。”
<br/>
<br/>        “我希望你能告诉我。”夏凌焕凝眉道，这件事情，还是他小时候无意间偷听到的，至今成为了他心里无法抹去的记忆。
<br/>
<br/>        夏玄灵微微耸肩，摊手回答道：“有一次，妈妈让我替她去她房间里找药，我无意间在床头柜的抽屉里翻到了一本关于你亲妹妹的血液分析书，上面全是英文，又是专业术语，而且文件已经旧的发黄了。我好奇，刚准备用手机偷拍的时候，爸爸就进来了。虽然没偷拍到文件，不过，我倒是记住了其中一个出现濒临极高的英文单词‘’，我网上搜了一下，这单词的大概意思就是‘解药’，哥，我没说错吧？”
<br/>
<br/>        听完夏玄灵的讲述后，夏凌焕的心思沉了下来。
<br/>
<br/>        童筱颖回去后，对于夏玄灵的那件事情，她并未打电话去质问傅夜沉。
<br/>
<br/>        这是他们的私事，虽然她很是好奇，但也不至于去八卦。
<br/>
<br/>        夏玄灵有意隐瞒，她跟她虽然谈不上是朋友，但是也算是熟人了。
<br/>
<br/>        有些底线，她还是不要去触碰比较好。
<br/>
<br/>        所以，她给傅夜沉发短信的话题，也变成了普通的问候短信。
<br/>
<br/>        短信刚编辑完，正准备发出去，傅夜沉一个措手不及的视频电话打了过来，童筱颖误按了接通键。
<br/>
<br/>        “宝贝，你这么想我，嗯？视频电话秒接。”傅夜沉此刻穿着运动衫，左耳戴着蓝牙耳麦，身子一上一下，像是在跑步机上运动。
<br/>
<br/>        童筱颖目光的焦点，却落在了傅夜沉的身后。
<br/>
<br/>        落以柔扎着马尾，穿着一件白色的运动背心和白色短裙裤，身段玲珑，前凸后翘，在另一台跑步机上小跑着。
<br/>
<br/>        “你跟落秘书在一块啊？”童筱颖漫不经心地问。
<br/>
<br/>        傅夜沉忍俊不禁，然后将手机拿了起来，对着他周遭的环境，扫描了一圈。
<br/>
<br/>        “我们整个团队都在健身房里晨练。”
<br/>
<br/>        童筱颖尴尬地笑了笑，忽然觉得自己是不是太小家子气太敏感了。
<br/>
<br/>        “大哥，就是现在傅氏集团的CEO裴金澈跟你是什么关系？”童筱颖主动转开了话题。
<br/>
<br/>        傅夜沉扬了扬眉，答非所问：“他帅，还是我帅？”
<br/>
<br/>        童筱颖语塞，顿了一下，咧嘴一笑，“大哥最帅！”
<br/>
<br/>        “他算是我的‘铁哥们’，金融才子，管理集团比我厉害。”傅夜沉微笑道。
<br/>
<br/>        童筱颖抿了抿唇，然后把今早裴金澈救过他的事情跟傅夜沉说了：“我看到他擦伤了手，都流血了，但是就一晃的时间，他那伤口居然自动痊愈了。我都怀疑，是不是我眼花了。”
<br/>
<br/>        “筱颖，这件事你还有跟其他人提起过吗？”傅夜沉忽然一脸严肃地问道。
<br/>
<br/>        童筱颖微微摇了摇头。
<br/>
<br/>        “你就当没看见吧！”傅夜沉若有所思地接着说道。
<br/>
<br/>        童筱颖还是忍不住，好奇地问：“他真有那种特异功能？”
<br/>
<br/>        “这不算是特异功能，医学术语是什么，只怕得请教华生了。”傅夜沉淡定道。
<br/>
<br/>        很显然，他早就知道了裴金澈的这个秘密。
<br/>
<br/>        “连医生他没跟你一起去法国吗？”童筱颖接着问道。
<br/>
<br/>        傅夜沉摇了摇头，回答道：“他在国内有他的事情要办。”
<br/>
<br/>        “大哥。”
<br/>
<br/>        “嗯？”
<br/>
<br/>        “我现在是不是仍旧被你二十四小时监视着？”童筱颖忽然试探性地问。
<br/>
<br/>        傅夜沉苦笑：“是被我上头派的人监视着。”
<br/>
<br/>        “那……我洗澡上厕所呢？他们也监视？”童筱颖脸色一变。
<br/>
<br/>        傅夜沉顿时忍俊不禁起来，解释道：“他们只监视你的异常行动，不会妨碍你的私生活。”
<br/>
<br/>        “那也就是说，我在外的活动，见了什么人，做了什么事，大哥你岂不是都知道？”童筱颖若有所思地问。
