　　第709章
<br/>　　一枪洞穿崔云生的胸口，枪头从他背心冒出来，另外一枪扎穿崔云生的喉咙。
<br/>　　瞬间，崔云生便像田野中的稻草人一般，只能靠木棍撑着。噗通一声，从马背上坠落在地，已是气绝人亡。
<br/>　　马孟起同卢俊义互望一眼，颇有一种惺惺相惜的感觉。
<br/>　　“呔。”
<br/>　　百里胜向前跨出一步，拦在李乐天身前，口中历喝道：“今日唯诛首恶，其他不论，帝国陛下在此，有持戈者视同谋反。你们还不放下兵器么？”
<br/>　　如今正是抢功的时候，卢俊义与马超已经立下首功，百里胜也赶忙刷一波存在感。
<br/>　　集结的城防军官兵眼神闪烁，不知该如何是好。若说拿起兵器拼杀一场，他们也没了领头羊。
<br/>　　李乐天眼睛眯起，道：“数十之内，再有手持兵器者，尽屠之。”
<br/>　　“是！”
<br/>　　他身边的鱼龙卫痛快答应一声，一瞬间杀机四溢，气冲斗牛。
<br/>　　在这个时候，有兵器落地声响起，有人匍匐在地上，口中道：“陛下恕罪。”
<br/>　　这道声音像是点燃了引线，一片一片的人跪倒在地，口中呼着『陛下恕罪』。
<br/>　　直到此时，李乐天方才松了口气：此战定矣。
<br/>　　远处屋顶之上，趴着白元飞和刘发财二人。看到这一幕，白元飞点点头：“快将消息传给大当家，长安变天了。”
<br/>　　……
<br/>　　程大雷收到消息，已经是十天后，长安距离凉州颇远，一来一去都需要时间。
<br/>　　白元飞仍留在长安城收集情报，刘发财从长安赶回，向程大雷汇报长安城的情况。
<br/>　　“如此说，李乐天还是把事情办成了，定了长安城的局势。”程大雷念叨一声，心中也是感慨万千。当真是不怕一万，只怕万一。事情开始时，程大雷觉得李乐天几乎没有胜算，却是没想到，李乐天竟然凭着东拼西凑的兵力翻了盘。
<br/>　　真实情形，比程大雷所想得更加残酷。李乐天勉强凑齐的两千余人，在那一夜的战斗中死伤过半。打扫战场的时候，崔家一个年轻后生给了李乐天一刀，若非抢救及时，李乐天也须在那一夜交待性命。
<br/>　　那一夜他能够获胜，源于有马孟起和卢俊义两员绝世高手助阵，但更重要的原因，是因为谁也没想到，堂堂帝国天子竟做了一会亡命徒。以有心算无心，如此打了崔家一个措手不及。
<br/>　　“那一夜之后呢？”程大雷问，灭了崔相也未必表示事情就结束了，后续如何稳住局势才是关键的。
<br/>　　“那一夜之后，百里胜成了城防军的统治，率领两万城防军。城防军迎来一轮大清洗，李乐天提拔了许多平民子弟。”
<br/>　　“如此，长安城的局面算是稳住了。”程大雷点点头：“之后呢，之后李乐天是如何处理的？”
<br/>　　“相府满门上千口，无分男女老幼，全部被屠尽。刑场上砍得脑袋一片连成一片。那崔相的尸体被悬挂在城门上，属下回来的时候，他还在城门挂着呢。”
<br/>　　“这也算是理所应当。”程大雷问道：“除此之外，李乐天是如何处理长安之外的京州兵。”
<br/>　　“他下了一道圣旨，只说城中有人冒犯，那些军官的家人都被他保护了起来。让这些在外的军官回京述职。”
<br/>　　“这一招玩得够狠呐。”程大雷忍不住赞了一声，道：“有人听他的么？”
<br/>　　“有人听，也有人不听，但遵旨的还是大多数。”
<br/>　　之后，程大雷又问了刘发财一些其他事。当听到马超的名字时，他顿了顿，原来随机出世的大将落到了李乐天手中，如此也怪不得。
<br/>　　千军易得一将难求，在这个冷兵器时代，一名绝世武将能在战场上发挥出难以估测的实力。
<br/>　　“对了，你赶紧差人给陛下送一封信，只说我程大雷对帝国忠心耿耿，与相府势不两立。”
<br/>　　刘发财愣了愣：“这个时候送信，怕陛下会不高兴吧。”
<br/>　　程大雷撇了撇嘴：“无论他高兴或者不高兴，他都会假装高兴的。李乐天的情况也没那么乐观，现在的他，需要一个盟友。我想，他是不介意与我们做盟友的。”
<br/>　　“大当家，高呀。”刘发财忍不住赞了一声，虽然他也不知道程大雷高在那里，不过随口奉承本就是他的天赋：“那么大当家，这封信由谁来写？”
<br/>　　程大雷想了想，这件事按常理来说该由崔白玉来写，不过崔家刚刚死绝，自己还让对方参与这件事，有些忒不人道。
<br/>　　“随便找个人去做吧，吴军师就可以么。”
<br/>　　程大雷没有把这个消息特意透漏给崔白玉，然而也没有特意瞒着她。崔白玉帮着程大雷处理一些事情，不需要多长时间，崔白玉便知道了事情的全部。
<br/>　　她表现得很平静，但在帮程大雷书写文札时，还是颤抖了几分，在纸上落下几滴墨。
<br/>　　崔白玉虽然被崔相视作掌上明珠，但她对相府其实没有多深的感情。在那样一个庞大的家族，有复杂的利益纠葛，暴露真感情是一件很愚蠢的事情。大家只是因为同样利益必须抱团在一起，感情稀缺。而相府也曾抛弃过崔白玉，更是在崔白玉心里狠狠扎了一刀。
<br/>　　这是事实，可崔白玉姓崔也是事实。龙游浅水，她陷在程大雷这里，不得不帮着程大雷做些事。可她从来没有死心，一直在琢磨着如何脱离苦海，返回崔家。
<br/>　　但世事出人预料，相府曾经是一颗参天大树，树下有诸多躲雨的人。在这场乱世赌局上，任谁也无法忽略相府的存在。可是，这股最有存在感的势力，竟然在赌局刚开始便被淘汰出局。
<br/>　　一场赌局，挤了许多人。有人挤破脑袋往前凑，想要捞一份利益，有人安坐不动，自以为稳操胜券，如何自己都能立于不败之地。
<br/>　　可既然是乱世，关键就是一个乱字，输赢从不是一开始就写好的。
<br/>　　崔家满门被斩，株连九族。本还想着脱离苦海，回返崔家。
<br/>　　可如今崔家已经没了，满天下……皆成苦海。
<br/>(本章完)
