　　刚刚楚风眠让这些人交出功勋值的时候，便是就武风云一人，不愿意交出来，按照楚风眠的话去做。
<br/>　　这武风云，虽然败了，但是心中还极为不服气。
<br/>　　“吩咐？就凭你，也配吩咐我？”
<br/>　　武风云看着楚风眠，怒声喝道。
<br/>　　“哼！我可不是那群废物！我已经即将可以晋升内门弟子了，受到学院关注，你今日要是敢废了我的修为，学院不会放过你的！”
<br/>　　“等我晋升内门，实力在进一步，几个月后，我会在彻底击败你的！”
<br/>　　武风云如今跪在地上，但是他的嚣张，却是丝毫不减。
<br/>　　他有恃无恐，相信楚风眠不敢怎么对付他。
<br/>　　毕竟这样一尊风云人物，一举一动都会受到学院重视，楚风眠废了他，学院肯定会怪罪下来的。
<br/>　　不过，又有谁说的，只有废了武风云，跟放了武风云这两条路？
<br/>　　楚风眠的心中，可是还有着第三条路。
<br/>　　“诸天生死！归我掌控！”
<br/>　　楚风眠低声自语一声，心神一动，一股黑色灵力便是出现在楚风眠的掌心之中。
<br/>　　悄然之间，便是冲着武风云的额头上打了过去。
<br/>　　这一股黑色灵力，极为隐蔽，众人明明都没有丝毫察觉到，武风云的脸上，突然变为了极为恭敬的神色来。
<br/>　　随后楚风眠便是假装，在武风云的耳边低语了一句，只见武风云便是直接跪在了地上，恭敬的对了楚风眠开口道。
<br/>　　“我武风云，愿意臣服少主，为少主做事。”
<br/>　　这武风云如今脸上的忠心恭敬，可不是伪装的出来的，这突如其来的变化，简直也是在无数人一时间有些无法理解。
<br/>　　刚刚还极为嚣张，想要跟楚风眠对抗的武风云，如今在楚风眠这悄然的一声低语之中，便是选择乖乖臣服。
<br/>　　这简直是超出了任何人的想象。
<br/>　　“怎么回事？武风云的态度，怎么会变化的如此之大。”
<br/>　　“楚风眠在武风云的耳边说什么了？”
<br/>　　众人都是无比惊讶的看着眼前的突然变化，在他们看来，便是楚风眠悄悄的一句话之下，武风云便是就选择臣服。
<br/>　　这其中的奥秘，是他们根本无法看出的。
<br/>　　只有在远方，天际之上的两道身影，却是不由的目光闪烁，似乎是察觉到了什么。
<br/>　　“云兄，你也察觉出了吧，这楚风眠的手段。”
<br/>　　这两道身影，位于天际之上，隐藏身形，看着楚风眠，其中一人若有所思的说道。
<br/>　　他似乎便是察觉到了一些，楚风眠悄然之间的举动。
<br/>　　“这武风云，应该是被一种禁制给控制了，只是这禁制到底是什么，我却是根本没有见到过。”
<br/>　　那另外一尊武者，那位云兄，低语一句道。
<br/>　　“没想到武风云，居然都败了，这小子的实力果然有古怪，而且他还只是御风境的境界。”
<br/>　　“偌大的北域，出现几个天才，倒是不足为奇，以你我的实力，想要击败他，倒是不成问题。”
<br/>　　令一尊武者，笑了笑开口道。
<br/>　　“不过此人的实力，只怕足以冲击人榜前三了，只在你我之下了。”
<br/>　　“哦？叶兄居然是如此看重此人？”
<br/>　　那云兄，思考了一下开口道。
<br/>　　“倒是找个机会，可以去会会他。”
<br/>　　“我们先尽管其变，看看这小子，到底是什么来路再说，虽然是一个新人，但是此人却是强的可怕，不过找个机会，在他心中种下心魔，倒是不错，遏制一下他的发展，也不是不行。”
<br/>　　“毕竟这样的天才，要是真的崛起，对你我来说，可不是好事。”
<br/>　　那另外一名武者，叶兄，也是暗自开口了一句，交谈了这一阵，这两道身影便是也消失在了天空之上。
<br/>　　他们的对话，隐蔽无比，但是却瞒不过楚风眠的灵识，被楚风眠全部听见。
<br/>　　楚风眠的嘴角，露出一份冷笑，果然他在这北邙学院中强势崛起，肯定会引起一些强者的不满的。
<br/>　　但是楚风眠也毫不在乎，想要真正的成就强者，自然是要经过无数磨练，这些人，迟早都是楚风眠的垫脚石。
<br/>　　“武风云，带我去你的洞府。”
<br/>　　楚风眠转过头，对着武风云开口道。
<br/>　　如今楚风眠所在的洞府，虽然是这片山脉中最好的了，但是这片山脉，却是属于北邙学院中最为贫瘠的了。
<br/>　　武风云的洞府，显然是比起这还要好，楚风眠当然要直接拿走。
<br/>　　“是。”
<br/>　　武风云恭敬的说了一句，直接一道遁光，飞了出去，同时楚风眠的身形，也是紧跟其后，离开了。
<br/>　　这一战，算是楚风眠崭露头角的一战，这一届新人的强势崛起的消息，已经是在外门中流传。
<br/>　　对于这些事，楚风眠自然不会去在意，如今楚风眠跟武风云，便是来到了另外一处山脉之中。
<br/>　　“少主，这处山脉便是地元山脉，是北邙学院中的八等山脉，我的洞府，就在其中。”
<br/>　　武风云看着眼前的山脉，对着楚风眠缓缓开口解释道。

<br/>　　如今眼前这八等山脉，地元山脉，其中最小的洞府，都要比起刚刚楚风眠的那一座洞府，灵气要浓郁的多，其中的许多大洞府，灵气更是都要浓郁数百倍。
<br/>　　武风云的洞府，便是位于这地元山脉中，最好的一座。
<br/>　　楚风眠跟武风云二人，直接大摇大摆，来到了那洞府之中，果然楚风眠感觉到，这周围的灵气，都浓郁了许多。
<br/>　　“在这洞府中修炼，倒是还算是有些效果，不过却微乎其微，以我现在的实力，只怕是必须要六等山脉中的洞府才行，甚至是五等山脉中的洞府，才对我有着大用。”
<br/>　　楚风眠坐在洞府之中，感觉着周围的灵气，暗自想道。
