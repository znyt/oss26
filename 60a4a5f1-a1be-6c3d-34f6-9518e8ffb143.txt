<br/>				    有人以为没睡醒，眼花看错了，于是，揉了揉眼睛翻身坐起一看，竟还真的是那两人，那个如冰山一般冷着一张脸的黑袍男子，以及那个穿着红衣的绝美女子。<br/>　　<br/>　　    只是不同的是，这两人还是那两人不错，但那今天那红衣女子却是作男子打扮的，依旧是一依红衣，却偏偏穿出了不同的味道。<br/>　　<br/>　　    如果说这女子穿着红色衣裙时是妖娆绝美的，那眼下这穿扮成男装的她，就是肆意而邪魅的，那一身的邪魅尊华的无双气质给人一种雌雄莫辨的感觉，若不是早就见过她穿过一袭女装，此时真要以为眼前之人是一名男子。<br/>　　<br/>　　    而且，那些原本跟在他们身边的人以及契约兽都不见了，连同那个抱在怀中的孩子也不见了，他们想，也许是送了那些人离开了，他们两人也想再试一试这登仙梯的云梯仙道吧！<br/>　　<br/>　　    登仙梯前，轩辕墨泽牵着凤九的手，看着站在身边的她，带着磁性的声音轻声问着：“准备好了吗？”<br/>　　<br/>　　    “嗯！”<br/>　　<br/>　　    凤九应着，露出一抹笑意来。早在来时就已经准备好了面对一个新的陌生环境了，纵然陌生，但是，有他在身边，一切便也无惧了。<br/>　　<br/>　　    他们牵着手往登仙梯上走去，步伐从容不迫，每一步，都透着一股的自信，就仿佛他们所走的不是什么登仙梯，不是通往另一片天地的仙道，而是在自家的后花园中牵着手散着步一般从容……<br/>　　<br/>　　    两人结伴而上，也不知在这登仙梯上走了多久，只知道一步步的往上走着，那顶端仿佛也随着越来越近。<br/>　　<br/>　　    凤九来过这里，走起来也轻松，而轩辕墨泽虽是第一次走，但也没什么压力，只是，他并没有像凤九一样在这登仙梯上进阶。<br/>　　<br/>　　    直到，两人牵着手来到这最顶的那处阶梯时，两人停下了脚步，相视了一眼。<br/>　　<br/>　　    “这就是那玄光门了，走过了这里，就是另一片天地了。”凤九说着，看着轩辕墨泽，道：“上回我走到这里时，听到那边有几个老怪说话的声音。”<br/>　　<br/>　　    “应该是那边的强者。”轩辕墨泽说着。<br/>　　<br/>　　    “不知走过这里之后，我们会出现在什么地方？”她心下有些好奇，这一迈过去，所出现的会是什么样的一个地方呢？陌尘走上这里时，又是出现在哪个地方的呢？他独自一人去到那边，如今又会在哪里？<br/>　　<br/>　　    “无论在哪，我都会陪在你身边。”他握紧了她的手，道：“不用担心，我们走吧！”<br/>　　<br/>　　    “嗯。”她应了一声，紧紧的握着他的手，迈着步往前走去。<br/>　　<br/>　　    两人迈出了最后那一步，走进了那玄光门，只感觉一股力量将他们吸了进去，眼前光芒一闪，顿时一阵天旋地转。<br/>　　<br/>　　    几乎在那一刻，轩辕墨泽伸手一拉，将凤九拥入怀中，以黑袍护在怀里替她遮去那刺眼的光芒，他感觉到周围浓郁的灵力气息，以及喧哗热闹的声音，于是稳住身体往下落下，同时朝那下方看去……<br/>
