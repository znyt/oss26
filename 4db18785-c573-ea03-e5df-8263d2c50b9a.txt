　　“梦，不得无礼。”
<br/>　　就在这时，聂风走出断情居，看到慕清霄后，连忙走过来，恭敬行礼：“前辈，真是有好些年没见了，快里面请。”
<br/>　　闻言，慕清霄微微点头，边走边细细打量着聂风。
<br/>　　相隔十五年的岁月，聂风依旧潇洒俊逸，但浑身的气息却沉稳至极，显然已经迈入金丹境界。
<br/>　　“前辈，区区寒舍招待不周，还请见谅。”
<br/>　　“无妨，你这里倒是一块清修的好地方。”
<br/>　　片刻后，聂风和慕清霄相继落座，第二梦也是端来茶水，静静的坐在一旁，好奇的打量起慕清霄。
<br/>　　半分钟后，聂风忍不住开口问道：“不知前辈来此，所为何事？”
<br/>　　慕清霄端起茶杯，抿上一口清茶，语气平淡道：“我来此的目的，有关于帝释天。”
<br/>　　话语刚落，客厅中的气氛就变得极为压抑，聂风和第二梦的脸色也变得很难看，可见这个名字，对两人有多大的影响力。
<br/>　　“前辈知道此人？”
<br/>　　压下心中的愤恨，聂风和第二梦的目光落在慕清霄身上，等他继续说下去。
<br/>　　“当然知道，你们斗不过他也是情理之中，毕竟他可是活了上千年的老怪物。”
<br/>　　“千年......怎么可能！”
<br/>　　慕清霄的话语，确实将聂风和第二梦惊呆了，人怎么可能存活千年之久，他们熟知的无名前辈，也就两个甲子的年龄而已。
<br/>　　“大千世界无奇不有，实力越高，寿命就越长，存活上千年并不奇怪，当然也有某些神物能让人存活千年，甚至永生，帝释天就是如此。”
<br/>　　慕清霄微笑道：“我来这里的目的，就是需要你们配合我，铲除天门，以及帝释天那个老怪物。”
<br/>　　果然，听到慕清霄要出手铲除帝释天，聂风拱手道：“前辈请说。”
<br/>　　“这里不是说话的地方，我们先去霍家庄，找到步惊云，一起商议此事。”
<br/>　　闻言，聂风和第二梦相视一眼，毫不犹豫的跟上去，他们对帝释天的恨意，可是非常的深。
<br/>　　......
<br/>　　......
<br/>　　此时，慕清霄徒步走在隐秘的山谷中，他离开霍家庄已经有两天的时间，前天从聂风口中得知天门所在地后，就直接赶来。
<br/>　　他必须得到龙源，所以需要帝释天帮忙，来到天门，自然是想打入内部，到时候好一同前往神龙岛，
<br/>　　慕清霄随着聂风所叙述的路线，避过天门的重重暗哨，同时剑意隐晦的释放而出。
<br/>　　当他身形闪烁间，抵达偌大的庄园门口时，一名身穿宫装，面带寒冰面具的男子，已然负手而立，站在门口打量着他。
<br/>　　感受到宫装男子身上散发着凤血的气息，慕清霄就知道，此人就是帝释天无疑，而在他的身后，还站着数名强者。
<br/>　　神母，冰皇，怀空，神将，破军，断浪等人都恭敬的站在身后。
<br/>　　慕清霄将自己的气息压制在金丹六层，剑意也稍加收敛，目的就是为了引出帝释天。
<br/>　　帝释天屠龙，需要七柄武器，插进七大穴位，才能杀死神龙，慕清霄露出强大剑意的目的，不言明了。
<br/>　　“你是何人，来此有何目的？”
<br/>　　帝释天站在庄园门口，寒冰面具下，一双犀利的眼神在慕清霄身上来回扫荡，不时闪烁出精茫。
<br/>　　“在下慕清霄，为求名利权势而来。”
<br/>　　“哈哈哈......好一个为求名利权势而来，来帮我做事，本座可以给你一人之下，万人之上的权利。”
<br/>　　听到慕清霄的话语，帝释天顿时觉得有趣，如同羊癫疯一样大笑起来。
<br/>　　“帮你做事，那就要看你能不能给我想要的，有没有这种实力了。”慕清霄故意表现出猖狂的形象，完全不像是有心机之人。
<br/>　　“狂妄自大，就让我来替主人收拾收拾你。”话语刚落，帝释天身后的断浪就跳了出来，一剑刺向慕清霄。
<br/>　　感受到朝自己刺来的寒芒，慕清霄嘴角勾起一丝冷笑，真是一条好狗，微微抬起手臂，恐怖的剑意彻底爆发而出，约数仗的长剑虚影乍然而现！
<br/>　　“铿锵！”
<br/>　　断浪还没反应过来，由灵气凝聚而成的长剑就斩在他的剑上，恐怖的力量，直接将他轰飞出去，如同断了线的风筝，重重摔在地上，狂吐鲜血。
<br/>　　这般模样，可谓要多狼狈就有多狼狈。

<br/>　　感受到迎面而来的掌力，慕清霄再次凝聚出一道剑气，直接斩出，两股强大的力量，瞬间又撞击在一起。
<br/>　　“轰轰！”
<br/>　　在巨大的冲击力下，帝释天纹丝不动。
<br/>　　慕清霄则脸色胀红，踉踉跄跄的退后几步，抹去嘴角的血迹，豁然抬头，看向帝释天的眼神，已经变得完全不同。

<br/>　　“是，门主。”

<br/>　　“慕使者，请随我来吧。”
