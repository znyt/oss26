　　　　道风心中也是感到诧异不已。
<br/>　　“啊……”
<br/>　　一声痛苦的叫声，在寂静中响了起来。
<br/>　　不是道风，也不是叶少阳。
<br/>　　循着声音望去，总算找到了惨叫声的来源：芮冷玉！
<br/>　　芮冷玉弯腰蹲在地上，双手用力捂着两只耳朵，口中发出痛苦的呻吟，随即又变成了嚎叫。
<br/>　　怎么回事？
<br/>　　叶少阳扑上去抱住她，上下检查的身体，没发现任何异常，但她的脸色却是痛苦无比，已经翻滚到了地上，就好像被念紧箍咒的猴子，双手抱着脑袋，用力撕扯头发，在地上不断翻滚起来。
<br/>　　“小玉，小玉！”叶少阳上前用力按住芮冷玉，却又不知道该怎么办，用力抓住她一只手，罡气灌入她体内检测了一下，顿时好像被烫了一下，立刻弹开了，傻傻地望着芮冷玉。
<br/>　　她的体内，充盈着一股极为恐怖的能量，在她体内急速游走，叶少阳完全懵逼了，不知道这是怎么回事，猛然回头朝道风望去。
<br/>　　“快来帮帮我，看看她怎么回事！”
<br/>　　道风没有作声，表情凝重地望着他。
<br/>　　这样的反应，引起叶少阳的疑惑，转头看去，发现不少人都在用怪异的目光望着自己。自己那些门人，一个个目瞪口呆，小白捂住了嘴巴，失声道：“怎么会是她……”
<br/>　　“美华，四宝，郭师兄，你们在干什么呀，快来救救冷玉啊！”
<br/>　　“少阳，这……”四宝眼神充满了悲伤。老郭浑身哆嗦，说不出话来。瓜瓜完全懵逼。
<br/>　　小九双眼流泪，心中感慨到了极点。
<br/>　　叶少阳紧紧抱着芮冷玉，望着众人，嘶声吼叫起来：“你们怎么回事！”
<br/>　　大家不知道该怎么说。
<br/>　　“怎么会是她！”摇光仙子吃惊地发出了跟小九一样的感慨。
<br/>　　空界联盟的所有人，也完全震惊了。谁也没有想到会是这个结果……
<br/>　　“怎么会是她！”远处一座山上，女魃也是发出了一样的感叹，转头看去，后卿阴着脸，表情十分凝重。
<br/>　　“我早就该想到的。”后卿叹道，“当初你那分身，将尸气灌入她体内，想要杀死她，结果她居然没死，我们都不知道这是为什么。”
<br/>　　女魃一听，恍然想起这件事，心头大悟，点头道：“是了，我后来以为她体质特殊，原来……果然是特殊啊，她当然不可能变成僵尸，恐怕连鬼也做不成的。”
<br/>　　随即微微一笑，望着后卿说道：“怎么了，你心疼了？”
<br/>　　后卿没有理会她恶毒的调侃，远远凝视着芮冷玉，心中盘算着某个疯狂的念头……
<br/>　　“怎么办？”摇光仙子转头望着黎山老母，黎山老母现在也现出了本尊，两个人站在一起，像是一对年少的姐妹。
<br/>　　黎山老母想了一下，道：“都一样的，反正是他们中间的一个，不过……先看看再说。”
<br/>　　李浩然继续摇铃，表情更加带着一种疯狂。
<br/>　　芮冷玉已经停止了挣扎翻滚，开始抽搐，并且不断地大口吐血。
<br/>　　叶少阳慌神了，愣在当场，心如刀绞，但是对眼前的突发情况，不知道怎么办好。
<br/>　　“少阳。”道风低声唤了叶少阳一声，“你松开她，快过来。”
<br/>　　“什么！”叶少阳觉得自己是不是听错了。
<br/>　　“我早就说过她想杀你，你却不信，今天你自己总看到了吧？”
<br/>　　“看到什么？”叶少阳震住。
<br/>　　道风手指芮冷玉，说道：“她，才是转世鬼童！”
<br/>　　宛如一道霹雳，砸在叶少阳脑海中。叶少阳身躯晃了一下，险些摔倒。看着怀中正在吐血颤抖的芮冷玉一眼，仿佛明白了什么，但情感上仍然不能接受，缓缓摇头，喃喃道：“不，我不相信，不会的……”
<br/>　　道风说道：“那她为什么听到铃声，会表现出这个样子？少阳，你接受现实吧，她……就是转世鬼童。”
<br/>　　最后几个字，如同锤子一样敲打在叶少阳心头。
<br/>　　转世鬼童……
<br/>　　“那她为什么要杀我？”叶少阳想要抓住最后一根稻草。
<br/>　　“你是应劫而生，她是转世鬼童，或许，她接近你，就是为了杀你，这样才能推动天劫降临。”
<br/>　　黎山老母等人听见他们的对话，再度感到了震惊。互相看了一眼，都是感到一阵后怕。
<br/>　　道风在这个时候跟叶少阳说的话，自然不会有假。原来他是应劫而生，好在之前他没有死在北斗七星阵中，否则真是做了与初衷完全相反的事情。
<br/>　　叶少阳定定地看着道风，道：“你意思说，她是为了杀我，才接近我？”
<br/>　　“还能有别的解释吗？”
<br/>　　“可是……她是有父母的啊，也是活生生的人！”
<br/>　　“鬼童转生，自然是潜伏在轮回中，如同正常人一样，有血有肉，不然怎么骗得过轮回司？之前的她可能一直没有觉醒。现在，她觉醒了。”
<br/>　　觉醒了……
<br/>　　叶少阳转头望着怀中的芮冷玉。芮冷玉已经停止了吐血和抽搐，闭着眼睛一动不动，似乎进入了一种忘我的境界。
<br/>　　有丝丝缕缕的煞气，从她身上不断渗出，如同火焰一样升腾而起，看见这一幕，更是让大家确定了对真相的判断。
<br/>　　道风心中，也是有一种造化弄人的感慨，世人一直都以为他是转世鬼童，而他则一直以为叶少阳是，原来，大家都错了……
<br/>　　“不，不可能的！我不相信啊！”叶少阳嘶声大叫，“她对我是真心的，绝不可能是故意接近我！我不信！”
<br/>　　道风沉默了一下，道：“你信也好，不信也好，既然她是转世鬼童，并且要杀你，你……必须有个决断了。”
<br/>　　叶少阳猛然震住，道：“什么决断，你让我杀了她？”
<br/>　　道风道：“我知道你下不去手，这件事，我可以帮你。”
<br/>　　说完朝叶少阳走过去。
<br/>　　叶少阳立刻警觉起来，猛然拔出七星龙泉剑，指着道风，咬牙说道：“你不要过来，你如果想杀她，先要杀了我！”
<br/>　　道风站住，冷冷道：“意气用事。不管是与公还是与私，她都必须死。”
<br/>　　“你住口！”叶少阳哀伤地望着他，“你曾经教过我，道之所在，虽千万人吾往矣，你怎么也没说出这种话来！”
