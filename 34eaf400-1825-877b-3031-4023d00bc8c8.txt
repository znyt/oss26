        连中凯暗自吃起了傅夜沉的醋来，见童筱颖迫不及待地从床上下来，连忙呵斥道：“去哪儿啊？”
<br/>
<br/>        “当然是去见我老公啊！”童筱颖没好气地应道。
<br/>
<br/>        连中凯一脸严肃地瞪着童筱颖：“你就给我在这老老实实的待着，没我的命令，就别想离开这儿半步！”
<br/>
<br/>        “你……”
<br/>
<br/>        童筱颖的话还未说完，连中凯便带着手下走了出去。
<br/>
<br/>        当童筱颖只身走到门口时，发现这门竟然从外反锁了，气得随口谩骂了一句：“该死的连老头！”
<br/>
<br/>        “臭丫头，我是你爹！”
<br/>
<br/>        门外忽然传来一声怒斥，将童筱颖吓了一跳。
<br/>
<br/>        连中凯那老头居然还没走！
<br/>
<br/>        童筱颖连忙说道：“你开门！放我出去！”
<br/>
<br/>        “我还以为你把最近所发生的事情给忘了，看样子是没忘！没忘就好！你给我记住了，从今以后，我是你爹。你不承认，也得承认！”连中凯碎碎念叨完，便离开了。
<br/>
<br/>        只留童筱颖一个人在房内生闷气。
<br/>
<br/>        随后，连中凯走出了基地，独自一个人去面对傅夜沉。
<br/>
<br/>        傅夜沉带着自己的手下，还真是全副武装地聚集在了基地的大门口。
<br/>
<br/>        看到连中凯走出来的时候，傅夜沉也没有命令手下将武器收好，而是将枪口对准了连中凯。
<br/>
<br/>        连中凯好整以暇地走到傅夜沉的跟前，嗤之以鼻地白了一眼傅夜沉，说道：“这就是你对待岳父的态度？”
<br/>
<br/>        岳父？！
<br/>
<br/>        傅夜沉懵了一下。
<br/>
<br/>        在这之前，他只知道，筱颖是夏爷爷夏海东的外孙女，和傅昼景确实是表兄妹关系，但是，他连中凯这个“岳父”是什么鬼？
<br/>
<br/>        “还不让你的手下把枪都给我收起来！”连中凯怒斥道。
<br/>
<br/>        傅夜沉不以为然地回答道：“把筱颖还给我，不然，别怪我对你不客气！”
<br/>
<br/>        “我说你是听不懂我的话吗？有你这么对待岳父的吗？我告诉你！你要是敢炸我这里！你这一辈子就别想见我女儿！哼！”连中凯顿时来脾气了，但是，发起气来的样子，却像一个老顽童。
<br/>
<br/>        傅夜沉抬了抬手，身后的黑衣人，纷纷收起武器。
<br/>
<br/>        连中凯得意地笑了笑：“我这地方，容不下你这么多人，就你一个人跟我进去得了！”
<br/>
<br/>        傅夜沉又打个响指，身后那群黑衣人立即识趣地转身离去。
<br/>
<br/>        连中凯却憋着嘴，转身进了基地的大门。
<br/>
<br/>        傅夜沉随后跟了进去。
<br/>
<br/>        “我可以让你见我女儿，但是，你得劝我女儿认我这个爹！”连中凯走在前头，有模有样地跟傅夜沉谈起了条件。
<br/>
<br/>        “你拿筱颖的血制药，用完后，还指望她会认你这个爹？你不觉得可笑？”傅夜沉冷嗤。
<br/>
<br/>        连中凯顿时停下脚步，转过身来，指着傅夜沉的鼻子，破口大骂：“要不是白薇当年看上你爹，我能落得个现在妻离子散的下场吗？”
<br/>
<br/>        “所以，你因爱生恨，杀了我父亲！”傅夜沉黑眸微眯，眼神里透着一股戾气。
<br/>
<br/>        连中凯吓得气势弱了下来，支支吾吾地说道：“我、我想杀来着！但是，我还没亲自动手，你老爸就被人先行下了病毒！”
<br/>
<br/>        “所以你见死不救？”傅夜沉呵斥道。
<br/>
<br/>        连中凯却立马反驳道：“谁说我见死不救了！我告诉你，要不是白薇求我，我才懒得去救你爹！问题是，你爹身上的那种病毒，用民间的说法，就是一种‘蛊毒’，来自苗疆。我还让龙半夏去找她母亲龙沉香，拿她母亲的‘蛊札’回来救你爹，哪知道龙半夏半路连同哪本‘蛊札’一起失踪了。”
<br/>
<br/>        “可你们当初给出的交代是，龙半夏是殉情。”傅夜沉微微皱起了眉头。
<br/>
<br/>        连中凯接着说道：“这事情牵扯太大，查起来很麻烦，更何况，你爹当年那刚正不阿的性子，肯定得罪过不少人。而且，当年夏阳枭又处于升职阶段，自然不想因为你爹跟龙半夏的事情阻碍了他的前途。所以，才不得不对外发报说，你爹是殉国，龙半夏是殉情。”
<br/>
<br/>        “你们可真够自私！”傅夜沉不禁双手握拳。
<br/>
<br/>        连中凯却在心底暗自迷茫了。
<br/>
<br/>        其实，下令暗杀他傅夜沉的人，真不是他。
<br/>
<br/>        就算童筱颖不是他女儿，他也不可能为了记恨白薇不爱自己，而去杀冷卫峰的长子。
<br/>
<br/>        只是，这件事情，对外来说，他不得不一个人扛下来。
<br/>
<br/>        连中凯有自己的苦衷，却不能说出来。
<br/>
<br/>        “你把筱颖带回去后，好好对她。不要再跟夏阳枭作对了，你再怎么斗，都是斗不过他的！”连中凯突然话中有话地说道。
