　　突然出现的喰种，让众人立即警觉了起来，有马贵将站在队伍前方，一头白发在走廊灯光的衬托下格外刺眼。
<br/>　　“我很好奇你是怎么逃出来的，监控录像上显示，你是使用了类似瞬间移动的方式，这种能力，我都还没有见过。”
<br/>　　将手中的长刀向上抬了几寸，有马饶有兴致的看着面前的喰种。
<br/>　　虽然戴着月山家标志性的红色面具，但和刘哔交过一次手，他自然能分辨出其身上的那股气味。
<br/>　　“不过，也没关系，总议长大人吩咐，让我活着把你带回去，不知道你是要自己走，还是想像上次一样，被我绑回去。”
<br/>　　听到有马略带嘲讽的话语，刘哔并没有动怒，依旧笑吟吟的坐在椅子上，一双赤红的瞳目中，满是玩味。
<br/>　　想要活捉他吗？
<br/>　　看来是被小瞧了呢，
<br/>　　他可不像上次一样弱小了。
<br/>　　慢腾腾的从老板椅上战起，刘哔舒展了一下身体，摘下了面具，露出一张有些清秀的脸庞。
<br/>　　有马贵将并没有动手，像他们这种级别的战斗，率先动手的，会非常吃亏。
<br/>　　除非是，敌方露出什么破绽，但刘哔明显不会给他这样的机会。
<br/>　　“我要上了，请准备好。”
<br/>　　脸上露出一个阴险的笑容，刘哔激活双翼，直接面向众人展开了进攻。
<br/>　　他的思路十分明确，先解决掉那几只碍事的喽啰，最后再去对付有马贵将。
<br/>　　但是现在，本身就有SSS级实力的他，再配合上赫者化和死亡之翼的加成，普通的S级，根本不可能阻挡住他的进攻。
<br/>　　如果没有提前死掉两人，这五人组成阵型，或许还能对他造成一点威胁，现在的话，就只能等死了。
<br/>　　在这次行动之前，他就特意向什造打探了一下，有马身边的五名手下，和CCG其余搜查官并不相同。
<br/>　　这五个人，如果分开来战斗，也是能算是很普通的S级，但五人如果联手，却能爆发出高于几倍的力量。
<br/>　　甚至能和SSS级别的喰种相抗衡，再结合上有马贵将，即使他火力全开，也没有必胜的把握。
<br/>　　故此，在监控室中看到有马贵将上来，刘哔便提前从窗户离开，找机会解决掉了两名监察走廊的搜查官。
<br/>　　为了避免战斗的时候被有马发觉，他特地从商城兑换了一张结界卷轴，将战斗区域给封闭了起来。
<br/>　　一切都没什么意外，战斗完成之后，甚至还给他留下了足够的时间布置了一下现场，方才主动显露了出来。
<br/>　　现在这个残缺的队伍，对他来说已经不会有太大威胁了。
<br/>　　“嗤啦！”
<br/>　　一只染血的拳套，和有马贵将的长刀碰撞在了一起，本来是攻向其余搜查官的攻击，被他及时挡了下来。
<br/>　　刘哔舔了舔嘴唇，心中略微感到意外，脸色变的有些认真了起来。
<br/>　　这个男人，比他预计中的强大了许多，至少光是这份速度和反应力，也都不容小觑。
<br/>　　借助着刘哔被挡下的机会，周围几个搜查官也反应了过来，立即动用自己的库克因，向他心脏的位置发动了攻击。
<br/>　　在众人惊讶的表情中，刘哔对于即将落在自己身上的攻击不闪不避，一脚踢出，踹向一名搜查官的头颅。
<br/>　　这么疯狂的动作，让这名搜查官一惊，眼看着他手中的库克因即将刺穿敌人的心脏，但被迫之下，他也只能放弃，无奈将库尔因转换成盾形，护住了自己头颅。
<br/>　　“砰！”
<br/>　　一声闷响，刘哔一脚踢出，狠狠踹在了盾形库克因上。
<br/>　　随后，在那名搜查官不可置信的眼神下，库克因表面升起无数蛛网式的裂纹，他整个人都被这势大力沉的一击给轰的倒飞出去，撞塌了身后的正面墙壁。
<br/>　　一击得手，刘哔硬扛下了其他几人的攻击，任由那些库克因，在自己背后留下一连串的伤痕，而他借助这宝贵的时间，直接冲到了那名搜查官面前，一脚踩了下去。
<br/>　　“噗嗤！”
<br/>　　脚下传来一股粘稠感，那名搜查官的头颅，如同西瓜一般破裂开来，暗红色的血液，顿时洒在了走廊的地板上。
<br/>　　“还有三个。”躲开了身后的几道攻击，刘哔舔了舔嘴唇，赤红色的瞳目中，露出一抹兴奋，向几人继续出手。
<br/>　　看到自己的手下身死，有马脸色阴沉了下来，这些人，跟随他出生入死了多年，却没想到折损在了这里。
<br/>　　手中做了一个的手势，场中几名搜查官顿时领会，瞬间四散了开来，有马袖口伸出一柄模样怪异的库尔因，随后猛然放大，对准了刘哔。
<br/>　　突如其来的变故，让他没办法从战场撤离，紧接着，一串沉闷的声音响起，一颗颗拖着焰尾的金色炮弹，向他站立的地方飞来。
<br/>　　鸣神？
<br/>　　心中微微一紧，刘哔不敢有丝毫大意，上次和有马贵将交手的时候，他就是被这招击败。
<br/>　　只是现在要面临的力度，和第一次比起来简直是天壤之别。
<br/>　　爆裂的声音不断响起，金色炮弹落在四周的墙壁和地面上，打出一个个坑洞，威力之强，整栋大厦都开始震动了起来。
<br/>　　在密密麻麻的炮弹中穿梭着，尽管他的速度已经够快，但身体还是受到了不轻的伤势，所幸的是，这并不怎么影响战斗力。
<br/>　　在付出了一些代价之后，刘哔也终于从轰炸区中窜出，来到了有马贵将面前。
<br/>　　此时，有马贵将也收起了进攻，将手中的库克因转换成一柄长刀，配合另外一把向刘哔攻来。
<br/>　　既然距离已经拉的如此近了，远程攻击也就失去了原本的作用，但没关系，近身战他也同样也很擅长。
<br/>　　“啧，还真是小瞧了你呢，不过，我很好奇，你明明如此强大，可为何却想要做一条狗呢？”挡下了有马的进攻，刘哔语气嘲讽的说道。
<br/>　　但是回应他的，依旧是两把锋利的长刀，对于他的话语，有马并没有太大反应。
<br/>　　反手一拳轰在了有马没有任何表情的脸庞上，刘哔也收起了微笑，红色的瞳目中，一片冰冷。
<br/>　　妈的，他讨厌面瘫！
