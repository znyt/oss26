　　经过巫妖大战之后，洪荒一片萧瑟，战后的洪荒，不再如同之前那般繁荣，大地被打沉，山川被打碎，河流断流，草木破败，星辰破碎……真不知道要到何时，洪荒才会恢复过来。
<br/>　　但，话又说过来，洪荒最不缺的就是时间，时间能够抹平一切伤痕。巫妖两族经过此次大战，两败俱伤，各自都在恢复生息，积累实力，全力发展各自的族群，等待着下一次大战，报仇雪恨。
<br/>　　没了巫妖两族的挑头，洪荒之中厮杀减少了许多，洪荒迎来难得的平静时期。
<br/>　　眨眼间五百年时间便过去了，鸿钧道祖第三次讲道的时期即将到来，洪荒之中的大神通者，纷纷结群组队地向着紫霄宫赶去，元始三人也停止讲道，准备前往紫霄宫听讲。
<br/>　　“此次讲道，道祖鸿钧将会分发灵宝，讲道之后，道祖便会以身合天道，从此鸿钧为天道，天道不为鸿钧，大劫不出，鸿钧不现，而到了那时，洪荒将会迎来真正的繁荣发展。此次讲道，那分宝岩之上宝贝众多，不如多带几个人？”元始如是想道。
<br/>　　“玄青、玲珑、白衣、晓月，尔等速来见吾！”想到此处，元始便传音叫来自己的四个徒弟。
<br/>　　“见过老师！见过两位师伯！”四人听到传音，不敢耽搁，立马前来拜见。
<br/>　　“道祖第三次讲道将要开始，吾观尔等四人这些年在修炼上也不敢怠慢，都已经到了大罗金仙中期了，此番讲道，为师便带你们去见识一下。”
<br/>　　元始看着眼前自己的四位弟子，脸上露出一丝满意的微笑，虽然他们只是记名弟子，但元始一点也没有亏待他们，他们的表现也让元始颇为满意。
<br/>　　“啊？！”
<br/>　　四人脑袋有些懵了，老师还从来没有带他们见识过紫霄宫，每次讲道，自己等人都只能无聊的待在家中，不得外出，这都待烦了。如今自己的老师竟然会带他们出去听道，顿时，他们感觉自己被巨大的幸福砸晕了。
<br/>　　“怎么？不愿意！”元始脸上露出一丝不奈，淡淡道。
<br/>　　“啊？不！不！徒儿谢过老师！”四人顿时反应过来，对着元始拜谢道。
<br/>　　“大兄，你怎么会想到带他们去听道？”这时，**疑惑道。自己的大兄不是一直讨厌麻烦吗，现如今又怎么会带着他的四个徒弟去听道？
<br/>　　“吾有感此次听道将会有一场机缘，将他们带上，看看能否得到些许。”元始神秘的说道。
<br/>　　“嗯？机缘？”通天不解。
<br/>　　“好了，不必再耽搁了，我们走。”之后，元始不再停留，带着几人，向着混沌中赶去。
<br/>　　待几人到紫霄宫后，发现已经来了不少人了。元始几人一入紫霄宫，顿时就吸引了所有人的眼球。准确的说，是元始吸引了所有人。
<br/>　　不是因为元始长得太帅，也不是因为元始修为最高，而是因为之前元始开创一个新境界，让众修士不太明白。如今，正主到了，他们正好观察一番。
<br/>　　元始眼神平淡如水，毫无波动，全身气息收敛，让人看上去，感觉如同是看到一个凡人，看不出丝毫虚实。但众修士又知道，元始不可能是凡人。众修士只觉得元始如同大海一般，深不可测。
<br/>　　妖帝帝俊、东皇太一等妖脸色变得更加难看了，眼中有仇恨、有不甘、更有一丝忌惮。他们没想到元始修为提升得如此之快，看来他们的仇是很难再报了。
<br/>　　元始脸色没有丝毫变化，眼神无丝毫波动，虽然被众修士看着，但他也不看众修士一眼，如同是对待一花一草那般平常到不能再平常的事物。众修士虽然感觉不到元始的情绪波动，但他们只觉得自己此刻被无视了，被彻底的无视了。
<br/>　　每人心中都升起一丝愤恨，自己等人在洪荒之中，好歹也是一方霸主，就算不是霸主，但也差不多了。在洪荒中，他们被无数人尊敬，何曾受过今日这般窝囊，但在你元始眼中，我们却被如此看轻。真是不想让人愤恨都不行啊。
<br/>　　元始不管众修士心中的复杂之情，他带着几人，径直来到大殿最前方，第一个蒲团上坐下，之后将四个徒弟安排在身边，便不再多看其他人一眼，而是开始闭目养神。
<br/>　　元始对众修士是不屑，但他的徒弟们可就惨了。四人坐在元始身后，只觉得如坐针毡。无数道眼光看向他们，而且好多人都修为比自己高许多，他们能坐得舒服才怪，但他们又顾忌老师，又不好动作。
<br/>　　元始看在眼中，心里神秘一笑。他这四个徒弟修为虽然不错，但没有见过什么大世面，以后办事，就算别人看在他元始的面子上不说，难免会被人心里瞧不起，如今此处有如此多大人物，想必足够他们炼心了。
<br/>　　果然，事实还是没有出乎元始意料，自己的四位徒弟从开始的有些惊慌，到之后的有些不舒服，再之后的些许不适，最后渐渐变得没有丝毫的不适，恢复淡然。他们一样学着元始，闭上双眼，参悟道法。
<br/>　　四人的表现被众人看在眼里，众修士点了点头，心道，果然不愧是玉清元始的徒弟啊，适应得很快嘛，心性不错！
<br/>　　时间一点点过去了，来听道的人也越来越多了，但紫霄宫只有如此大的一点地，仅够三千人。三千人一满，其他人便只能算是无缘了，被拒在混沌之中，找不到路。毕竟，道不可轻传呐。
<br/>　　随着最后几人的到来，紫霄宫已满，宫门被关闭，还是昊天与瑶池这两个童子，宣布规则之后，便去敲响金钟，宣布讲道开始。
<br/>　　“第三次讲道开始，尔等不得大声喧哗！”
<br/>　　道祖鸿钧那越发缥缈大声身影缓缓出现在殿前云台之上，让人看不清丝毫虚实，只觉得高深莫测。
<br/>　　“鸿钧，你已经被天道影响如此之多了？！真不知道那是好是坏？呵！”元始蓦然瞳孔一缩。
<br/>　　道祖鸿钧还是身穿一声玄色道袍，面容苍古，他的双眼蓦然睁开，其内流露出的，是一种无法想象的淡漠，冰冷无情，仿佛蕴含无尽沧桑。他微微打量了一眼紫霄宫众修士，看到元始之时，眼神微微一动，之后便不再出现丝毫波动。那眼神，比之前元始的眼神还要淡漠，如同是在看一群刍狗。
<br/>　　天地不仁，以万物为刍狗！
<br/>　　或许，在鸿钧眼中，众人如同那刍狗一般，并无其他区别。但众人此次却并没有之前被元始无视那种愤恨，他们只觉得道祖深不可测，这样的表情，十分正常。
<br/>　　鸿钧不理他们心思，那淡漠的声音在大殿之中响起：
<br/>　　“第三次讲道，亦是最后一次讲道开始，尔等各自明悟！”
