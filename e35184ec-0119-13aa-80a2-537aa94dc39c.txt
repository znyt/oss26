        家，应该是最温暖的地方。
<br/>
<br/>        但是对于诺筱颖来说，她的那个家真的好冷。
<br/>
<br/>        诺母看到诺筱颖拖着行李箱风尘仆仆地从临海城回来，不但不关心她，反而冷嘲热讽：“就你那破文凭，还想去大城市混？真是自不量力！”
<br/>
<br/>        “妈，我还没毕业，没有毕业证，所以才不好找工作！”诺筱颖反驳道。
<br/>
<br/>        诺母仍旧一脸鄙夷：“我看你大二就别读了，反正读了也是浪费钱，就算你拿到毕业证又怎样？大专文凭而已，大城市里又有哪家公司能看得上你？”
<br/>
<br/>        诺母的话字字带刺，还真刺痛了诺筱颖的心，她不想再跟诺母费口舌，提着行李箱，充耳不闻地上了楼。
<br/>
<br/>        她就知道，她不应该回来的！
<br/>
<br/>        “你拽什么拽？你连梦萌一半都不如！还有，你别指望我和你爸再给你交大二的学费了！”
<br/>
<br/>        背后，诺母扯着嗓子泼冷水。
<br/>
<br/>        梦萌是谁？她诺筱颖的表妹，而且还是个千金大小姐。非常得诺母的喜爱，甚至，诺筱颖都有种错觉，童梦萌才是她老妈亲生的，而她诺筱颖是捡来的。
<br/>
<br/>        诺筱颖重重地关上房门，委屈地落下了眼泪，就连晚饭她也没心情出去吃。
<br/>
<br/>        门外，仍然能听到诺母在跟诺爸唠叨：“筱颍这臭丫头真是白养了！你瞧咱们的梦萌多好，上的是国外一流的本科大学，每学期都拿一等奖学金回来！又风光又体面！筱颍那臭丫头压根就不如梦萌的一半！”
<br/>
<br/>        “筱颍在学校里，也是每个学期拿一等奖学金回来啊！所以，老婆，你就少说两句，梦萌毕竟是筱颍的亲表妹，你妹妹嫁得比你好，家里条件比咱家好，所以，两个孩子之间怎能相提并论呢？”诺爸有些不耐烦。
<br/>
<br/>        诺母却振振有辞地接着说：“怎么就不能相提并论呢？筱颍那种三流大学的奖学金，能和梦萌那一流大学的奖学金相比吗？你说说，两人同年同月同日同时在同一个产房里出生，可差别怎么就这么大？”
<br/>
<br/>        不过差别大也正常，反正筱颍这丫头又不是……
<br/>
<br/>        “老婆，其实筱颍并非……”
<br/>
<br/>        诺爸欲言而止，打断了诺母内心的思绪。
<br/>
<br/>        诺母敏感地一怔，莫非老公也知道筱颍的那事了？
<br/>
<br/>        诺爸随即转开话题，痛心疾首地感慨：“筱颍并非成绩不好，而是高考那年为了照顾我，给我凑医药费，把学习时间花在了去医院照顾我和兼职打工赚钱上，她才会落榜的啊！”
<br/>
<br/>        诺母顿时觉得自己虚惊一场，不以为然地回答：“照顾你，是她当女儿应该做的！不然，我还真白养了她十八年！”
<br/>
<br/>        瞧，她的妈妈，对她是有多么淡漠。
<br/>
<br/>        诺筱颖待在家里的这些天里，天天都要遭受妈妈的冷嘲热讽。
<br/>
<br/>        不管她在家里多么的勤快，洗衣服做饭帮忙照看家里的药店，她的妈妈，永远都对她有说不完的不满，道不完的抱怨。
<br/>
<br/>        这天吃晚饭的时候，诺母突然提起了苏漫雪的事。
<br/>
<br/>        “筱颍啊！苏漫雪和你是大学同班同学吧！”诺母手里拿着筷子一边夹菜往嘴里塞，一边吧唧吧唧地说道，“我听说，苏漫雪在临海城里交了一个非常非常有钱的男朋友。她那男朋友给她的零花钱，每个月就好几十万咧！现在，苏漫雪不仅给她的爷爷奶奶，在我们这小镇上买了一幢三层楼带门面的房子，还在咱们村上盖了一栋四层楼的大别墅，听说还是前有花园果园，后有池塘菜园子的那种！”
<br/>
<br/>        诺母说得羡慕不已，眼睛都红了一圈。
<br/>
<br/>        诺筱颖吃饭的动作怔了一下，忽然间想起了什么，很快又装作若无其事的样子继续吃饭。
<br/>
<br/>        她知道苏漫雪突然间有了一个未婚夫，要结婚了，但是她没想到苏漫雪的未婚夫竟然这么有钱，有钱到能给苏漫雪每个月几十万的零花钱用。
<br/>
<br/>        “我说筱颍你啊！你长得又不比苏漫雪差，眼光怎么就比苏漫雪差那么多？那个何明旭有什么好的？不就是咱们村里唯一一个考起了临海城大学的本科大学生嘛！他家里穷得要死，上个大学的钱，还是村上帮忙凑的。比苏漫雪的那个男朋友，万分之一都不及了。真不知道你看上了何明旭什么，居然还想着在毕业后就嫁给他，帮他照顾家里的老父老母，还有他那年迈多病的爷爷奶奶！”诺母又絮絮叨叨了起来。
<br/>
<br/>        一提起何明旭，诺筱颖突然气恼地将碗筷重重地往桌上一放，将诺爸诺母给吓了一跳。
<br/>
<br/>        “我和他已经分手了，这下你满意了吧！”诺筱颖眼中含泪，一肚子委屈地怒吼。
<br/>
<br/>        诺母顿时火气一上来，便怒冲冲对诺筱颖凶巴巴地谩骂：“你这臭丫头！吼什么吼？没大没小了是吧！你两分手，关我什么事儿啊？你这臭丫头有什么资格在我面前大吼大叫？”
<br/>
<br/>        “老婆，你就少说两句嘛！”诺爸叹了口气，深沉地劝阻道。
<br/>
<br/>        其实，比起何明旭的家庭，她诺筱颖的家庭也好不到哪里去。
<br/>
<br/>        她的妈妈在这小镇里经营着家里一楼那十几个平方的小药店，爸爸双腿不便，待在家里只能帮着坐在药店里的收营台前收收钱找找零钱什么的。
<br/>
<br/>        而且，因为爸爸的腿疾，家里还欠了一些亲戚朋友们的债。
<br/>
<br/>        至于她的哥哥……
<br/>
<br/>        虽然在外打工，但每个月还要从家里拿钱去贴用。
<br/>
<br/>        话题到这里，诺母缓了口气，语气也稍稍缓和了一些，瞥了诺筱颖一眼后，接着说道：“你不是跟苏漫雪是闺蜜是好朋友吗？你去跟她说说，让她接济接济你，拿点儿钱出来，供你把大学读完。”
<br/>
<br/>        “妈，你这说的是什么话？”诺筱颖顿时就恼火了。
