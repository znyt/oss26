　　第305章
<br/>　　无论如何，程大雷和莫鸣米保持了表面上的和平，双方互不侵犯。大家都知道，这和平维持不了太久，趁这时间，程大雷争分夺秒，赶紧升主基地。
<br/>　　而除了要求互不侵犯外，程大雷还向莫鸣米要了一条商路。云中龙、乔鹤、游九楼就是这条商路的负责人，相对而言，这三人是十八家头领中稍弱的势力。本来云中龙还称得上强，只不过经过最近的事情，也是损兵折将大半。
<br/>　　相比三人的尴尬，程大雷倒是很坦然。
<br/>　　“做贼么，无非就是求一个发财而已，愿意跟着我混，日子只会比以前好，不会比以前差。如果不愿意跟着我混……好像，你们也没有其他选择了。”
<br/>　　三人面面相觑，最后决定由云中龙开口，他和程大雷算是熟悉的。
<br/>　　“程老大，我们都愿意跟着你。只是，这样就站在了莫老大的对立面，和莫老大为敌？”
<br/>　　“和莫老大为敌怎么了？”程大雷问。
<br/>　　“琴川这里是莫老大说了算的？”乔鹤道。
<br/>　　“以前是他说了算，但现在我来了，你们觉得他说话还能算么。”程大雷微挑眉毛。
<br/>　　看三人忐忑的模样，心里定是还没有拿定主意，主要原因是，这莫老大积威太重，这三人心里还是畏惧对方的。
<br/>　　程大雷也知道，不可能凭三言两语就让他们纳头便拜，江湖上人还得让人家吃饱饭，然后才能谈义气。他点点头：“做着试试吧，我想我不会亏待你们的，你们可以先干几天，边走边瞧。”
<br/>　　“程当家要我们做什么？”
<br/>　　“开一条商路，和戎族做生意。”
<br/>　　“程当家，这个，那个……”
<br/>　　“有话直说，吞吞吐吐做什么？”程大雷补充一句：“我一直都很民主的。”
<br/>　　“这条道上的生意不好做，只有咱们四家负责，许多路就都不能走，走别人地盘就是坏规矩。而戎族那边的生意也不好做。”云中龙。
<br/>　　“为何？”
<br/>　　“从琴川到关外，只有一条路能走，就是琴川关，现在这条路是莫老大守着。”
<br/>　　“走私的路呢，别告诉我没有？”
<br/>　　“当然有，只是都是山路，道路十分凶险，而且曾经有件事，让戎族不肯和我们做生意了？”
<br/>　　“什么事？”
<br/>　　“当初，莫老大以做生意的名义，把戎族聚在一起，然后给人家来了个一网打尽。靠这件事，莫老大向朝廷请了功，然后戎族也不肯再信任我们。”
<br/>　　“这……也太无耻了吧。”程大雷忽有种前人挖坑，自己要去负责填的感觉。
<br/>　　“所以这条路不是很好走，当初也有人打过这条商路的主意，最后死在戎族手中。”云中龙道：“我劝程当家还是不要搞这件事，我们抢劫不是同样无本万利么。”
<br/>　　“抢劫是涸泽而渔，偶尔搞一次还行，天天抢，会把这块地方搞破产的。”
<br/>　　云中龙三人无语，心道：前些日子你不就是天天抢我们么。
<br/>　　“好啦，正因为很难走我们才要走，不然我怎么搞垄断。”程大雷道：“我已经想好了，先拿三种货物开一开市场，药材，盐巴，茶叶，药材的货源我负责了，其他两项你们三个看着办。”
<br/>　　“私盐我有法子。”云中龙本就是原先此地的私盐贩子，搞这些他不陌生。
<br/>　　“我倒是认识几个茶叶商。”乔鹤道。
<br/>　　“如果能把戎族的玉石，皮草运过来，我有销路，只是这条路真能走通么？”游九楼不确定道。
<br/>　　“世间本来没有路，正因为走的人多了才有了路。”程大雷道：“备货吧，第一批货我亲自走。不过大家注意一点，悄悄的进村，打枪的不要。”
<br/>　　现在程大雷一抬头好像就能看见莫鸣米，很多事情不能做得太招摇，当然，只要升到三级本，自己就不用怕他了。
<br/>　　如今云中龙、乔鹤、游九楼是和程大雷绑在一起的蚂蚱，目前也只能听程大雷的。
<br/>　　程大雷回到山寨后，就开始筹划这条商路的事情，和戎族做生意，他没有什么心理负担，毕竟他也不是帝国人。只是这条路如何走通，还需要仔细思量。
<br/>　　如果能打通这条商路，蛤蟆寨就有了源源不断的收入渠道。有了钱，诸多事情也就好做了。
<br/>　　在筹划这件事的时候，程大雷开始发力升三级本。三级本是在二级山寨的基础上扩张，便不再是简单的在岛上建一座军师堡垒，而是把整座岛变成一座军师堡垒。防御圈从水中就开始建立，土壕，木栏等等防御设施。
<br/>　　这是一个大工程，靠蛤蟆寨的人无法完成，还是像上次一样，只能选择外聘劳力。
<br/>　　……
<br/>　　青叶镇。
<br/>　　黄昏时分，赵三壮将门关了，烫了一把豆叶，从酒铺打了一壶老酒，与几个伙计边喝边聊。
<br/>　　“那边可是又来信了。”赵三壮。
<br/>　　“嗯嗯，这次还是咱们哥几个，那边说得可仔细，不能走漏风声。谁如果走漏了消息，那边是干什么的，我想大家也都知道。”
<br/>　　“明白，明白。”几人纷纷点头。
<br/>　　“若说那边还是不错的，不打不骂，每天一顿肉管够，就算不给工钱也做得来，何况人家还给工钱。”
<br/>　　“所以才不能走漏消息，人知道得一多，这样的好事可就没有了。”赵三壮。
<br/>　　“三哥，嘿，喝着呢。”
<br/>　　门外忽然走进来一人，手里用稻草拎着一条鲜鱼：“我知道三哥在喝酒，特意抓了一条鱼，给哥几个加个菜。”
<br/>　　赵三壮皱起眉头，这人是他的邻居，街里街坊虽然没有撕破脸，但关系无论如何都称不上好。这人怎么会好心给自己送鱼。
<br/>　　“三哥，有好事你可能忘了我啊。”
<br/>　　“什么好事？”
<br/>　　“嘿，三哥你还瞒我，我都听说了。你家小子不是还没成亲么，我家外甥女正好也没婆家……”
<br/>　　赵三壮眼睛一亮，如此倒是不好推却了。他想了想，低声道：“这件事，你可是不能说出去。”
<br/>　　“三哥。”旁边一人道：“我家小舅子也想去……”
<br/>　　“对对，我表弟也……”
<br/>　　赵三壮一楞：“不是不许说出去么？”
