        直到课间休息时间，童筱颖才发现，苏漫雪身上穿着的是一件金丝牡丹旗袍。
<br/>
<br/>        其实，她两身上的旗袍和高跟鞋，是傅夜沉的母亲为她两准备的。
<br/>
<br/>        苏漫雪身上那套旗袍价值上万，而她童筱颖身上这套旗袍只不过才一千块钱左右。
<br/>
<br/>        是人都看得出，她两身份悬殊，就像古时候的嫡庶有别一样。
<br/>
<br/>        所以，这辅导老师把更多的耐心用在了苏漫雪身上。
<br/>
<br/>        不过，童筱颖一点都不介意，反正，只要这辅导老师愿意倾囊相授，不排斥她就行了。
<br/>
<br/>        一整天的补习结束后，傅老太爷询问了两个孙媳妇的补习情况。
<br/>
<br/>        辅导老师虽然有意想要巴结苏漫雪，但是高考又不能作弊，一试就能辨真假的事情，他还是如实告知了傅老太爷，说二少奶奶有望考起临海大学，至于大少奶奶嘛……
<br/>
<br/>        得知实情后，傅老太爷还是有些失望，但也不是完全失望。
<br/>
<br/>        其实，筱颖那孩子虽然不爱甜言蜜语，倒也脚踏实地。
<br/>
<br/>        晚餐过后，童筱颖被傅家的佣人搀扶回了偏院，她趴在圆桌上正在背记英语单词的时候，门外传来了佣人们来往的脚步声。
<br/>
<br/>        童筱颖放下手中的笔，单脚跳着来到门口，看了看外边的情况。
<br/>
<br/>        只见一个男佣正拖着行李箱，往她这屋的旁屋里搬。
<br/>
<br/>        她这个偏院，是一套三合院。
<br/>
<br/>        除了她这间主屋，两边各还有一间空屋子。
<br/>
<br/>        就在童筱颖纳闷是谁要搬过来住的时候，傅夜沉出现在了院门口，他手里提着一个纸袋，朝她走了过来。
<br/>
<br/>        童筱颖一看到傅夜沉，便连忙转身进屋想要关门，谁知傅夜沉突然伸过来一只手，挡了一下。
<br/>
<br/>        门板差点就夹到他的手，吓得童筱颖松了手，后退了一步。
<br/>
<br/>        傅夜沉皱了下眉，将门推开后，不满地质问：“我是瘟神吗？你一看到我就躲！”
<br/>
<br/>        “大、大哥……对不起！我刚刚……没看到你……”童筱颖不知道自己为什么要撒谎，只知道自己要与这个男人避嫌比较好。
<br/>
<br/>        “阿景给你买了一双新鞋，托我带回来给你试试，看合不合脚？”傅夜沉将手中的纸袋向童筱颖递了过去。
<br/>
<br/>        童筱颖一听到是傅昼景为自己买的，心里一阵欣喜，立即接过傅夜沉手中的袋子，从里面拿出鞋盒，并打开鞋盒，提起里面的鞋子穿在了脚上。
<br/>
<br/>        这是一双舒适又精致的白色平底鞋，穿在脚上非常的柔软。
<br/>
<br/>        童筱颖抬起双脚，满眼看着这双鞋子，欢喜不已。
<br/>
<br/>        傅夜沉看着童筱颖脸上那灿烂的笑容，心里也跟着高兴，俊脸上情不自禁地浮泛着欣慰的笑意。
<br/>
<br/>        其实，这双鞋子并非傅昼景所买。
<br/>
<br/>        只不过，傅夜沉是怕童筱颖拒绝，所以才用了阿景的名义。
<br/>
<br/>        门外，苏漫雪也随着佣人一起把自己的行李搬了过来。
<br/>
<br/>        她就不明白了，那梅园里住的好好的，离冷夫人的住处又近，这傅大少爷偏偏要她也搬来这鸟不拉屎的偏院，还要住在童筱颖的隔壁。
<br/>
<br/>        童筱颖穿上新鞋，无意间抬眸看到苏漫雪也搬过来了，脸上的笑意瞬间全无。
<br/>
<br/>        傅夜沉见童筱颖见到苏漫雪时的表情显得有些不开心，顿时有些狐疑起苏漫雪所说的话来。
<br/>
<br/>        她两真的是很要好的闺蜜？！
<br/>
<br/>        如今见这情形，她两这关系，就像……
<br/>
<br/>        傅夜沉看不懂，猜不透，只不过，他这次搬过来，只是因为今早，他见童筱颖两眼有了黑眼圈，心里估摸着她昨晚一定是没睡好，所以才搬过来陪她的。
<br/>
<br/>        但是，他一个人搬过来，难免会让他老妈觉得不成体统，索性，他便让苏漫雪也跟着搬过来。
<br/>
<br/>        正好这偏院有三间屋子，他们三一人一间。
<br/>
<br/>        “我要复习功课了，大哥，好走，不送。”童筱颖回过神来后，便对傅夜沉下逐客令。
<br/>
<br/>        傅夜沉自然识趣，毫不犹豫地便转身离去。
<br/>
<br/>        童筱颖见傅夜沉后脚离开了门槛，立马便把房门给合上了，并落上了门栓。
<br/>
<br/>        傅夜沉听到身后“嘎吱”地关门声和落栓声，不禁停顿了一下，然后无奈地摇了摇头，只身离开。
<br/>
<br/>        苏漫雪见傅夜沉从童筱颖屋内出来，立即迎上去挽住了傅夜沉的臂弯。
<br/>
<br/>        傅夜沉刚想甩开，但又见一旁的佣人还在，这人多口杂，不想给筱颖和阿景带来困扰，便任由苏漫雪挽着一起进了屋。
<br/>
<br/>        “夜沉，我们一起洗澡好不好？我会把你伺候得很舒服的！”苏漫雪却有些得寸进尺，一边撒娇地说道。
<br/>
<br/>        女人的玉手一边顺着傅夜沉的手臂，滑到他的手心，拉起他的手想往自己的胸部上按。
<br/>
<br/>        但他的大手还只是在半空中，就毫不犹豫地从她的手中抽了出来。
<br/>
<br/>        “他们人都走了，你也可以滚了！晚上不管你有事没事，都不要来打扰我！”傅夜沉冷哼道。
<br/>
<br/>        苏漫雪抿了抿唇，识趣地向傅夜沉欠了欠身，退出了他的屋子。
<br/>
<br/>        她明明长着一张和童筱颖一模一样的脸，为什么他就不能像怜惜童筱颖那样怜惜她呢？
<br/>
<br/>        他到底喜欢童筱颖的什么啊？
<br/>
<br/>        一定是她哪里还不够像童筱颖！
<br/>
<br/>        对，她要找出这其中的原因来！
<br/>
<br/>        苏漫雪离开后，傅夜沉关门了，却打开了纸窗，坐在坐榻上看向了主屋那边。
<br/>
<br/>        白炽灯照亮了屋子，透过纸窗可以看到童筱颖的人影还匍匐在桌前看书。
<br/>
<br/>        童筱颖在自己的屋内默写完英文单词，又做了一套数学卷子。但期间，她不忘给傅昼景打了通电话问候晚安。
<br/>
<br/>        她在傅家大庄园里补习的这段日子，傅昼景从未主动给她打过一个电话，发过一条短信，甚至也未回来看过她。
<br/>
<br/>        童筱颖心里孤寂，但又无可奈何，即便有些抱怨傅昼景对她太过淡漠，但是她也只能将这抱怨往肚子里咽。
<br/>
<br/>        其实，她不该奢求得到他的爱。
<br/>
<br/>        本来，他当时娶她，就是因为同情她。
<br/>
<br/>        他替她那毫无血缘关系的赐姓父母还了高利贷，还将她从那小镇里带出来，才有了如今复读重新参加高考的机会，她还有什么不满足？
<br/>
<br/>        殊不知，人都是会有私心的吧！
<br/>
<br/>        童筱颖希望能得到傅昼景的心，希望他作为她的老公，真心地爱她，和她过一辈子。
<br/>
<br/>        她只希望，自己的这个想法，不是痴心妄想。
<br/>
<br/>        童筱颖虽然每天都有跟苏漫雪见面，但是不曾再主动和苏漫雪说过一句话。
<br/>
<br/>        不过，苏漫雪主动搭理她的时候，她还是会微笑着非常客气地回她的话，这完全只是妯娌之间的礼貌。
<br/>
<br/>        苏漫雪也发现童筱颖跟自己生疏了，便故意当着童筱颖的面，跟她抱怨起来：“筱颖，你以前不是这样的啊！”
<br/>
<br/>        “不是这样，那是哪样？”童筱颖反问。
<br/>
<br/>        如果，当初，她把她拿了傅夜沉送给她的玉坠，顶替她当上了傅家大少奶奶这事，直截了当地说给她听，她会一如既往地把她当好朋友并且还会成全她。
<br/>
<br/>        可是，当初她不但没有告诉她，而且还为了一己之私，跟她断绝来往。
<br/>
<br/>        如此爱慕虚荣的一个女人，要让她拿真心相待，当她傻？
<br/>
<br/>        更何况，苏漫雪当初若是真把她童筱颖当闺蜜当朋友，就不会隐瞒这么大的一件事情吧！
<br/>
<br/>        所以，由此看来，苏漫雪并未把她童筱颖当做是真心朋友来对待。
<br/>
<br/>        那么她童筱颖，又为何要在得知真相后，依旧真心待她苏漫雪呢？
<br/>
<br/>        这样矫情的事情，她童筱颖可做不到。
<br/>
<br/>        “筱颖，你是不是后悔把这傅家大少奶奶的位置让给我了？”苏漫雪心一急，抓住童筱颖的手腕，小声质问。
<br/>
<br/>        童筱颖微微偏头瞥了一眼苏漫雪，将自己的手挪开，淡然道：“你想多了。”
