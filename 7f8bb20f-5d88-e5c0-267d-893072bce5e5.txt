　　逐鹿之野，神魔战场。
<br/>　　狂风怒吼，阴雨淅淅，大江奔腾，浊浪排空，混杂着人的呼喊声，惨叫声，厮杀声，兵器的交锋之音，神通的爆炸声，逐鹿战场渐渐演变成了一方人间地狱。
<br/>　　雨借风势，让雨变得更加犀利，风助雨威，战场变得风雨交加。
<br/>　　“诸位，谁愿替我破了对方那滔天雨势？”
<br/>　　黄金战车上，轩辕手持圣剑，眼神冷厉，向着四周问道。
<br/>　　“我来！”
<br/>　　话音刚落，却听道一句坚定的声音响起，众仙看去，待看到来人，眼中满是惊异之色，竟然是龙族的应龙。
<br/>　　“龙族镇压四海，区区洪水想来是挡不住应龙将军？好，那我便命你去破了那雨势，擒住雨师。”公孙轩辕看到应龙走出，眼睛一亮道。
<br/>　　龙族虽然落寞了，但好歹也是远古称霸洪荒的大族，这么多年过去了，谁知道他们还隐藏着什么，有应龙去对战雨师，想来没有多大问题。
<br/>　　“末将领命！”
<br/>　　应龙虽然是龙族，但同时也是轩辕手下大将，帮助轩辕多次战争，也立下不少战功，如今要对付九黎一方的雨师风伯，他自然也没有推辞。
<br/>　　“应龙哥哥，我来助你！”
<br/>　　就在此时，一道清脆而充满炙热的声音响起，紧接着，从轩辕一方众将之中，走出一位身穿红衣，满头红发的女子，那满头红发上还时不时的冒出丝丝炙热的火花，女子面容姣好，妩媚多情，正是轩辕的义女女魃。
<br/>　　女魃又名旱魃，是公孙轩辕收下的一位义女，跟随轩辕多次大战，只是后来长病不起，有上古神兽“犼”的三分魂魄入侵，占据女魃的肉体，一番意志纠缠交战，并与她的三魂七魄融合，之后才成了旱魃。
<br/>　　旱魃一出，赤地千里！
<br/>　　因此她给人带来巨大的恐慌，但因为女魃得以保存一点神识，所以她并没有展开大规模的灾难破坏。
<br/>　　应龙看了一眼女魃，两人眼中闪过一丝柔情，随后在女魃那坚定的目光下，他点头道了一字：
<br/>　　“好！”
<br/>　　女魃顿时脸上露出微笑，她踏步走出，与应龙一同飞向高空，任由它风吹雨打。
<br/>　　“昂——！”
<br/>　　应龙变幻出龙族本体，背生双翼，鳞身骨棘，头大而长，吻尖，鼻目耳皆小，眼眶大，眉弓高，牙齿利，前额突起，尾尖长。
<br/>　　应龙身缠云雾，脚踏四方，轻轻一游，就仿佛是回到了自己家中，之前那些如洪荒猛兽般汹涌彭拜的洪水立刻变得安顺如绵羊。
<br/>　　“噗通——”
<br/>　　只听噗通一声，应龙尖尾轻轻一甩，那安静的洪水便又再次卷起无尽波浪，浪涛翻滚，卷起千层重浪，这次却是向着九黎一方冲刷过去。
<br/>　　“轰隆隆！”
<br/>　　逐浪排空，卷起千层浪，浊江后浪推前浪，一浪更比一浪浪！
<br/>　　无尽浪层堆叠而起，劲道一道一道的加持下去，天知道那会有多强。波涛翻滚，无尽浪花翻起，卷走无尽生命。
<br/>　　“昂——”
<br/>　　应龙一声咆哮，数十上百道洪水形成一条条巨大的水龙，向着雨师与风伯扑去。
<br/>　　“咯咯咯！”
<br/>　　一道轻笑声突然响起，雨师与风伯暗道一声不好，紧接着，一道轻柔妩媚的声音再次传入他们耳中：“应龙哥哥都大发神威了，奴家怎么也不能落后，你说，是吗？”
<br/>　　那声音轻柔似水，忽明忽暗，忽左忽右，好似能够将一颗千年寒冰都融化似的，同时也让人分不清方向。
<br/>　　“今天，人家就让你们见识见识，什么叫做‘旱魃一出，赤地千里！’，咯咯咯——”
<br/>　　说着，又有一连串的咯咯笑声响起，让人分不清东南西北。
<br/>　　“不好，大家小心！”
<br/>　　战场中，刑天突然喊道，想要上前去阻止，但他已经被广成子拦住了，正与之厮杀在一起，分不开身。
<br/>　　“与我作战，还敢分心，真是找死！”
<br/>　　广成子见到这一幕，眼神一冷，翻手间便是一方大印向着刑天脑袋拍去，刑天手中战斧应了上去，交战在一起。
<br/>　　“怎么回事？你有没有发现什么不对？”突然，战场上有人出声问道。
<br/>　　“我也感觉有些不对，就是说不上来？”一人疑惑道。
<br/>　　“我怎么感觉越来越热了？”
<br/>　　“是啊，怎么回事？”
<br/>　　“啊，热死了，我受不了了！”
<br/>　　“啊！我怎么会着火了，怎么回事？啊，不——”
<br/>　　“不——”
<br/>　　“……”
<br/>　　只是转瞬间，战场上便有数千勇士感觉自己身体冒火，之后怎么也不能扑灭，最后活活被烧死。而且随着时间的推移，被烧死的勇士也越来越多，死得连灰烬都没有剩下。
<br/>　　“水！我要水！”
<br/>　　有人呼喊道，紧接着投入到洪水之中，想要以此来熄灭身上的火焰。
<br/>　　“没用的，区区凡水，又怎能抵挡我的旱魃之火呢。咯咯咯……”
<br/>　　果然，那些跳入水中的勇士，依旧没能逃脱被活活焚尽的凄惨下场。就算是跳入水中，那诡异的火焰依旧在燃烧着，好似不会熄灭似的。
<br/>　　轰隆隆！
<br/>　　战场上的洪水开始以肉眼可见的速度迅速蒸发，天上的雨不再开始下了，大地开始变得开裂，石子变得赤红，方圆千里，赤红一片。
<br/>　　大地在坍塌，石子变成细沙，之后连细沙都被烧成了粉末。甚至最后连虚空都被炙烤得通红一片。
<br/>　　只不过几个时辰，方圆千里，便再也没有一个生物存活。方圆千里，赤红一片，寸草不生。
<br/>　　旱魃一出，赤地千里！
<br/>　　果然很强大，可惜，这个神通不分敌我，虽然敌人被杀了，但是自己一方的人也全部死在里面。这就是代价。
<br/>　　“雨师，你说你若是没了雨，还能叫雨师吗？”
<br/>　　突然，一道轻微的声音传入雨师耳边，紧接着，雨师只感觉自己的身体一紧，还没来得及做出反应，便被拉到了地底。
<br/>　　轰隆隆！
<br/>　　紧接着，地底传出强烈的动荡声，地面不断起伏，爆炸声不断，更有火光冲天而起。只是半刻钟，那声音便消失不见，仿佛战斗已经停止。
<br/>　　“噗！”
<br/>　　眨眼间，一道赤红的身影出现在虚空中，她手里提着一具尸体，正是之前大巫雨伯的尸体。
<br/>　　与此同时，应龙那里也结束了战斗，他的龙爪穿透了风伯的心脏，灭了他的神性，已经死了不能再死了。
<br/>　　两人站在虚空之中，对视一眼，突然心有灵犀地相视一笑。
<br/>　　但正在此时，变故陡生，一道浓郁的血煞的刀气突兀地破空而出，斩向虚空中的女魃，并牢牢锁定她。
<br/>　　“不——”
<br/>　　应龙突然发出凄厉的惨叫声，在应龙的眼中，那一向调皮的女魃突然血洒长空，由虚空之中坠落下去，他来不及多想其他，急忙飞了过去，接住落下的女魃。
<br/>　　而女魃原先所立的虚空，此刻却站着一个九尺大汉，他手持一柄魔刀，冷冷地盯着众生，目光之中，一片血色……
<br/>　　……
