　　下方围观的众人，看到天空上，陡然出现了一个大鼎，将叶尘和叶无伤全部吸了进去，顿时一个个目瞪口呆。
<br/>　　很快，
<br/>　　一道惊恐的咆哮之声，从里面传来，
<br/>　　“小野种！你对我做什么？”
<br/>　　随着叶无伤身上的妖龙力量，被奥维斯不断的吞噬，叶无伤也终于恢复了几分理智，当即便想要逃离吞天鼎。
<br/>　　“想跑？晚了！”
<br/>　　叶尘直接拦住了叶无伤的去路。
<br/>　　“闪开！”
<br/>　　叶无伤暴喝一声，狠狠一拳砸了过来，叶尘不躲不避，也一拳迎了上去。
<br/>　　轰隆！
<br/>　　两人双拳相交，顿时发出一道爆炸般的撞击声，连同整个吞天鼎都轰轰作响。
<br/>　　叶无伤登时发现，他一催动功力，体内的妖龙之力，就在迅速的流逝！
<br/>　　“该死！这到底是什么东西！”
<br/>　　叶无伤又惊又骇，顿时更加想要逃离吞天鼎。
<br/>　　可是叶尘又岂会让他如愿？
<br/>　　砰砰砰！
<br/>　　一眨眼的功夫，两人已经又交手了数十招，整个吞天鼎，都因为两人的打斗，而被震的飞来飞去，忽大忽小。
<br/>　　“小野种！你给我，滚开！”
<br/>　　眼看这体内的妖龙之力越来越少，叶无伤顿时大急，忽然从腰间取出了一把剑柄。
<br/>　　是的，只有剑柄，却没有剑身。
<br/>　　不过很快，
<br/>　　唰！
<br/>　　自那剑柄之中，猛然爆发出一道璀璨的赤色光芒！
<br/>　　量子剑？
<br/>　　叶尘顿时一眼便认出，叶无伤突然拿出来的武器，赫然正是米国超能局最顶级的武器，量子剑！
<br/>　　据说此剑，能够将万物分解成量子，威力奇大无比，不过之前据说还处于试验阶段，并没有真正造出，虽然有相关图片流传于世，可是从来没有人见谁真正使用过。
<br/>　　叶尘没想到，这叶无伤的手里，居然会有一把。
<br/>　　“死！”
<br/>　　叶无伤手握量子剑，一声暴喝，狠狠劈落下来！
<br/>　　叶尘手掌一翻，也几乎同时取出草雉剑，当即抵挡了上去。
<br/>　　轰！
<br/>　　双剑相交在一处，叶尘顿时感觉到一股强大的冲击力，
<br/>　　蹬蹬蹬！
<br/>　　叶尘和叶无伤两人，几乎同时倒退了数步，算是旗鼓相当。
<br/>　　可是很快，叶尘惊愕的发现，自己手中的草雉剑，竟然出现了一个缺口！
<br/>　　“好强！竟然连圣器都险些抵挡不住！”
<br/>　　叶尘不由得脸色微微一变。
<br/>　　对方的这个量子剑，单单论杀伤力，恐怕已经可以媲美仙器！
<br/>　　这米国超能局，不愧是号称掌握着地球上，最顶尖的科技！
<br/>　　而这时，叶无伤已经再次抡起量子剑，又冲了上来！
<br/>　　铿铿铿！
<br/>　　一连三剑！
<br/>　　叶尘接住两剑之后，
<br/>　　咔！
<br/>　　双剑第三次相交，叶尘手中的草雉剑，顿时便在也抵挡不了，竟然从中间断裂开来！
<br/>　　多亏叶尘反应及时，在草雉剑断裂的同时，就已经闪身躲开。
<br/>　　“啊！”
<br/>　　吞天鼎的器灵，奥维斯忽然发出一道惨叫，显然连它，也有些承受不住量子剑的强大破坏力。
<br/>　　叶无伤看到叶尘的兵器被自己斩断，顿时精神大振，
<br/>　　显然连他也没有想到，波尔送给他的这件武器，竟然有如此强大的威力！
<br/>　　当然，也只有叶无伤这样的实力，方才能发挥出量子剑真正的威力。
<br/>　　否则若换成其他人，连叶尘的速度都跟不上，就算有逆天神兵，也根本伤害不了叶尘分毫。
<br/>　　“死吧！”
<br/>　　叶无伤手提量子剑，再次狠狠一剑斩出！
<br/>　　关键时刻，叶尘手掌一翻，又取出了另外一件圣器，十拳剑！
<br/>　　轰轰轰轰！
<br/>　　然而，双剑再次相撞数次之后，十拳剑竟然也被叶无伤的量子剑斩为两段！
<br/>　　“哈哈哈！”
<br/>　　叶无伤不由得得意狂笑一阵，
<br/>　　“小野种！受死吧！”
<br/>　　眼看叶无伤就要趁胜追击，用量子剑杀掉叶尘，可是就在这时，原本璀璨的量子剑，一下子变得暗淡许多。
<br/>　　“糟了！似乎能量不足！”
<br/>　　叶无伤顿时暗道倒霉，不过这最后一剑，终究还是斩了出去。
<br/>　　“哼！”
<br/>　　叶尘心念一动，圣器八咫镜，已经陡然出现在他的面前。
<br/>　　轰隆！
<br/>　　这一次，量子剑的威力大大减弱，只是将八咫镜周围的金光护罩斩破，八咫镜却丝毫无损。
<br/>　　而叶无伤斩出这一剑之后，那把量子剑上面的光芒，顿时越发黯淡。
<br/>　　叶无伤顿时脸色大变，来不及多想，趁着叶尘还没有稳定身形，当即纵身一跃，已经从仙鼎中挣脱了出去，继而朝远处狂奔而去！
<br/>　　没有了量子剑，体内妖龙之力又消耗了不少，他如今已经没有对付叶尘的底牌，而且再加上叶尘手里，还有仙器吞天鼎，此时又在华夏境内，他若是再不走，只怕生死难料！
<br/>　　叶无伤也是果决之人，自知不敌之后，说逃便逃，一眨眼的功夫，已经飞奔到了千米之外。
<br/>　　可惜，叶尘又岂会善罢甘休？
<br/>　　“想跑？你跑的掉吗？”
<br/>　　叶尘一手收起吞天鼎，同时施展出缩地成寸的神通，瞬间便已经跨越了上千米的距离，然后一掌拍落！
<br/>　　轰！
<br/>　　长达近百米的庞大巨掌，犹如一座巨山，狠狠朝叶无伤砸落下来。
<br/>　　“哼！”
<br/>　　叶无伤冷哼一声，也当即一拳轰出！
<br/>　　轰隆隆！
<br/>　　一拳一掌相撞，天地间顿时发出一道巨响，继而叶无伤整个人，便被叶尘这一巴掌狠狠扇落到了地上！
<br/>　　砰！
<br/>　　地面出现一个巨大的深坑，而叶无伤整个人则完全没入地底。
<br/>　　“给我，出来！”
<br/>　　还没等叶无伤完全反应过来，叶尘虚手一抓，空中那道金色巨掌，顿时便探入那深坑之中，将气息萎靡的叶无伤，一把擒了出来。
<br/>　　“空间封锁！”
<br/>　　叶尘当即又施展出空间法则，将叶无伤禁锢在空中无法动弹。
<br/>　　“吼！”
<br/>　　惊怒无比的叶无伤，顿时大吼了一声，想要再次调动体内的妖龙之力。
<br/>　　然而，叶尘早有防备，在施展出空间封锁之时，手中的吞天鼎，已经飞至叶无伤的头顶上方。
<br/>　　“吞噬！”
<br/>　　随着叶尘一声暴喝，吞天鼎之中，顿时飞速旋转，同时鼎内出现一股强大的吞噬之力！
<br/>　　片刻之后，
<br/>　　嗖！
<br/>　　自叶无伤的体内，顿时窜出一根晶莹剔透的骨头，赫然正是那嗜血妖龙的精骨！
<br/>　　“不！！”
<br/>　　失去了妖龙精骨，叶无伤整个人的气息，顿时急速衰落，脸上也终于露出了浓浓的惶恐之色。
