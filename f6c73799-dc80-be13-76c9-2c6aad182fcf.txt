罗富来了，有人给李玉兰撑腰了，场面迅速变得不一样了。

<br/>
<br/>        罗富是一个胖子，脸上有生意人的精明，他拍了拍李玉兰的手，“振国，我听说最近有人欺负了玉兰，让玉兰不开心了，我把干女儿嫁给你，是让你带给她幸福的，而不是让她受委屈的，你明白这个意思么？”
<br/>
<br/>        夏振国一僵，他刚向夏夕绾允诺过了，现在又被罗富来了一个下马威，这脸简直是左右开弓，啪啪打脸。

<br/>        夏振国特别封建爱面子的人，现在也顾不得这些了，只想让罗富注资先解决了资金短链。

<br/>        “是干爹，我会对玉兰好的，我和玉兰的感情绝对不会受外界那些谣言的影响，这一场结婚纪念日就是最好的证明。”夏振国闭着眼睛豁出一切道。
<br/>
<br/>        罗富看向李玉兰，“玉兰，你觉得呢？”
<br/>
<br/>        李玉兰的尾巴快要翘上天了，之前她有多郁闷，现在就有多得意，她看了夏振国一眼，勉强道，“看他以后的表现吧。”
<br/>
<br/>        “振国，听到没，以后要好好表现，让我的干女儿满意。”罗富发话道。
<br/>
<br/>        “是是是。”夏振国狂点头。

<br/>        那些富太太们脸色不是太好看。

<br/>

<br/>
<br/>        夏小蝶没想到夏夕绾会这么说，她就觉得一拳头砸在了棉花里，没使上劲。
<br/>
<br/>        这时大厅里响起了悠扬的钢琴声，到了跳舞的环节了。

<br/>        作为主角的夏振国走上前，绅士的伸出了自己的手，“夫人，能请你跳支舞么？”

<br/>
<br/>        李玉兰直接看向罗富，“干爹，这第一支舞我跟你一起跳吧。”

<br/>        虽然罗富名义上是干爹，但两个人的关系可不正当，看着李玉兰妩媚的眉眼，加之两个人很久没见了，罗富迅速伸手搂住了李玉兰，“好，就依你。”
