　　第708章
<br/>　　事实上，崔相在几天前就已去世。这位在朝堂斗了大半辈子，很少败绩的老人，终究不是岁月的对手。
<br/>　　即便在相府，这件事也只有几个人知晓。相府势力很大，有诸多世家依附。但崔相压得住这些世家，崔云间未必压得住。这几天他忙得焦头烂额，便是为了收拢力量，以保证相府的势力延续。
<br/>　　却没料到，正这个时节，堂堂帝国天子打算做一次亡命徒。一柄尖刀，扎得正是相府虚弱时。
<br/>　　看到这口棺材，李乐天瞬间明白了。他愣了愣，似乎是因为自己没能亲手杀了老人而觉得遗憾，也似乎是因为对这个老人的无声无息离开而感慨。李乐天也不得不承认，这个老人对帝国是有功的。
<br/>　　这念头一闪而逝，今日的乱局尚未结束，不是瞎发感慨的时候。外面喊杀声震天，城防军也正向这边赶来。这是一场你吃我，我吃你的死局。形势目前对李乐天有利，但不能证明他已经是胜方。
<br/>　　“砸破棺材，将老贼的脑袋砍下来。”
<br/>　　七八柄兵器都砸在棺材上，只听咔嚓一声，棺木碎裂，里面浮现一个老人穿着寿衣的老人尸体。
<br/>　　在火光的照耀下，他脸色苍白，众人看到这一幕，竟都下意识的往后退了一步。
<br/>　　这个老人执掌帝国朝政多年，虽然手无缚鸡之力，但挥挥手就能搅起一片风雨，决定上万人的生死。
<br/>　　面对这个已身死的老人，众人竟怯懦不敢上前，似乎担心他会突然活转过来。
<br/>　　李乐天也被一种可怖的气氛缠绕，不过别人可以怕，他不能怕。从身边武士手中接过一把刀，他迈步走过去，一手抓住老人的头发，刀摁在他的脖子上，咔嚓一声，将整个脑袋砍了下来。
<br/>　　鱼龙卫的佩刀是帝国工匠坊精心冶炼，此刻李乐天方才体味到，这种刀……好快。
<br/>　　他提着一颗发枯的脑袋走出房门，眼见火光冲天，杀声四起。
<br/>　　“陛下，城防军冲过来了。”有手下飞跑着过来。
<br/>　　“来得好。”
<br/>　　李乐天铁青着一张脸，他今天倒要看看，城防军谁敢向自己动手，莫非要弑君造反不成。
<br/>　　双方在相府大门口撞上，隔着大门往外看，黑压压一片，长矛林立，此刻堵在门口，带来泼天杀气。
<br/>　　大门洞口，李乐天迈步往外走，手下赶紧阻拦。李乐天喝了一声滚开，站在大门正当中。
<br/>　　“朕乃帝国天子，执掌帝国十三州一百零八城，那个逆臣贼子胆敢造反。”
<br/>　　他一手提着匹夫剑，一手提着人头，身上鲜血浸满龙袍，立在众人面前，不怒自威。
<br/>　　城防军官兵怯懦不敢近前，他们之中多有世家子弟，大小将官都是世家安插在其中。可说到造反二字，他们都没有这个胆量。
<br/>　　李乐天举起崔相的头颅，高声道：“贼首已经伏诛，此人密谋造反，罪大恶极，今日唯诛首恶，其他不论，放下兵器，我算你们无辜。”
<br/>　　一片喧哗声在城防军当中响起，他们受崔相控制多年，早已将他视作高高在上。但如今，那个高高在上的老人已经被杀，自己这些墙头草有必要去卖命嘛。
<br/>　　这城防军的首领，名唤崔云生，乃是崔相的第六子，崔云间的兄弟。
<br/>　　如此重要的职位，崔相自然要交给自己人。此刻见到父亲的头颅，他哇呀呀一声大叫，口中咆哮道：“尔胆敢杀我父，与吾纳命来。”
<br/>　　李乐天仰天大笑：“朕乃天子，朕想要谁死谁便须死，何况是谋反的逆贼，尔真当这天下姓了崔么？”
<br/>　　百里胜等人护在李乐天面前，为李乐天提防暗箭。天子可以耍狂，大家却不能让他出事，一旦他有个闪失，今天这场乱局便注定失败。
<br/>　　崔云生咬住牙齿，脑海中急速思索着。今天李乐天突然动手，他也是完全没想到，但凡有个提防，也不至于让事情落到这种地步。
<br/>　　现在已经是这幅局面，其他人可以做墙头草，自己却没能力摇摆。必须舍身一战，拼个鱼死网破。
<br/>　　妈的，今天就是今天，杀了面前这李乐天，自己明天就是帝国天子。
<br/>　　皇帝轮流做，明天到我家，你姓李的都可以，我姓崔的为什么不行。
<br/>　　“来人！”崔云生忽然喝了一声：“李乐天滥杀无辜，屠戮功臣，已不配做这天子。今日与我杀了他，明日你们便是从龙重臣。”
<br/>　　有人怯懦不敢近前，有人眼里却出现炙热的光芒。从来富贵险中求，苦打苦熬的过日子，什么时候是个头。重新洗牌，也许自己也能搏一场富贵。
<br/>　　“怎么，真敢造反么？”李乐天心中并不轻松。真如果城防军一拥而上，自己绝没能力阻挡。
<br/>　　擒贼先擒王，必须杀掉崔云生，否则事情不妙。
<br/>　　马超虽然有些生猛，但他并不是个憨夫。此刻怪叫一声，身子冲了上去。
<br/>　　“我去取他性命。”
<br/>　　他倒提大枪，迈步向前，长枪由下向上刺去。
<br/>　　“找死。”
<br/>　　崔云生冷喝一声，用手中大刀将其挑开。双方战在一起，一瞬间竟难分伯仲。
<br/>　　崔家也有高手，只是仗着崔相的地位，一入官场便是平步青云。所以纵然有些本事，有时候也显不出来。
<br/>　　这崔云生竟然能和马超交手二三十招不落下风，实在是令人刮目相看。
<br/>　　李乐天颦起眉头，此战必须快。一旦时间拖长，城防军的野心也会生长。假若马超再败了，事情就当真不妙了。
<br/>　　“谁去助马将军一臂之力。”
<br/>　　有李乐天这句话，事成之后，马超至少也是一个将军的职位。卢俊义眼珠一转，心里也明白建功立业就在今日。
<br/>　　“贼子休狂，吾来杀你。”
<br/>　　他提着大枪跃了出来，一枪刺向马上的崔云生。
<br/>　　“喂，用不着你！”
<br/>　　马超口中大喝，手里动作也在加快。谁也知道，能杀了崔云生便是头功一件，大家都在抢功，皆拿出了看家本事。
<br/>　　面对一个马孟起，崔云生已经是狼狈不堪。何况是两个绝世高手同时想杀自己。
<br/>　　二人几乎同时拿出看家本领，两柄枪，几乎同时扎中崔云生。
<br/>(本章完)
