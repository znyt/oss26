　　“这一次攻打，只是试探？”
<br/>　　听见楚风眠的话，寒月漓，洛汐，云老都是一愣，旋即脸上都是无比凝重起来，眼神之中露出几分惊骇。
<br/>　　这样厉害的攻击，居然只是一次试探？
<br/>　　要知道这一次攻打，如果不是楚风眠及时赶来，只怕是剑道门都承受不住，只怕都是灭门之灾了。
<br/>　　这一次都只是试探，那七大宗门的实力，到底是多么恐怖？
<br/>　　不管是寒月漓，还是洛汐，都不清楚七大宗门的真正实力。
<br/>　　毕竟在如今这个时代之中，七大宗门，已经远没有曾经那般强大了。
<br/>　　可是论起真正的强者，现在七大宗门之中，并不缺乏，万年之间，对帝尊境来说，并不算什么。
<br/>　　万年时间，只怕是曾经七大宗门那些强者，还有不少都存活于世，这些人才是真正的麻烦。
<br/>　　楚风眠可是万年之前的人，自然知晓那个时候七大宗门的可怕，别说像是盘石，纯剑长老这等九劫古帝巅峰的存在。
<br/>　　半步天人境的人物，七大宗门内都未必没有。
<br/>　　“七大宗门现在的真正实力，只怕是九劫古帝，加起来要有上百位，不亚于盘石等人实力的强者，也至少有着二十人之上。”
<br/>　　楚风眠眉头一皱，冷声说道。
<br/>　　“甚至可能还会有半步天人境级别的强者。”
<br/>　　楚风眠说出的，还是最好的结果，因为在屠人间的记忆之中，六道魔门之中，如今六位新的尊主，真正的实力，都跟屠人间在伯仲之间。
<br/>　　这就证明，单单是六道魔门之中，堪比九劫古帝之中无敌的强者，都有着足足五位之多。
<br/>　　而且这还只是明面上的实力，暗地里，不管是那一方势力之中，都会有些外人很少知晓的老古董。
<br/>　　如果算上七大宗门的底牌，隐藏起来的老古董，强者的数量，还要比起楚风眠说的，还要翻上一倍。
<br/>　　“这么强横？”
<br/>　　洛汐，寒月漓对视一眼，都是从对方的眼神之中露出不可置信的神色。
<br/>　　数百位九劫古帝，这简直比起圣龙秘境的九劫古帝，都要多得多了，而且盘石这样的人物，都有数十人。
<br/>　　瘦死的骆驼比马大。
<br/>　　现在的九域，无论如何也是曾经的圣域之一，就算是在摔落，也有着归墟天，万宝天这等宝藏秘地。
<br/>　　单单是得到曾经九域强者的传承，就足以诞生出一大批的强者出来。
<br/>　　虽然整体实力，也许不如大罗天星域，但是现在九域之中的强者加起来，数量未必不如大罗天星域，甚至可能还要更强。
<br/>　　而且这还不是最遭的，在屠人间的记忆之中，楚风眠居然是发现了圣龙秘境的人，跟七大宗门在接触。
<br/>　　似乎是这样，才让楚风眠的身份，彻底的暴露了出去。
<br/>　　圣龙秘境之中，想要杀了楚风眠的人，也不知有着多少，要是圣龙秘境的强者也参与进来，那可就更加的麻烦了。
<br/>　　“这种实力，我们硬拼，没有机会，那如果逃呢？”
<br/>　　寒月漓脸色过了一会，还是相信了楚风眠的话，她虽然不知道楚风眠的来历，可从七大宗门的态度看来，只怕是楚风眠的来历，绝对不凡。
<br/>　　对七大宗门的了解，也比起她知道的多得多。
<br/>　　硬拼不成，那便是只有退，这一条路。
<br/>　　“一旦我们离开，剑道门将毁于一旦，而且剑道门一旦被毁，在想要建立起来，几乎不可能了。”
<br/>　　楚风眠听到寒月漓的话，摇了摇头道。
<br/>　　“而且离开九域，也不代表麻烦会消失，逃得了一世逃不了一世。”
<br/>　　“这一战，并非没有胜算，我需要闭关几日，只要是我可以踏入九劫古帝境界，哪怕是有着半步天人境的强者，我也有着一战的自信！”
<br/>　　楚风眠语气冰冷无比，只有他足够强大，才足以硬撼七大宗门。
<br/>　　“不过七大宗门的攻打，也就会在这几日，月漓，洛汐，我闭关之后，七大宗门来攻打，就由你们来拖延，燕皇青梦，会助你们一臂之力。”
<br/>　　楚风眠平静的开口道。
<br/>　　闭关突破九劫古帝，已经是事不宜迟了。
<br/>　　只有踏入了九劫古帝，楚风眠才真的有跟半步天人境强者一战的资本。
<br/>　　这一次楚风眠出手，以雷霆之势轰杀了盘石，纯剑长老等人，七大宗门的人，也肯定会知晓这些。
<br/>　　所以下一次，一旦是七大宗门要来攻打剑道门，只怕是可能会有半步天人境的强者前来出手。
<br/>　　楚风眠在没有踏入九劫古帝之前，面对半步天人境的强者，最多也就是只能自保，拖延住。
<br/>　　可要是想要真正击败半步天人境的强者，楚风眠只能够是踏入九劫古帝，才有可能。
<br/>　　现在楚风眠虽然是有着自信，肯定是可以踏入九劫古帝。
<br/>　　但是到底是要花费多少的时间，楚风眠自己都不确定。
<br/>　　毕竟楚风眠突破一重境界的难度，比起其他武者要困难的多，是同境界武者突破难道的数万倍，数十万倍。
<br/>　　而这段时间，一旦七大宗门前来攻打，楚风眠也不可能分心，突破失败的代价，是楚风眠无法承受的。
<br/>　　所以只能够是靠寒月漓，洛汐等人来抵挡。
<br/>　　刚刚楚风眠急忙凝炼护山大阵，重塑剑道门门户，加固剑道门的防御，就是为了这一点，可以用护山大阵，来拖延一段时间。
<br/>　　“放心，全部交给我们。”
<br/>　　寒月漓跟洛汐，都是点了点头，她们也知道，一旦是楚风眠在突破的关键时刻，无论如何也不能够打扰到楚风眠。
<br/>　　“樊老，如果有危机，还请你也出手相助一下。”
<br/>　　楚风眠脑海之中，突然对着面前开口道，一尊老者的身影，便是突然现身，出现在楚风眠的面前。
<br/>　　“这位是樊老，如果有半步天人境的强者出现，可是请樊老出手相助。”
<br/>　　樊老在楚风眠给予大量圣品灵石之下，如今实力也恢复了不少。
<br/>　　这一次楚风眠是要闭关突破，去冲击九劫古帝的境界，无论如何也不能让人打扰到他，所以只能请樊老出手来坐镇了。
