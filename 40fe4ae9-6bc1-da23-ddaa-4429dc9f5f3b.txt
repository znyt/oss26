　　第334章
<br/>　　攻其不备，出其不意。若时间拖延下去，等城内守军赶到这里，堵在将军府门口，程大雷就要面临腹背受敌的情况。
<br/>　　所以，必须快。
<br/>　　大斧砸了过去，鲜血淋漓，溅在程大雷脸上，他将脸上的血一抹，口中大吼一声：
<br/>　　“杀！”
<br/>　　直面程大雷的人恍惚如见到一只下山猛兽，有些人下意识颤抖，膝盖竟有些发软。
<br/>　　两只队伍的厮杀已经开始。
<br/>　　程大雷是杀意弥漫，而莫鸣米此刻还沉浸在失神中，如今才慌张回过神来。一快打一慢，在战场上就是数十条人命的代价。
<br/>　　看着程大雷在战场上横冲直撞，杀人如麻。一腔血勇撞上来，手握大刀向程大雷冲来。
<br/>　　“姓程的，吃我一刀。”
<br/>　　程大雷眼睛一亮：“怕你不来呢。”
<br/>　　莫鸣米一跃而下，大刀向程大雷脑袋劈来，程大雷挥起斧头，硬接了对方一招。
<br/>　　双方战在一起，打得都很凶，偶然卷入这场战斗的人，都被程大雷一斧敲掉。
<br/>　　鲜血和死亡唤醒了莫鸣米体内某些东西，当初他也是一点一点打下的江山，但过去很长时间，莫鸣米已经成了坐不垂堂的贵人，一双手已不知有多久没握过刀。
<br/>　　可今天这一战关乎生死，身外的一切都被抛下了，只关注面前敌人的脑袋。
<br/>　　程大雷也不敢大意，此战的关键便是能否杀掉莫鸣米。二人一交手，程大雷就发现这莫鸣米半点不弱，若自己有一丝松懈，可能被杀的就是自己了。
<br/>　　战场之上，反败为胜，逆袭翻盘的事情太多了，一点都不会令人奇怪。
<br/>　　二人正打得凶险时，一柄刀插了进来，替程大雷接住了莫鸣米。
<br/>　　是樊梨花。
<br/>　　“他交给我了。”
<br/>　　“好，我替你压阵。”程大雷挥动斧头将面前的人驱开。
<br/>　　“程大雷，好出息，让一个女人打头阵。”
<br/>　　“别废话了，你快死了，快想个终场台词吧。”程大雷。
<br/>　　莫鸣米气得青筋暴露，他之所以如此愤怒，还有另外一个原因。那就是樊梨花比程大雷打得更凶。
<br/>　　绣绒刀抖开，刀光如梨花，眼前只见刀光不见人影。况且，还有程大雷在旁边压阵，偶尔抽冷子来上一斧头，令莫鸣米防不胜防。
<br/>　　莫鸣米和樊梨花交过手，也知道对方的厉害。所以心中对程大雷恨意更盛，这个人太无耻了。
<br/>　　“将军，看飞机！”程大雷。
<br/>　　“快看，有神仙呐。”
<br/>　　程大雷偶尔喊这一嗓子，搞得莫鸣米心慌意乱，心中对程大雷越来越恨。猛地一咬牙，避过樊梨花的刀锋，冲着程大雷脑袋砍去。
<br/>　　程大雷接住了这一刀，用斧头缠住莫鸣米的刀。此刻樊梨花的绣绒刀砍来，莫鸣米以一敌二，瞬间落了下风。
<br/>　　“快说，最后一句啦。”程大雷打得虎虎生风。
<br/>　　“我把……”
<br/>　　锵啷一声，程大雷佩剑出鞘，快若闪电的一剑，洞穿了莫鸣米的喉咙。
<br/>　　莫鸣米睁大眼睛，一身力气被这口剑摁住，随着程大雷的剑拔出，一腔鲜血喷了出来。
<br/>　　“啧，说了最后一句了，你也不好好说。”
<br/>　　程大雷撇撇嘴，一手握剑，一手拎着斧头，目光看过去，见将军府内的乱战已经快要结束，还是己方占了上风。随着莫鸣米被杀，此地的战斗已经分出胜负。
<br/>　　“莫鸣米已经被杀，降者免死！”程大雷一斧砍下莫鸣米脑袋，举着他的脑袋大喊。
<br/>　　众人只看见程大雷站在高处，满身鲜血，刚才还威风八面的莫鸣米，现在一颗头颅被程大雷拎在手中。
<br/>　　众人瞬间不知该如何是好，有人已放下兵器投降，有人握着兵器脸上露出一丝茫然，有人还在战斗。
<br/>　　“我乃明帝钦封军官，再敢有反抗者，一概以反贼论处。”程大雷大喝。
<br/>　　反贼……这些人此刻一头雾水，不该你们才是反贼么。
<br/>　　剩下小股反抗者被围而杀之，剩下的人都举手投降，跪在地上。
<br/>　　时间紧，任务重，程大雷也来不及高兴。他用一百人留守将军府，也顺便看着这些俘虏。对于其他人，自己将命令吩咐下去，程大雷打开随身的包裹，去后面换衣服了。
<br/>　　将军府外的战斗，依旧是一团乱麻。秦蛮此刻已攻占马圈，将栅栏放开，用火把驱赶着马匹。
<br/>　　群马奔腾，开始在关内乱窜，所过之处，烟尘鼓荡。
<br/>　　此刻程大雷的人从将军府中窜出，所过之处，响起他们吼声。
<br/>　　“莫鸣米已死，降者免死！”
<br/>　　赵子龙、关鱼、张肥、高飞豹、秦蛮等人皆凶恶得如狼似虎，他们也不恋战，遇到难啃的骨头，直接也就放过了。目的不是杀人，是将琴川关搞乱。
<br/>　　这五人的强悍本就无人可挡，听到这声音更加心慌意乱，有些人直接放下兵器跪倒，而那些迷茫着直接被一刀杀了。
<br/>　　战斗从深夜开始，一直打到黎明，还未分出胜负。琴川关的人聚集在一起，占了一处院子，据坚而守。
<br/>　　他们的指挥官名叫叶历，很是有几分本事。关鱼带人轮流冲了好几拨，都未能打下来。
<br/>　　此刻程大雷骑着高头大马赶到，骨碌碌把一颗人头丢进里面。
<br/>　　“姓叶的，出来说话。”程大雷马上大吼。
<br/>　　那颗人头自然就是莫鸣米，见到这颗脑袋士气已经丧了大半。
<br/>　　院门打开，一员将站在门前，横眉怒目，口中大骂：“狗贼，要打就打，不用废话，我绝对不会向你一个山贼投降。”
<br/>　　“什么叫山贼。”程大雷气得大叫：“仔细看清楚我身上穿的什么，我是御赐七品安边校尉，守琴川关，谁是贼，你们才是贼！”
<br/>　　叶历这才注意到，程大雷今天好模样，头戴高冠，身披黑袍，的确是帝国的七品官袍。
<br/>　　“呔，告你们这些反贼听真，莫鸣米已死，其他人不问。再敢有负隅顽抗者，一概罪诛九族，把你们的祖宗都挖出来，挫骨扬灰。”
<br/>　　叶历的确有些摸不准程大雷来路，一时间却也不知该怎么办。
<br/>　　“你们真以为自己还能守得住么？”程大雷冷哼一声：“数十之内，再不投降，与我放火烧了他们。”
