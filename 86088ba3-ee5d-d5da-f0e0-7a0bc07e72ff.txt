　　“什么？”
<br/>　　宫殿之内的众人，都是纷纷一愣，旋即纷纷倒吸了一口凉气。
<br/>　　“这怎么可能？”
<br/>　　“陈柏被打破肉身？”
<br/>　　“一招轰杀？”
<br/>　　“这可是一位七劫古帝，古帝都能被一招轰杀？”
<br/>　　不管是陈家的人，还是北邙学院的高层，都是当即愣住了，这可是一位七劫古帝。
<br/>　　在阎罗星域之中，七劫古帝可也都是大人物了，在陈家也是长老一辈的人物。
<br/>　　在九域更不用说，七劫古帝几乎都是无敌的人物了，在九域剧变之前，只有一些古老的势力之中，才有这样的强者。
<br/>　　现在楚风眠居然是出手，一招之内，轰杀古帝？
<br/>　　在这龙爪落下之后，三道身影，步入到了宫殿之中。
<br/>　　在这宫殿之上布满的无数禁制阵法，对楚风眠来说，根本起不到任何的作用。
<br/>　　“楚风眠！”
<br/>　　“此人怎么会这么厉害！”
<br/>　　那被龙爪打碎肉身的陈柏，不可置信的大喊道。
<br/>　　他本来以为楚风眠最多就是九域的土著，根本算比不是什么人物，可是他没有料到楚风眠的实力居然是达到了如此强大的地步。
<br/>　　面对楚风眠的龙爪轰击，他几乎都做不到任何的抵挡，肉身就已经是被打碎。
<br/>　　转眼之间，在宫殿之中便是就化为了一团血雾。
<br/>　　“蝼蚁一样的废物，也敢出手？”
<br/>　　楚风眠目光盯着那化为一团血雾的陈柏，冷哼一声，大手一挥，那一团血雾就被楚风眠收入到了手心之中。
<br/>　　只要是楚风眠愿意，可以将这陈柏，一下子捏死。
<br/>　　“小辈，住手！”
<br/>　　“放肆！”
<br/>　　“放开陈柏！”
<br/>　　几名陈家古帝，这时才反应了过来，大声的怒喝道。
<br/>　　“楚风眠，你在做什么？这几位陈家的人，可是我们北邙学院的贵客！”
<br/>　　那北邙学院的院长，突然站了出来，看着楚风眠质问道。
<br/>　　“还不速速放了陈柏长老，赔礼道歉！”
<br/>　　“楚风眠，你如果还是北邙学院的弟子，就赶紧放开陈柏长老！”
<br/>　　那一名北邙学院的长老也焦急的开口道。
<br/>　　“是吗？贵客？我看这些人，可都没安好心，敢在北邙学院之中耀武扬威？外人该有外人的态度。”
<br/>　　楚风眠嘴角露出几分冰冷笑容道，扫过了北邙学院一行人，不屑的开口道。
<br/>　　“诺大的北邙学院，就被你们拱手相让了？一群废物。”
<br/>　　“楚风眠，你不过就是一个小小的弟子，有什么资格在这里放肆！”
<br/>　　站在一旁的夜皇，看着背后陈家的诸位古帝，仿佛也是得到了靠山一般，有了底气，对着楚风眠冷声说道。
<br/>　　“我们北邙学院跟陈家联合，乃是一件大事，未来我们北邙学院足以占据北域，跟七大宗门平起平坐，这件事今日要是因你而坏，那你就是我们北邙学院万年一来的罪人！”
<br/>　　“还不速速放了陈柏长老！跪下道歉！”
<br/>　　“跪下道歉？成为罪人？”
<br/>　　楚风眠目光闪烁，露出了几分杀意看向那夜皇。
<br/>　　“我看你们这群废物，才是北邙学院的罪人，我要重振北邙学院，你们这些人，都是害群之马，留下也是毒瘤，今日正好将你们全杀了！”
<br/>　　“给我死！”
<br/>　　刹那之间楚风眠出手了，一道龙爪直指这夜皇轰击而去。
<br/>　　“楚风眠，你敢在这里出手？”
<br/>　　夜皇愤怒的吼道，见到楚风眠出手的一刻，他也在凝聚灵力，在他的身边，凝聚出一片黑夜，整个人的身形，跟着一片黑夜，融为一体。
<br/>　　“黑夜无边！永夜天存！”
<br/>　　如今这夜皇，也是在九域剧变之中得到了惊天的奇遇，已经是踏入了帝尊境，成为了一位五劫真帝。
<br/>　　可五劫真帝，在这九域也许是算一个人物，但是在楚风眠的眼中。
<br/>　　与蝼蚁无异。
<br/>　　“灭！”
<br/>　　楚风眠冷笑一声。
<br/>　　轰隆！
<br/>　　战龙之爪笼罩而下，顿时这夜皇化为的一片黑夜，都被当即撕碎，漫天黑暗，被一爪撕裂开来。
<br/>　　他的身躯也被随即毁灭，彻底湮灭，就连一点点的精血都没有留下，当即陨落了，生机断绝，再也没有复活的可能。
<br/>　　“你，你居然敢杀夜皇！大逆不道，大逆不道！”
<br/>　　北邙学院的一位老祖看着夜皇陨落，不可置信的大喊起来，他转过头看向陈家的诸位古帝，大声的喊道。
<br/>　　“诸位陈少主，还请您出手，降伏此子，为我北邙学院平息动员！”
<br/>　　“请陈少主出手！”
<br/>　　阳皇，风皇等人，都是跪在了地上，大声的喊了起来。
<br/>　　“小家伙，你就是楚风眠？没想到九域的土著之中，还能出现你这样的人，不错，以准帝境界，能够拥有如此实力，就算是在我们阎罗星域，也算得上是天才了。”
<br/>　　那坐在王座之上的陈少主，突然动了，他看着楚风眠，昂起头，冷笑一声道。
<br/>　　“不过这种实力，在我们陈家的面前，不值一提，小子，看来你也是北邙学院出身的？那好，本少给你一个机会，现在跪下臣服，我可以让你当这北邙学院的院长，统御北邙学院！”
<br/>　　“什么？”
<br/>　　“陈少主，这？”
<br/>　　被北邙学院院长，跟其他几位老祖，听到这话都说一愣，随即刚刚想要开口。
<br/>　　旁边的一名陈家古帝突然冷哼一声道。

<br/>　　随即这陈家古帝转过头来，看向楚风眠说道。
<br/>　　“小子，少主仁慈，赐给你天大的好处，还不速速跪下？从此之后，安心为我陈家做事，少不了你的好处，识相一点，快点过来跪下磕头认错！可以饶恕了你的罪孽！”
<br/>　　“先放了陈柏！然后速速跪下磕头认错！”
<br/>　　又一名陈家古帝，恶狠狠的开口道，脸色阴狠无比。
<br/>　　“不然的话，就你的小小实力，敢跟陈家对抗？我陈家出手，屠你满门，轻而易举！”
<br/>　　“哈哈哈哈，让我跪下？屠我满门？就凭你们这些跳梁小丑？陈家？阎罗星域？你们也敢来九域放肆？都拿命来吧！今日这些人，今日一个也别想活着离开！”
<br/>　　楚风眠突然间大笑起来，在这笑声之中，楚风眠也突然出手了。
