　　“叶少阳，你若一意孤行，今日怕出不了这海眼！”
<br/>　　御灵说完，命令手下另一部分开始攻击法阵，因为西面法阵已经被打出了缺口，裂缝很快就蔓延到了这边，等法阵破碎，鬼兵一下子涌了上来，叶少阳这边也早做好了战斗准备，跟小白一起冲了上去。
<br/>　　小白现出真身，钻入水中，巨大的尾巴伸出水面，用力一卷，立刻一片惨叫声，一大堆邪物被卷入水中，支离破碎，小白就这么在水中翻腾着，不断绞杀着对手。
<br/>　　那御灵看在眼里，化作一道黑气，钻入水中，跟小白纠缠起来。
<br/>　　更多的邪物，开始抢滩登陆。
<br/>　　叶少阳手持七星龙泉剑冲上去，一通狂砍之下，对方死伤无数，但更多的鬼兵还是源源不断地涌来。这根本是不是战斗，而是杀戮。
<br/>　　身后那些个岛上的士兵，也迅速加入了战斗。
<br/>　　叶少阳抽空看了看左右两边，也都打起来了，心中十分不解：和海岛的防御法阵，怎么如此轻松地就被破了？也没看他们有什么动静啊。
<br/>　　莫非……是暗处还藏着什么强者，提前破了法阵？
<br/>　　玄教教主？
<br/>　　毕竟是在海上，要是有人藏身在水中，的确不容易被发现。
<br/>　　不过，就算知道这点，对眼下的处境也没什么帮助——虽然他们拼死杀敌，但对面人实在太多，总有漏网之鱼，冲他们身边掠过，朝高台上方冲过去。
<br/>　　叶少阳心中着急，但也没太好的办法，只能努力拦截，好在林三生站在高台上，指挥着自己几个手下，和阿黄四人，形成了一个包围圈，在台下死守，凡是冲到跟前去的邪物，都被解决了。
<br/>　　叶少阳正砸杀敌，突然头顶一道黑影掠过，抬头一看，是那只巨大的御灵，带着一大帮手下，从自己头顶上空飞过去。
<br/>　　“天地火疾！”
<br/>　　叶少阳凌空打出三张暗金神符，试图拦住御灵，但御灵看也没看，身后一众手下立刻聚集过来，与暗金神符一接触，神符的灵力迅速爆发，立刻炸死了一大堆。
<br/>　　但这些悍不惧死的邪物，还是不断飞来，主动进入神符的灵力范围之内，不断冲来，不断被杀死。
<br/>　　终于，灵符的灵力被他们耗尽，虽然击杀了不知道多少邪物，但御灵却是抓住机会飞了过去。叶少阳愣了一下等他想过去拦截时，已经晚了。
<br/>　　一道人影，破空而至，是吴嘉伟，手持宝剑，与御灵斗了起来，御灵再度化作一道黑气，与他交战了几个回合，吴嘉伟有些挡不住，但御灵志不在他，找了个破绽，将他逼退之后，继续朝高台飞去。
<br/>　　到了高台下方，他故技重施，让身边那成群结队的鬼兵缠住阿黄等人，自己飞身上了高台。
<br/>　　林三生无奈上前迎战，一边大声骂道：“你们都在干什么，这就守不住了吗？”
<br/>　　黑气在他面前，重新聚集成了御灵的形象，也不多说，与他斗起来。
<br/>　　林三生手拿天地规尺，奋力抵挡，但他不是御灵对手，仗着天地规尺的强大灵力，也只能勉强保护自己。这时候吴嘉伟赶到，两人夹攻，才勉强将御灵抵挡住，就在这时，上空昏暗的天空中突然亮起了一道金光，如从天而降，越来越大，快降落在高台上方时，众人方才看出是一尊金色的佛像，当空直接砸下来。
<br/>　　“快走开！”
<br/>　　林三生拉着吴嘉伟离开，御灵看情况不妙，也想逃走，但被一道佛光罩住，立刻就不能动了，只好转身迎上去，死命抵抗，希望还能有一丝生机，结果还是想多了，那金佛身上射出的佛光，轻松化解了一切攻势，在金佛的威压下，御灵很快就化成了一团黑气，接着被佛光渗透。
<br/>　　就好像光明驱散了邪恶，本来如深渊般黑不见底的黑气，被一点点驱散，最后化作虚无，只剩下星星点点的精魄飞出来，飞进了高台上的古井中，再也不见了。
<br/>　　众人看的心惊。
<br/>　　这巨大的金佛，显然是金刚三藏的法身，不愧是证道的强者，那御灵，连吴嘉伟和林三生都不是对手，结果就这么轻松地被金刚三藏干掉，这，就是实力的差距。
<br/>　　“大伙收缩，退到岸上来，小白你们四个继续在水中袭击，打乱他们阵型！”林三生吩咐下去，大伙立刻后退，按照之前的计划，以高台为中心，在岛上围成了两个小圈，外圈是捉鬼联盟的成员和海岛士兵，林三生和小九的手下，也都凑了进去，拼死挡住上岸的鬼兵。
<br/>　　叶少阳、小九，冲和子、四宝、吴嘉伟五人，组成了内圈，时刻关注外圈的战况，帮他们减轻压力，一边消灭闯过外圈的漏网之鱼。
<br/>　　慕寒等人，本来还自持手段，不愿跟叶少阳合兵，坚持单干，如今被情势所逼，也不得不加入了外圈防线，免得被当成了活靶子。
<br/>　　“上面咋样了！”叶少阳回头对林三生喊了一声。
<br/>　　“还有快半小时，坚持下去！”林三生又补了一句，“我们现在很被动。”
<br/>　　“被动？有大和尚在，怕什么？”四宝好奇问道。
<br/>　　林三生看了他一眼，低声道：“这才开打没多久，三藏法师就被逼出手，你觉得是好事？”
<br/>　　“你是说……”
<br/>　　“你怎么知道对方没有这个境界的强者？”林三生一句话说的大家哑口无言，他面带忧色地环顾一下战局，叹道：“我本以为，光是海岛法阵，便能坚持个二十分钟，没想到开局就被破了，显然对方有一位极其厉害的强者，躲在暗处，三藏法师先现身，并不是好处……不过事已至此，也只有死守了。”
<br/>　　他话刚说完，外围防线突然一阵骚动，大伙一起看去，发现在一堆邪物的中间，出现了一个全身黝黑的……猴子？
<br/>　　这东西长着一张慎人的鬼脸，身材巨大如银背猩猩，龇牙咧嘴，不断冲击着外围防线，大伙看过去的时候，他已经抓了一个海岛士兵，生撕成两半，凑在嘴边大嚼起来，画面十分可怖。
