徐少南：宝宝心里苦啊！
<br/>
<br/>        厉老夫人并不想让夏夕绾跟自己的外孙徐少南在一起，她看得出来这两个人并不是一路人，现在厉老夫人看着陆子羡，厉老夫人对陆家这两个儿子还是很清楚的，陆子羡就像是开在淤泥里的一朵水莲，干净而清华。
<br/>
<br/>        陆子羡的私生活又是零，还没有交往过女朋友，厉老夫人看得出来陆子羡很喜欢夏夕绾，小心翼翼的呵护和怜爱，所以厉老夫人像是选亲的孙女婿般好好打量了陆子羡两眼，然后满意的点头。
<br/>
<br/>        夏邦和蓝烟对陆子羡也是很满意的，这帝都公子里，陆子羡才华无双，又生的俊美，和夏夕绾倒是天生一对。
<br/>
<br/>        这枚玉扳指是林水瑶一直戴上手上的，夏邦和蓝烟一眼就认出了小姐当年的贴身之物，当年小姐的确为夏夕绾这个小小姐亲口许配了一桩婚事，他们没想到小小姐竟然许给了陆家二少。
<br/>
<br/>        陆子羡是帝都城最年轻的院士，金刀圣手，夏夕绾也是学医的，中医精湛，这两个人走在一起，真是人间眷侣了。
<br/>
<br/>        夏邦和蓝烟对这位女婿非常满意，夏邦当即摆了摆手，声音洪亮的宣布道，“各位贵宾，我们夏家跟陆家的确是有婚约在身的，那我家绾绾就是陆家二少的未婚妻，陆家二少也就是我们夏家的新姑爷，大家既然来了，那就都做个见证，等过些日子，我们找个黄道吉日就让绾绾和子羡完婚！”
<br/>
<br/>        众人，“…”
<br/>
<br/>        他们来干什么了，见证夏家的新姑爷上门？
<br/>
<br/>        不是说夏夕绾已经被那几个黑衣男人给玩过了吗，没有人愿意接盘，这话谁说的，他们想将造谣的那个人脸给打肿了！
<br/>
<br/>        谣言不攻自破，夏夕绾一定是清白的，要不然她怎么这么抢手，厉陆徐三大豪门都给夏家递了帖子，来给她撑腰。
<br/>
<br/>        现在好了，夏夕绾愿意不愿意当厉家老太君的孙女另外再说，反正她已经是陆家二少的未婚妻了。
<br/>
<br/>        那些帝都名媛们：不，这一定是一场噩梦！
<br/>
<br/>        她们拿出手机，再次给厉嫣然发送了小视频。
<br/>
<br/>        厉嫣然还是跟夜明珠在一起，厉嫣然先看到了小视频，她瞳仁一缩，“少南表哥知道自己在干什么吗，他是不是疯了？”
<br/>
<br/>        “怎么了？”夜明珠也点开了徐少南的那个视频，她当即“靠”了一声，“徐少南竟然带着八大箱的金银珠宝去夏家提亲，要娶夏夕绾？他眼睛瞎了吧？”
<br/>
<br/>        徐少南，厉嫣然和夜明珠都是亲戚关系，但是徐少南不跟她们玩的，现在厉嫣然和夜明珠都觉得徐少南被下降头了。
<br/>
<br/>        本来大家都说好了帝都的名门公子一个都不出席夏家寿宴！
<br/>
<br/>        “下面还有一个视频。“厉嫣然点开了下面一个视频。
<br/>
<br/>        很快厉嫣然瞪大了双眼，她惊恐的看着身边的夜明珠。
<br/>
<br/>        夜明珠还不知道发生了什么事情，“嫣然，你看到了什么，你怎么这个表情？”
<br/>
<br/>        “明珠，告诉你一件不好的事情，陆子羡也去了夏家寿宴，而且…夏夕绾是陆子羡的未婚妻！”
