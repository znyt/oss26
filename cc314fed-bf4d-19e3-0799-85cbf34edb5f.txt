　　第573章
<br/>　　黑牛的速度是极快的，四蹄奔起后，荡起一道烟尘。宛若一道暴怒的推土机，浩浩荡荡轰过。
<br/>　　什么书院弟子，东海高手，都无法对它形成有效拦截。程大雷毕竟领着它上过战场，千军万马也是敢冲，何况现在岛上的人数根本无法和千军万马相提并论。
<br/>　　很短时间内，程大雷就把整座岛趟了一遍。因为本来白沙岛就没有多大，也大概明白，书院在这里准备了多少人。
<br/>　　约有一百余人，通过查看他们的信息，程大雷了解到，他们中间的绝世高手共有九人。
<br/>　　九位绝世高手，包括：夫子、孟玄青、宋寒之、张六、背九柄飞矛的孙飞鹰、使手戟的周沧海……
<br/>　　他们每一位都有和程大雷捉对厮杀的能力，程大雷不敢说可以稳赢他们任何一位。毕竟高手过招，决定最后结果的因素太多了。而今天，程大雷面对的敌人不是一位，而是九位。
<br/>　　此外，还有数十名书院弟子，他们的剑阵程大雷已碰过，自己不是对手。
<br/>　　当真是陷入了绝地。
<br/>　　把海岛趟过一遍后，程大雷也看见了李婉儿和李莫愁，她们困在木笼中，生死不知。
<br/>　　程大雷停下了座骑黑牛，想救的人就在眼前，他自然没办法再兜圈子。今天本就是一场硬仗，此程大雷也没指望能轻而易举的破局。
<br/>　　“程大雷！”宋寒之立在高处喝道：“今日就是你的死期，任你有泼天本领，也是插翅难逃。”
<br/>　　程大雷根本没有搭理他，今天这场仗根本不适合多说废话。他从牛背上腾起，向山顶上突进，剑已出鞘，杀威淋漓。
<br/>　　最主要的目的，就是不能被缠住，否则真的书山剑阵成型，自己并无破阵的本事。所以只有利用本身超高的机动性，和他们玩突袭战。务求一招杀敌，杀不了就转向下一个敌人。
<br/>　　至于那头黑牛，程大雷根本不担心。它跑起来，几乎没人拦得住，实在不行，它就跑水里去了。
<br/>　　毕竟是水陆两栖的座骑，插上翅膀能飞程大雷都不怀疑。
<br/>　　程大雷突进到宋寒之面前，一剑刺出，论对剑道的了解，宋寒之肯定比程大雷强，但论快剑术，他不是程大雷的对手。
<br/>　　但宋寒之毕竟也是绝世级别的高手，也有压箱底的绝招。一招靴底清风，轻飘飘避过程大雷一剑，回身一招扶柳观月，攻击的是程大雷侧身。
<br/>　　程大雷轻身功夫闪过，身子腾在半空中。
<br/>　　“恶贼，与我下来吧。”
<br/>　　孙飞鹰出手，背后的九柄短矛出手了两柄，一柄攻击程大雷，一柄堵住了他的退路。
<br/>　　程大雷没有退，他用灵犀一指接住，挥手还给孙飞鹰，身子缓缓落下。而张六的大棍已经恭候多时，一招举火焚天向天空中捅去。
<br/>　　程大雷脚尖在棍上一点，再次借力，飘出数丈远。那里有一人等候，双戟并发，程大雷如花间蝴蝶，与对方对了两招，抽身退在一块突起的山石上。
<br/>　　一切都在电光火石间，功夫低的根本不知道发生了什么。而在刚才的每个瞬间，程大雷都十分危险，稍微一丝疏忽，他就要命丧当场。当然，宋寒之等人也不好受，程大雷守中有攻，他们如果有些许含糊，死的也就是他们了。
<br/>　　初相逢时的交手，双方都是胆战心惊。
<br/>　　对方的高手还没全部出动，已经让程大雷应对得无比艰难。而宋寒之等人心中也均想：此人有如此本事，果不愧是天下第一贼。
<br/>　　程大雷平复呼吸，继续往山上闯，先摸清楚李婉儿二人的情况要紧。当初李行哉离开时，托付自己好好照顾李婉儿。如果她真的有个闪失，自己怎么向李行哉交待。
<br/>　　他又如何向自己交待。
<br/>　　再次闯，面对的敌人是夫子。夫子很强，这是一个不争的事实。但程大雷也不怕他，因为上次交手，程大雷已经发现，战斗时间一旦拉长，他并不是自己的对手。
<br/>　　可今天，敌人会给自己时间么。
<br/>　　争的就是时间，如果能在最短时间内击杀夫子，书院一方人心必乱，那也就是自己获胜的机会。
<br/>　　二人交手，都是以快打快。夫子明显研究过程大雷的招数，而且他还很聪明，现在已经有了防备，在交锋的一开始，他竟占了上风。
<br/>　　可他毕竟老了，老人是注定要输给年轻人。哪怕他再聪明，能一眼看破程大雷的破绽，却也敌不过时光的无情。
<br/>　　程大雷已经有获胜的契机，寻得破绽，一剑刺出。
<br/>　　夫子胸口中了一剑，只因入剑不深，没有伤到心脏，不然，夫子这条老命也就交待在此地了。
<br/>　　程大雷之所以没有办法一剑杀了夫子，是因为张六的大棍从身后到了。程大雷不得不防御，用灵犀一指的手法握住了大棍。
<br/>　　“与我拿来吧。”
<br/>　　他们只以为程大雷剑法快，东海却很少有人知道。程大雷本不是用剑的，他的本命武器是斧头，至于剑……程大雷一向是对剑这种华而不实的武器很鄙夷的。
<br/>　　张六的力气没有程大雷大。
<br/>　　他同样也老了，力气和反应都在丢流逝。
<br/>　　程大雷一把夺过大棍，丢到一人脑袋上。继续往前冲，闯到李婉儿身边是第一要务，过程中能杀掉夫子也是最好。
<br/>　　然后，又有一人拦住了程大雷的路。
<br/>　　这人用一柄长枪，有个很欠揍的名字：王无敌。
<br/>　　白沙岛本就是书院弟子苦修之所在，的确有人远离世间繁华，在此地磨炼武艺和学问。这王无敌就是其中一位。
<br/>　　他本是旧楚国的贵族，祖上是楚国王族的家臣。所以，他枪法的名字也就不言自明：霸王点金枪。
<br/>　　枪重三十六斤，精铁打制，不算太重的枪法，讲究一个字：快。
<br/>　　世间不知王无敌，但不代表王无敌不是高手。他甚至是程大雷目前遇到的最棘手的一人，对方武艺未必是最强的，但他还年轻。今年不过四十余岁，正是精力和气血最旺盛的时候。
<br/>　　枪动如金蛇狂舞，枪围之内，程大雷没办法靠近。
<br/>(本章完)
