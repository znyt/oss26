　　第586章
<br/>　　韩通的手下一刀一个，就将李如和李大头砍了。两具尸体趴在地上，脑袋滚到一边，血流了一地。
<br/>　　程大雷忽然睁开眼睛，呆呆看着这一幕，最后无可奈何的叹了口气：早就说过的，这样程度的争斗，已不是你们能够插手的。
<br/>　　今日他们沦落到这般境地，确实不是程大雷的错。
<br/>　　是这世道的错。
<br/>　　书院、刘黑虎、韩通三方谁也不会因为李如二人的死有半分动容，他们更多的注意力还是在程大雷三人身上。
<br/>　　“青州的事和你们书院扯不上关系，我劝你们还是滚回徐州去吧。”刘黑虎握紧手中的刀，已随时准备出手。
<br/>　　钟牧山七人也已经将剑出鞘：“都是贼人，杀一个是杀，杀两个也是杀，干脆一起杀了。”
<br/>　　韩通的兵器是一杆铁枪，他目光转了一圈，想着怎么浑水摸鱼，从书院和刘黑虎两大强敌手中，把程大雷劫走。
<br/>　　眨眼间，三方人马便厮杀在一起，刘黑虎与韩通都警惕着另外一方，所以打起来很谨慎，都提防着另外一方。
<br/>　　相比之下，钟牧山七人没有江湖经验的缺点就暴露了出来。他们一开始便全力以赴，最后不知怎地搞得，成了刘黑虎和韩通双方联手攻击书院的剑阵。
<br/>　　书院剑阵的确很强，哪怕只有七人，也可立于不败之地。但也仅仅如此而已，在双方联手下，他们只有防守之力，而无还手之功。
<br/>　　钟牧山的心情愈发着急，久守必失，他虽然没闯荡过江湖，但书院兵书无数，这个道理他还是懂的。
<br/>　　若想获胜，必须击中全力，攻破敌人一点，最后再收拾程大雷。
<br/>　　钟牧山把主意打在韩通身上，相比刘黑虎，他的确不算强悍。
<br/>　　在他的指挥下，书山剑阵由守转攻，七人的剑宛若一体，不停向前推进，七人连环攻击，七柄剑斩的是一个人的脑袋。
<br/>　　钟牧山小看了韩通，能成一州大将，他又怎么会是泛泛之辈。钟牧山之所以觉得他不太行，是因为韩通一直在藏拙。三方乱战，只有钟牧山这样人，才会一开始就出全力。
<br/>　　如今面对强力攻击，韩通已不可再藏拙，他抡起铁棍，转了个圈，将七柄剑同时挡住。
<br/>　　钟牧山毕竟大意了，以为所谓书院第一便是天下第一，不过这也平常，只要他们多碰几次壁，就能迅速成长起来。毕竟，书院培养出来的弟子，底子还是不错的。
<br/>　　问题是，他们是否还有成长的机会。
<br/>　　在攻击韩通失利后，刘黑虎的攻击已经从背后来了。现在钟牧山已经腹背受敌，而且书山剑阵因为调动不及，已经出现乱象。
<br/>　　三方分开，钟牧山肩上受了伤，到现在，他才知道不该小窥天下英雄。
<br/>　　“什么书院弟子，也不过如此么？”刘黑虎冷笑道。
<br/>　　韩通摁着铁棍不说话，他现在也渐渐有了些底气。
<br/>　　钟牧山等人吃亏不小，但书院的弟子向来有一往无前，虽千万人吾往矣的勇气。现在他已收起轻视之心，决定放手一搏。
<br/>　　“来！”
<br/>　　钟牧山手握青锋剑，死又如何，书院弟子向来不怕死。
<br/>　　“住了吧。”
<br/>　　一个声音突然响起，三方都是一激灵，目光落在程大雷身上。
<br/>　　咔嚓，咔嚓。
<br/>　　两声脆响，麻绳被一截一截挣断，程大雷长身而起，手中摁着匹夫剑的剑柄。
<br/>　　不约而同，刘黑虎、韩通、钟牧山三方势力同时往后退。在刚才的乱战中，大家的确都将他忽略了。
<br/>　　而偏偏程大雷是一个最不该被忽略的人，这魔头，但凡有一丝力气，都是一个令人忌惮的存在。
<br/>　　三人的目光都落在程大雷身上，全身肌肉绷紧，面对这样一个存在，谁也不敢有任何大意。程大雷的匹夫剑还未出鞘，已带给三人巨大的威压。
<br/>　　只是一刹那间，程大雷的身子突然跃起，宛若山林中的白猿扑到韩通面前，匹夫剑就在这一刹那出鞘。
<br/>　　韩通能和刘黑虎与钟牧山打得难解难分，可又怎么是程大雷的一招制敌。他根本无法看清眼前发生了什么，就感觉剑光一闪，韩通的喉咙出现一道血线。
<br/>　　随之，血就窜了出来，韩通的尸体噗通一声坠地。
<br/>　　程大雷落下的时候，黑牛正好奔到他身下，程大雷没有任何废话，骑上黑牛便一路狂奔。
<br/>　　一切就在电光火石间发生，等三方反应过来，韩通已经被杀，程大雷已经突出重围。
<br/>　　三方人马开始分头追击，在程大雷身后，紧追不舍。
<br/>　　程大雷一刻也不敢留，骑着黑牛也来不及分辨方向，只是往前走。
<br/>　　他现在是有苦说不出，程大雷并没有那么强，至少没有看上去那么强。李如给程大雷下的药的确很猛，如此也可以想见李如要杀程大雷的迫切。
<br/>　　程大雷昏昏沉沉了许久，才勉强恢复一些气力。他挣断束缚，击杀韩通，这点力气已经用尽。如果现在被追上了，程大雷握剑的力气也无，也就只能是被人乱刀分尸一个结果。
<br/>　　他如今不是骑在牛背上，而是完全趴在牛背上，脑袋越来越沉，黑牛颠簸着他身体起伏。
<br/>　　“好畜生。”程大雷用仅有的力气道了一声：“快点跑啊，不然咱们都有死在这里了。”
<br/>　　眼皮越来越重，程大雷不知何时已昏迷了过去。黑牛仍驮着他往前跑，最后不知向前跑了多久，四野黑茫茫的，不见人家。
<br/>　　噗通一声，程大雷从牛背上跌了下来，滚到一堆荒草中，昏迷不醒。
<br/>　　那黑牛守在他身边，一声一声打着响鼻，时不时扯扯他的衣服，希望程大雷会醒来。
<br/>　　不知过了多久，天已经彻底黑下来，今天没有月亮，也没有星星。恰恰有人骑着一头小毛驴从这里经过，看到黑牛时吓了一跳。
<br/>　　黑天半夜，这黑牛又格外丑陋些，当真如一头怪物相仿。这人也是胆子颇大，竟然从毛驴上下来，一步一步靠了过来。
<br/>　　看到这头黑牛后，想不看到程大雷也不容易。这人将程大雷翻了个身，看清他的相貌后也是一惊。
<br/>　　“咦。”
<br/>(本章完)
