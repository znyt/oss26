　　老头子想要长生的事情李承乾一直都知道，只是没想到这老头儿会这样执着，为此竟然连皇位都不要了。
<br/>　　不过这事儿想想也是，如果老头子真的可以长生不老，皇位什么的对于老头子来说还不是手到拈来，反正老李相信，打仗什么的自己就绝对不怕自家那个熊孩子。
<br/>　　便是因为这样，老头子在魏黑子的背后暗暗推了一把，同时警告长孙无忌，柴绍等人静观其变随大流，这才造成了眼下这个局面。
<br/>　　否则以老李对长安城的监控力度，魏征一个快死的老头子，如何能掀起这和大的风浪。
<br/>　　当然，这些都是背后的故事，不可为外人道，就连长孙皇后甚至都被蒙在鼓里，一切都是老李在暗中操纵。
<br/>　　……
<br/>　　话分两头，时间一天天过去，很快李淳风就到了李淳风算好的日子，三月三！
<br/>　　李承乾也迎来了人生中最大的转折。
<br/>　　祭天、祭地、祭祖先，长长的祭文念的李承乾脑壳疼。
<br/>　　想着后世看历史书的时候，那个赵匡胤把皇袍往身上一披就特么是皇上了，多简单啊，怎么论到自己就这么麻烦呢。
<br/>　　不过尽管心中再不愿意，他还是要忍，毕意下面还有那么多人看着呢，那些什么御史言官就等着他出错，然后出来表现自己，最好能来个撞柱而死那就完美了。
<br/>　　浑浑噩噩间，李承乾像个木偶一样任人摆布，木然看着老头子离开皇位，然后自己再木然走上去，接受加冕，最后老头子亲手把那个玉玺交到自己手里。
<br/>　　原本李承乾看着老头子禅位心里还是有些难过的，总是觉得有些不忍心。
<br/>　　可是被人摆弄了那么长时间，身心俱疲之后，丫便彻底麻木了，只知道点头，点头，再点头。
<br/>　　直到坐上那个让所有人都要仰视的位置之后，才反应过来自己已经是大唐的皇帝了。
<br/>　　而搞笑的是，这一切过程除了开始念祭文的时候自己还能记得一些，其它过程竟然全都没有记住，人生中最大的一件事情，就这么被错了过。
<br/>　　……
<br/>　　“臣等参见陛下！吾皇万岁万岁万万岁！”还不等李承乾脑子清醒些，下面已经一大群人弯腰拜了起来，顺便还不忘投其所好，来一句‘万万岁’。
<br/>　　“平身！”李承乾一脸木然伸了伸手，觉得除了称呼不一样之外，其他好像都和他以前监国的时候没什么不同，人还是那些人，位置也还是老位置，特么一点新意都没有。
<br/>　　“陛下，臣有本奏！”众人刚刚站直了身体，一个家伙已经跳了出来。
<br/>　　李承乾看了眼出来的那人，却是刚刚回京的许敬宗，于是便点了点头：“讲！”
<br/>　　许敬宗本就是心思灵巧之辈，自然知道应该在什么时候表现自己，见李承乾点头，便恭声说道：“陛下今日登基，臣以为应当大赦天下，以示普天同庆！”
<br/>　　李承乾皱了皱眉头，又是大赦天下？为什么这帮人都喜欢玩儿这一招呢？拿别人的东西送人情有意思么？
<br/>　　想到这里，李承乾暗暗摇了摇头，看着下面的众说道：“众卿，此事你们怎么看？”
<br/>　　一群老鬼你看看我，我看看你，明显在这个时候不想出来顶雷，于是便齐声说道：“臣等听凭圣裁。”
<br/>　　我呸，听你大爷！李承乾心中骂了一句。
<br/>　　因为两世的记忆，他是真的不想大赦天下，后世的教育让他无法接受随随便便就把那些打架斗殴，杀人放火，小偷小摸的家伙都放掉。
<br/>　　另外就是一些政治犯了，也就是一些因为说错话或者办错了事被关起来的，这些人大部分都是世家的门人子弟，被许敬宗和李义府前段时间给坑进了大牢。
<br/>　　这些人就是鸡肋一样的存在，关着也只能浪费粮食，放了也没啥用，世家也不会因为这样就对他这个新皇心存感激，从此抛弃一切成见，全心全意为国家办事。
<br/>　　想通了这些，李承乾心中便有了决定，看了一眼下面的许敬宗说道：“朕刚刚登基，寸功未立，却把以前父皇宣判有罪之人都放了，与孝道不合，所以这大赦天下就算了吧！”
<br/>　　算了？不赦了？那些世家一系的官员全都愣了，原本还指望着那些亲朋故友可以借着这个机会重得自由，现在一看这还是要继续关下去啊。
<br/>　　可是有人郁闷，同样也有人兴奋，一些性格耿直的，听到不会大赦天下立时高兴的拜了下去：“吾皇圣明！”
<br/>　　“圣不圣明的以后再说吧，现在朕有件事要宣布一下。”李承乾也不等其他人发表不同意见，抬手向下虚按一下，示意众人闭嘴，然后才说道：“有谁知道自秦皇建第一代皇朝以来，至今有多少年了？”
<br/>　　“这……”话落，大殿下所有人都皱起了眉头，武将不用说了，一个个全都傻呆呆的站着，别说从秦始皇那一代开始算，就算是从李渊那一代开始算，能算明白多少年的都不多。
<br/>　　文臣一方倒是有很多人在那里掐手指，几年几十年的开始往一起加，但加来加去却得出好多数值，有说七百多的，也有说八百多，还有说一千多的，乱七八糟没个统一的标准。
<br/>　　良久，李承乾终于是看不下去了，哼了一声说道：“不必再算了，是八百六十三年！”
<br/>　　八百六十三年？殿上众人齐齐一愣，但却没有人不相信。
<br/>　　毕竟李承乾会这样问，那就一定会提前有所准备，这个数字应该是可信的。
<br/>　　可是，为什么会无缘无故的提出这个问题，难道是要……。
<br/>　　就在众人心中疑惑不解的时候，李承乾为他们解开了谜底：“朕前些时候偶有所感，觉得现在这样计算历法实在有些麻烦，不如我们就统一一下，从秦皇称帝的那一年算起，形成一个统一的年份，以后便叫成皇历！”
<br/>　　皇历，呵呵，比西元强不少，明明有着五千年传承，却要过什么西元多少多少年，这怎么可以，作为一个华夏人，要过就过皇历！
<br/>　　李承乾如此想着，心中决定，一会儿不管下面的人说什么，自己也要把这件事情做成。
