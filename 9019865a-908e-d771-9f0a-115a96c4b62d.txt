　　从这武斗大会中的收获，要远远超出楚风眠的想象。
<br/>　　如今这才仅仅是擂台战的第一轮，许许多多的剑修强者，都还没有动用实力，他们的手中许多人，也有着压箱底的剑术。
<br/>　　看来这些擂台战，以后要经常参加了。
<br/>　　楚风眠心中暗自想到。
<br/>　　不管是在北域之中，还是整个九域之中，像是这样的擂台战，都是数不胜数。
<br/>　　这种擂台战，便是跟强者交手最好的机会，许许多多的武者，都会参加，而且参加擂台战，也是最容易扬名立万的办法。
<br/>　　之前的楚风眠，倒是对这种擂台战没有太多的兴趣，所以也不去参加。
<br/>　　可是现在，楚风眠要创造九域剑术，便是要将这九域之中的所有强大剑术，都必须要领悟，这种擂台战，绝对是楚风眠面前各种强者最好的机会。
<br/>　　楚风眠一边观看这，一边在不断的改变着他的九域剑术。
<br/>　　他的九域剑术，只是有了一个雏形，还没有彻底凝聚，这一切还在积累，一切种种玄妙可取的剑术，楚风眠都会融入其中。
<br/>　　等到这九域剑术真正大成的一刻，便是足以化为天地之间最为强大的剑术。
<br/>　　第一轮的擂台战，很快便是过去，来到了第二轮。
<br/>　　在这第二轮中，楚风眠所碰到的对手，却是来自一个二流宗门流风宗的弟子，在北域百绝榜上，最多排到五十位左右。
<br/>　　能够冲击到第二轮，其实已经是他的极限了，若是碰到一些弱者，也许还有着再进一步的希望。
<br/>　　不过显然，碰到了楚风眠，便是已经没有任何的机会了，被楚风眠轻轻一招便是击败，打飞了台下。
<br/>　　对此人，楚风眠倒是没有伤他，将他轻松击败便是。
<br/>　　善恶分明，对这种敌人，楚风眠只会击败，不会伤到他。
<br/>　　寒月漓也是碰到了一个不强的对手，轻轻松松一招，便是击败。
<br/>　　不过在这下一场之中，倒是出现了两尊强者。
<br/>　　分别便是楚风眠所看好的葬天，跟北邙百绝榜上的第四位的武者，来自沧澜世家的少主。
<br/>　　这一战，本来是没有多少悬念的。
<br/>　　葬天虽然称得上是这一届中的“黑马”。
<br/>　　但是他如今所碰到的，却是北域百绝榜上排名前五的人物，沧澜惊。
<br/>　　在强大的黑马，想要跟这种北域百绝榜上排名前五的人战斗，其中的结局，众人也都心中明白。
<br/>　　葬天，必败。
<br/>　　只是接下来的战斗，却是有些出乎意料，葬天跟沧澜惊交手，居然是将沧澜惊击败了。
<br/>　　甚至是三招之内，便是做到了这一点。
<br/>　　这让在场无数围观武者，都是沸腾了起来。
<br/>　　沧澜惊可是北域百绝榜上第四名的人物。
<br/>　　虽然说他跟着风无心，夜天君，冷面魔君这前三人相比，似乎有些差距，但是也是三人之下，万人之上的存在。
<br/>　　现在居然是让一个默默无闻的葬天击败，一下子便是令在场沸腾起来。
<br/>　　对于葬天的议论，甚至是要远远超越了楚风眠。
<br/>　　“此人，好强。”
<br/>　　寒月漓看着葬天，缓缓的开口道，她的脸色都是微微变化。
<br/>　　寒月漓面对沧澜惊，她心中还是有着极大的自信，可以将沧澜惊击败，但是三招做到这一切。
<br/>　　寒月漓自问自己，就算是动用全力，都不一定能做到。
<br/>　　这葬天，可以说是真正的默默无闻，根本没有听说过的一个人物，如今突然冒出来，居然便是有着如此实力。
<br/>　　“这葬天，我似乎见过他。”
<br/>　　楚风眠看着葬天将沧澜惊击败，却是若有所思的开口道。
<br/>　　这结果，楚风眠并不意外，甚至他还感觉到，葬天的实力，还有隐藏。
<br/>　　击败沧澜惊，葬天动用的是一种纯正的武道，但是楚风眠看得出来，这武道，只是葬天的隐藏，他在隐藏着自己的真正实力。
<br/>　　尤其是观看葬天战斗的时候，楚风眠从他的一举一动之中，都察觉到了几分熟悉。
<br/>　　以楚风眠的灵识，早已经是达到了过目不忘的地步，这种熟悉，并非是偶然，只是楚风眠也如今一时间无法想到，这葬天，到底是谁。
<br/>　　但是楚风眠肯定是跟这葬天曾经见过，甚至是交手过。
<br/>　　“此人，应该是武胜国度的人。”
<br/>　　楚风眠开口道。
<br/>　　“什么？”
<br/>　　寒月漓听到楚风眠的话，有些诧异的说道。
<br/>　　“武胜国度中，居然有着如此人物？”
<br/>　　如今离开了武胜国度，前往北域，寒月漓也早已经明白。
<br/>　　武胜国度，简直就是一个偏僻到不能在偏僻的小地方了，什么武胜学院，一个北邙学院的外门弟子，都可以将其毁灭。
<br/>　　像是这样的小地方，能够出现楚风眠跟寒月漓两个人，都已经极为让人惊讶了，没想到居然还有第三人？
<br/>　　“不错。”
<br/>　　楚风眠点了点头。
<br/>　　能够让他感觉到熟悉的，交手过的，只有武胜国度的人。
<br/>　　“算了，反正迟早我们也会碰到他，碰到他的话，一定是要动用全力。”
<br/>　　楚风眠开口道。
<br/>　　既然都是来参加这武斗大会，自然是将来肯定会碰到，倒时候这葬天到底是什么身份，真正交手，便是就知道的一清二楚。
<br/>　　起码楚风眠有着自信，足以是将这葬天的真正实力给逼出来。
<br/>　　“嗯。”
<br/>　　寒月漓也是表情凝重的点了点头。
<br/>　　葬天，的确是一个强敌。
<br/>　　葬天击败沧澜惊，是第二轮上的一个插曲。
<br/>　　这一战，虽然令人震惊，但是越往后，强者相遇的时刻，每一场战斗，都是极为精彩，惊天动地。
<br/>　　第三轮，很快便是开始。
<br/>　　这第三轮的首战，楚风眠手中的身份牌，便是一下子亮了起来。
<br/>　　“到我了。”
<br/>　　楚风眠随意开口一声，他的身躯便是一跃之间，踏上了擂台。
<br/>　　而这一次楚风眠所面对的对手，可不再是什么弱者了，而是一名实力在生死境四重，更是北域北域百绝榜上的前十的人物。
<br/>　　圣党弟子，北邙学院的核心弟子之一，北域百绝榜第九位，武英。
<br/>　　在这第三轮中，楚风眠便是碰到了北域百绝榜上的前十位弟子，更是圣党中人。
