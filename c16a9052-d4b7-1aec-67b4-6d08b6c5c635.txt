　　昆仑玉虚宫中，元始在细细参悟鸿钧所讲的斩三尸之法。
<br/>　　细细品味这斩三尸之法，以他的眼界与经验，很快就发现了，鸿钧所讲的这斩三尸之法，有着巨大的潜力，还有些意犹未尽。
<br/>　　这倒不是鸿钧藏私了，以元始自己的推测，斩三尸之法的后续部分，实际上是因人而异，毕竟每个人的道都会有所不同。在鸿钧手中，就是混元一气，三尸成道，在后世太清老子身上，他就以盘古元神配合三清气演化，也就形成了后世赫赫有名的一起化三清神通。
<br/>　　所以说，鸿钧以道祖之名，布道洪荒，紫霄宫中三千客，都算是他的弟子，但后世中，得到他真传的，终究只是太上老子一人，至于其他的，虽然都修行了斩三尸之法，但终无一人能够发现其中的真髓，空守宝山而不知，怪不得，前世传说中，诸圣以老子为首。
<br/>　　元始这样想着，一边参悟着这套斩三尸之法，争取从中领悟出自己的大道神通。
<br/>　　六百年过去了，这一日，昆仑山上，骤然光华大盛，清光耀天，好似远古大神，从沉睡之中苏醒一般，陡然，一股强大的气势传遍洪荒大地。
<br/>　　之后，不到一个月的时间，昆仑山再次传出强大而凌厉的气势。
<br/>　　昆仑山之变，洪荒大神通者，自然有所感应。但，人们好奇的是，为何昆仑只传出两道气息，却没有第三道，难道是那人还没有突破。
<br/>　　这一刻，有人在嘲笑，有人在讥讽，也有人在疑惑。
<br/>　　洪荒西方，灵山道场之内，接引与准提二人相对而坐，体内有禅意散发而出！
<br/>　　“师兄，东方大地果然人杰地灵，不似我西方这般贫瘠。”准提道人面色难看道。
<br/>　　“师弟，天道循环，必有吾西方大兴之日。”接引道人面容上带着惯有的疾苦，沉声道：“我等只需静心修持，尽早斩尸即可。”
<br/>　　说着，接引道人眼睛睁开，目中有一丝寂灭之意一闪即没。
<br/>　　“师兄，你说那元始，作为三清之首，他为何还没突破？”准提问道，语气之中，有些幸灾乐祸。
<br/>　　“元始这人不简单，他想必有什么用意吧，师弟以后若是遇上此人，切莫大意。”说道元始，接引眼中闪过一丝凝重之色，他完全看不透元始。
<br/>　　“嗯。”
<br/>　　之后的数百年时间内，仿佛引爆了什么，突破之人，一个接一个。
<br/>　　不周山，一股浩大的造化之光，照遍天地，玄光闪耀，众生开启灵智，一股强大的气势冲天而起。
<br/>　　太阳星上，火光大盛，红芒耀天，无尽热量散发而出，炙烤洪荒天地众生。
<br/>　　万寿山，五庄观，一股强横的气势冲天而起，地书飞上五庄观高空，垂下道道土黄色光芒，镇压大地。
<br/>　　与此同时，东海之上、火云洞中、极北之地、不周山脉、北冥之海，西方灵山，各有强大的气势升腾，遥相呼应。
<br/>　　你在进步，我也在进步，洪荒不止你一个人。
<br/>　　但，令众人疑惑的是，始终不见昆仑山有第三道气势出现。
<br/>　　……
<br/>　　太阳星深处，靠近扶桑神树的附近，有一座巨大的山谷，山谷之中，坐落着一尊奢华古老的宫殿，雕梁玉栋，带着一股恢宏、大气，正是太阳神宫。
<br/>　　“大哥，如今我们已经斩去一尸了，突破准圣境界，是时候收服那些妖族了！”太一手持一柄凶剑，望向帝俊，语气之中带着一股兴奋的叫道。
<br/>　　这剑名为混元剑，为极品先天灵宝，乃是诞生于混沌之中，却没有来得及成型，就因为盘古开天而落入洪荒之中。它本来是有机会成为先天至宝的，如今却只能成为极品先天灵宝。
<br/>　　这是他此次游历洪荒之时，在一处充满煞气的深渊之中发现的，当时这柄剑正在吸收着无尽的煞气，他也是费了好大的劲，才收服此剑。
<br/>　　恐怕就连元始都没想到，太一失去了混沌钟后，却又得到了混元剑。这气运，真不愧是量劫主角啊。
<br/>　　“哈哈！”
<br/>　　帝俊哈哈大笑，看向太一，郑重道：“二弟说的不错，以我们兄弟的实力，如今就算碰到大神通者，也不惧。只是收服一些散落的妖族，并不困难。”
<br/>　　于是，帝俊与太一两人共同下了太阳星，朝着洪荒而去。二人之前已经收服了众多妖神势力，再加上联合了女娲伏羲两位妖圣，其他的妖族势力更是挡不住，倒也让他俩收服了一些势力。
<br/>　　妖族之中，讲求强者为尊，帝俊太一两人具备强横的实力，再展露出招揽之意，众妖王自然率领麾下势力归附！
<br/>　　就这样，洪荒大势，如滚滚流水，不可阻挡！
<br/>　　……
<br/>　　“呼……”
<br/>　　狂风呼啸，遥远的北海之上，一座宛如大陆般的岛屿，朦朦胧胧，看之不清，却是有无穷大阵在其中运转！
<br/>　　岛中，浓郁的先天灵气在空中飘着，无数灵兽嬉闹着。无数鲜艳的花朵与灵果点缀着，白云飘飘，阳光明媚，宛如仙境！
<br/>　　岛的中央，有一座宫殿，名曰“北冥宫”。
<br/>　　宫殿中，住着洪荒大妖鲲鹏道人。此刻，鲲鹏道人站在宫殿外，心中突显许多画面，在天道的牵引之下，他似有所悟。
<br/>　　鲲鹏大喜，接着闭关，经过五百年的时间研究，他根据妖族的形状，终于创出了妖文。
<br/>　　“天道在上，吾鲲鹏感妖族修炼不易，今日创妖文，传修炼之法！”
<br/>　　妖文一出，天道顿时生感应，一阵牵引之下，洪荒大地的所有妖族顿感脑海之中出现文字，随后就是一段修炼法门，妖兽们大喜，随即全部朝着北冥的方向跪拜道：
<br/>　　“鲲鹏妖圣大德！”
<br/>　　此刻那些妖族脑海之中出现的文字，也就是最初的传承记忆。
<br/>　　“嗡……”
<br/>　　天道有感，天空一片涟漪，骤降异象，一团功德神光从天而降，却是天道的奖励！
<br/>　　那团功德神光九成直接降入鲲鹏体内，一成洒向洪荒大地。顷刻间，经过功德神光的入体，鲲鹏发现以前很多不懂的地方，一下子全部明白了。修为如坐火箭一般，一路飙升到准圣初期巅峰。
<br/>　　鲲鹏大喜，接着闭关修炼。
<br/>　　鲲鹏这一番动作，却是落入了洪荒众多大神通者眼中，众修士纷纷惊讶不已。这鲲鹏竟然如了得，竟能够得到如此多功德，他们是不是也要如鲲鹏一样，试着创造别的东西？
<br/>　　且不管别人的心思，鲲鹏这一番作为，却让帝俊与太一脸色变得难堪，自己等人刚收服众多妖族势力，也不见什么天道功德，你鲲鹏一下子创出个妖文，就得到如此多功德，而且还赢得了所有妖族的尊重？
<br/>　　“大哥，怎么办？”太一脸色阴沉，看向身旁同样一脸不愉的帝俊。
<br/>　　帝俊阴沉着连，沉默了一会儿，道：“这样也好，我们便以此邀请鲲鹏加入妖族，其他妖族就更会同意了，到时候只要我们给他一个‘妖师’之位，想必他也不会不识趣了。”
<br/>　　“‘妖师’？他会同意吗？”太一道。
<br/>　　“他不同意也得同意，走，我们去请女娲与伏羲两位妖圣，有了他们的帮助，鲲鹏会识趣许多。”
<br/>　　……
<br/>　　同时，在巫族之中，鲲鹏的做法也被他们看到了，于是，他们也学着鲲鹏，经过长时间的研究，终于创出了巫族的文字传承——“巫文”！
<br/>　　天道依旧降下功德，只不过，这功德并没有鲲鹏道人那般多。只因为，第二永远没有第一多。
<br/>　　……
