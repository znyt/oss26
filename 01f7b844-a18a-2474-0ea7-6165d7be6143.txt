　　第490章
<br/>　　会议从山寨的生产问题，转而开始讨论未来的军事战略。
<br/>　　帝国的未来，必定打得一团乱麻。但因为有凉州城的屏障，目前倒也不用担心谁会打到蛤蟆城。
<br/>　　但蛤蟆城……也就是以前的琴川关，位于帝国和戎族接壤处，是戎族入侵帝国的必经之地。当初的琴川关，目的便是为了防卫戎族入侵，保护凉州土地。
<br/>　　假若戎族会打过来，大家该如何应对？
<br/>　　“目前戎族那边的情况如何？”程大雷询问和珅。
<br/>　　在大家去并州搞事情的这段时间，山寨的大小事情都是由和珅管理。包括生产、生活、军事以及情报工作。
<br/>　　他清了清嗓子站起来，道：“我们和附近几个戎族部落的关系一直不错，互相有货物往来。因为我们卖过去的粮食、盐铁、布匹等物，他们的日子倒还过得下去。不过其他戎族部落，日子就有些惨了。各部落之间厮杀部落，也有帝国的败兵逃到草原，在草原上烧杀劫掠为生，如今战乱已有向附近蔓延的趋势。”
<br/>　　“大当家，我还得到一个情报。”阿喜道。
<br/>　　“什么事？”程大雷问。
<br/>　　“情报是我们从草原撤退的途中听到的，据说草原上，有一伙新势力正在崛起，为首人名叫野原火，虽然才出现短短时间，但手下聚拢许多人，洗劫了戎族很多部落。”
<br/>　　“野原火？”程大雷嘴里念叨着这三个字，恍惚是记得在何处听说过，一时却也想不起来。他摆摆手，道：“戎族越乱越好，这样他们才没时间来骚扰我们。不过，还要提防有流贼过来打我们的主意。至于野原火，要持续关注他，有他的任何情报，第一时间递到我面前。”
<br/>　　“是。”阿喜干净答应。
<br/>　　这场会议从开始到结束，持续了很长时间。通过这次会议，基本敲定了山寨以后的发展策略。首先就是发展生产，包括种植、养殖以及生活必须用品的生产；其二就是壮大军事，扩充兵员，加强训练，为未来有可能到来的战斗做准备。
<br/>　　在程大雷离去的这段时间，山寨由和珅管理，和珅的工作做得不错，山寨一直有条不紊的发展。
<br/>　　在那次会议结束后，山寨立刻投入轰轰烈烈的生产运动。开垦荒田，种植水稻、小麦、大豆、高粱等等作物。凉州的土壤是很肥沃的，只是因为连年战乱的缘故，地广而人稀。
<br/>　　程大雷亲自抓这件事情，带着手下人一起垦荒，开荒种地的事情，他本不擅长。不过程大雷有把子力气，种地的事情，只要力气到了，总也出不了太大差错。
<br/>　　而程大雷也只是带个头，鼓舞一下大家的士气，他再如何说也是琴川之主，总也不需要天天在田地里扎着。
<br/>　　这一日下午，程大雷从城外回来，刚在城主府洗掉身上一身汗水，还没来得及和苏樱逗两句闷子，和珅就赶着过来汇报事情。
<br/>　　“大当家，现在山寨都在开荒，包括山寨的五支队伍也都有任务，不过，如此大面积耕种，急需牲畜和犁锄头等工具，现在这些咱们山寨都缺。”
<br/>　　“需要的工具，让工匠坊打一些便好，至于下田的牲畜，龙字骑不是有战马么，难道不能下地干活。”程大雷有些不耐烦，和珅不至于这种小事都处理不好。
<br/>　　“我来找大当家就是为这件事。大当家有所不知，龙字骑对他们的战马看得比媳妇都宝贝，当然，他们大部分人没有媳妇。我去找赵将军，赵将军差点大嘴巴抽我，这件事，我可是管不了。”
<br/>　　“喔……”程大雷恍然大悟，原来和珅是来告小状的，他安抚道：“和大人，你也理解理解，他们是把脑袋别在裤腰带上干活的，脾气暴些也正常，毕竟你是文职官员。”
<br/>　　“那牲畜的事情怎么处理？”
<br/>　　“这件事情你不用管了，本大家只有妙计，天黑之前，必然有你需要的牲畜。”
<br/>　　山寨升级成为城池后，已可从系统商店兑换马匹，每二十万恐惧值可以兑换一匹黑鬃马。程大雷在并州走了一趟，杀人如割草，狠狠的收割了一波恐惧值，如今，他也算得上富裕人士。
<br/>　　在和珅离去后，程大雷便开始通过系统商店兑换马匹。一连兑换了一千匹，大概觉得数目够了，总也要节省一下恐惧值做其他用处。
<br/>　　和尚离了城主府，心中仍旧有些郁郁不乐。大当家说得轻松，可他究竟有什么主意，也不告诉自己呐。世间哪有那么轻松的事情，还真以为他是仙人之徒么。
<br/>　　看了看日头，距离天黑不远了，就该知道，大当家有时候说话也并不靠谱。
<br/>　　正这个时候，和珅忽然看到所有人都往城门口跑，有一人还差点撞在他身上。他抓住那人的肩膀问：
<br/>　　“发生什么事了？”
<br/>　　“马，来了很多马。”这人表情有些亢奋。
<br/>　　和珅一惊，也随着人潮往城门口去，他登上城头，秦蛮、赵子龙等几人已经到了。大家的视线都望着远处的，只见不知从哪里跑过来一群马，统一的黑色，像一面面黑旗，跑得整个大地都震动了起来。
<br/>　　和珅整个人呆住了，揉了揉眼睛确认这不是自己的幻觉，这些黑马奔来，却也不冲撞人群，就聚在城门下，有人大着胆子去摸它们的鬃毛，它们也不过是打声响鼻。
<br/>　　和珅想起程大雷说的话，耳边嗡地一声响，在心底发出一声惊呼：“大当家当真是仙人之徒。”
<br/>　　他抬眼看了看西方，太阳也不过刚刚有些倾斜，天还没有黑。
<br/>　　秦蛮发现了他的异样，感觉和珅立在那里，浑身瑟瑟而抖，脖子后面一层又一层的冷汗。
<br/>　　“和大人，你怎么了，是身体不舒服么？”秦蛮拉了他一把。
<br/>　　“没什么，没什么。”
<br/>　　和珅如梦初醒，摇摇脑袋，将额头的冷汗擦去。
<br/>　　“快快派人，将这些马匹带到城里，这可都是仙人所赠。”
<br/>　　通过这件事，和珅算是对程大雷心服口服，此生不敢有二心。
<br/>(本章完)
