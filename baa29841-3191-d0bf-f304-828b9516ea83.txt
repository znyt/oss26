　　随后，一个赫包被打伤的肥胖喰种被押解了进来。
<br/>　　而此时的BigMadam，已经没有了原著中贵妇的样子，她身上那些价值非凡的昂贵首饰，都被浸染上了一些血液，在灯光的映射下，显的通红。
<br/>　　注意到什造不知所措的表情，刘哔心中更加不开心了，对这些押解BigMadam进屋的人，他没有一点客气，挥手之间就捏碎了他们的头颅。
<br/>　　紧接着，押送者们的血液溅射到BigMadam脸上，像是带有强烈腐蚀性一般，将BigMadam脸庞腐蚀出许多坑坑洼洼的小洞。
<br/>　　刘哔有些奇怪，这些押送者的个体实力并不强大，最高的也只是S而已，凭借他们，是怎么抓到SS级喰种BigMadam的。
<br/>　　痛彻心扉的惨叫声响起，BigMadam的墨镜被击毁了一半，露出来的一只赫眼中，充满了痛苦。
<br/>　　“妈……妈妈……”什造站在刘哔身后，怯怯的喊了一句。
<br/>　　BigMadam的惨叫声微微有些减弱，充满血丝的赫眼，惊疑不定的望着什造。
<br/>　　揉了揉什造的头发，刘哔走到诗的身前，指了指包厢大门，做了一个请的手势：“走吧，我不希望有人打扰到她们。”
<br/>　　虽然自己的手下被屠，但诗脸上并没有任何反应，听见刘哔的话语后，脸上露出一个礼貌的笑容，两人双双走了出去。
<br/>　　很快，包厢很快安静了下来，唯一存在的声音，就只有BigMadam微弱的呻吟声。
<br/>　　感受到身边的空旷，什造目光深处出现一抹慌乱，她有些责备，为什么刘哔要将她一人留在这里。
<br/>　　“是……什造啊，真没想到……你……还活着。”BigMadam摇摇欲坠的依靠在墙壁上，鲜血肆意的从伤口中流出，落在了脚下的大理石地面上。
<br/>　　BigMadam的语气有些虚弱：“能不能……帮妈妈解开手铐啊，这样……妈妈好难受。”
<br/>　　看了一眼BigMadam手腕上亮晶晶手铐，什造清楚，这是能限制喰种能力的特质手铐，是CCG专门研制出来，用以抓捕身份特殊的喰种使用。
<br/>　　但很奇怪，为什么这些喰种，会有这种手铐，而且还知道该如何使用。
<br/>　　即使是CCG，也只有个别小组才会配备，需要使用时，也必须提前向总部申请。
<br/>　　因为，造价实在是太昂贵了，除了一些珍稀材料之外，还要用到不低于S级喰种的高阶赫包，故此才无法在内部范围推广。
<br/>　　“什造啊，为什么不听妈妈的话，你在妈妈心中，可一直是一个乖巧懂事的好孩子啊。”BigMadam挣动了一下镣铐，心中有些开心。
<br/>　　他今天只是来参加一场拍卖会，竟然会被莫名其妙绑在这里，可没想到，竟然幸运见到自己几年前收养的‘表演者’。
<br/>　　当初，铃屋什造可是他最满意的杰作，光靠什造，他每个月都能获得大笔的金钱。
<br/>　　那些喰种们，最喜欢看这种小萝莉虐杀人类的反差表演了。
<br/>　　虽然现在他被限制住了能力和行动，但是没关系，BigMadam相信什造会帮他解开的。
<br/>　　对于这些被洗脑了的表演者来说，是绝对不会违背主人意志的，甚至可以说，这这种表演者，几乎已经失去了自有思想。
<br/>　　果不其然，在BigMadam满意的目光下，什造走到了他的身前，从地面的尸体上，摸出了一把银白色的钥匙。
<br/>　　脱离自己身边了的表演者，果然不可以信任，只是一个解开手铐的命令，竟然会犹豫这么长时间，简直不可饶恕。
<br/>　　嘴角勾起一抹残忍的笑容，BigMadam转过身来，等待什造解开束缚他的镣铐，然后，杀掉什造。
<br/>　　……
<br/>　　包厢门外，刘哔靠在窗边，口中衔着一根日式的香烟。
<br/>　　这种烟口味太淡，他抽起来并不是特别习惯，但条件所限，他也只能抽一抽这个。
<br/>　　“喂，你为什么杀掉我的手下啊，BigMadam……是你的朋友？”站在窗口，诗眺望着远处的夜空，有些奇怪的对刘哔开口。
<br/>　　实力相差这么悬殊的人，他可不知道是如何成为朋友的。
<br/>　　即使再像，喰种和人类也是不同的，在他们的世界里，弱者只会成为强者的食物或者仆从，而永远都不可能成为朋友。
<br/>　　“没什么理由，想杀便杀了，事实上你得感谢他们，要不是杀到他们让我平复了一下心情，估计要死的人就会变成你了。”
<br/>　　吐了一口烟，刘哔语气有些轻松，似乎是再说一件和他没有什么关系的事情一般。
<br/>　　面对这样毫不客气的话语，诗也没有生气，从刚才突然之间的动手看来，他根本不会是刘哔的对手。
<br/>　　窥一斑可见全豹，和修宗太的死，也不是偶然啊。
<br/>　　“对了，副首领让我来问你，要不要加入我们组织，不过我想估计，你应该不会加入吧？”想到自己来包厢的目的，诗脸上露出一抹苦笑。
<br/>　　这样让人为难的差事，该怎么去完成啊，这根本就没有半点希望。
<br/>　　“知道还问。”刘哔撇了撇嘴，随手丢下了手中抽剩的烟头：“说起来，我也有一件事情找你，你有没有兴趣，成为我的下属？”
<br/>　　看着尚未熄灭的烟头从窗边落下，橙黄色的火光在夜幕中划过一条优美的线条，煞是好看。
<br/>　　刘哔在思考，怎么处理小丑组织成员的事情。
<br/>　　对于白日庭那边，他不是很放心，毕竟自己不可能一直呆在这个世界里，恐怕有很多事情，他都不会顾及的到。
<br/>　　这也是他忍下心中的杀意，将这家伙留下活口的原因。
<br/>　　毕竟是十分重要的养猪场，他想在这个世界培养几个监管者，而小丑组织，就是一个十分不错的底子。
<br/>　　当然，他也要找个机会，将和修家族这个病毒给铲除掉。
<br/>　　“所以，连凶名赫赫的修罗，都对我们小丑感兴趣了吗？”叹了口气，诗的表情有些无奈。
<br/>　　树倒猢孙散，像他们这样的大型势力，实力削弱之后，自然会引来很多人的惦记。
<br/>　　毕竟，发展了这么久，他们拥有的资源，是一个极为恐怖的数字。
<br/>　　就像怀中揣着珍宝的失势富翁，或者说一个走在街上没有一点防备的赤裸美人，根本没人能抵挡住这样的诱惑。
