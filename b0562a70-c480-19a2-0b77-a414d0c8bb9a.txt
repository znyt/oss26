　　　　叶少阳招呼大伙朝山谷里退去，至少三面环山，不至于被人围攻，腹背受敌。
<br/>　　加入了这数十个人之后，空界联盟在人数上瞬间呈压倒优势，为了仿佛道风突围，将谷口几个出口完全封堵住。
<br/>　　“林长生！你也在这！”建文帝在人群中发现了一个熟面孔，是自己生前当皇帝的时候，最为赏识的一个道士，靖难之役后，自己一路逃亡，跟林长生再也没见过。没想到居然在这里碰上了，心中不免有些感慨，叹道：“当年我就觉得你有些法力，没想到你真的修成正果了。”
<br/>　　林长生拱了拱，笑道：“陛下，许久不见，我也是没想到还能见到你，一路不易，没想到你居然从贼，与转世鬼童厮混成一伙。”
<br/>　　建文帝面色微变，想要发作，最后还是控制住了。沧海桑田，他现在已经不是皇帝，林长生也不是他的臣子了。
<br/>　　林长生继续笑道：“你我总算是君臣一场，不想刀兵相见，你离去吧。”
<br/>　　建文帝冷笑不语。
<br/>　　黎山老母环视对面众人，说道：“在场所有人，除贼首道风之外，余者都可自行离去，概不追究。”
<br/>　　道风转头朝叶少阳看过来，道：“带上你的人，走。”
<br/>　　叶少阳耸了耸肩，用不屑作为回应。
<br/>　　道风也没有坚持。
<br/>　　在场的所有人，没有一个人离去，甚至连这个念头也没有动过。
<br/>　　一个和尚宣了声佛号，说道：“我等都是为了诛杀鬼童而来，圣母你与仙子口口声声说道风是鬼童，是时候验证了吧！”
<br/>　　黎山老母转头朝苏沫看去，道：“时机到了，还等什么。”
<br/>　　苏沫一脸狞笑望着道风，从衣服里拿出一只铜铃，就要摇动。
<br/>　　道风见到铜铃，瞳孔一缩，人影狂掠出去，尽管速度极快，但对方阵营也都不是善茬，黎山老母离的最近，手一挥，一串花瓣从袖子里飞出，朝道风击打过来。
<br/>　　普法天尊也作法打出数张玉符，还有附近几个人来得及出手的，也都纷纷出手相救。
<br/>　　道风身后，猛然飞出五道人影，是十二门徒中的几个，与杨宫梓一起扑向苏沫。
<br/>　　叶少阳之前经过广宗天师帮忙调息，法力也恢复了大半，见这情景，也扑了上去。
<br/>　　黎山老母祭出的花瓣雨，被风之谷几大门徒挡住，顿时惨叫声起，五个人有三个在看似美丽柔弱的花瓣雨中，被绞杀成了一堆精魄，另外两个也重伤而退，不过五个人集中起来的力量，也消耗了花瓣雨的法力，道风从中间穿了过去，黎山老母招式已老，再想出招已经来不及了。
<br/>　　另外一边，杨宫梓也用混沌真气裹住了普法天尊释放的灵符，相杀起来。
<br/>　　这一瞬间之后，更多人反应过来，一拥而上，目标很明确，也不颤抖，只是为了帮助道风架住攻击，腾出空间。
<br/>　　叶少阳因为速度慢了半拍，没赶上迎击，干脆挥舞勾魂索，主动袭击那个李浩然，一来早就看他这副装逼的样子很不爽，二来知道他厉害，也是想先提前锁住他，免得他对道风出手。
<br/>　　勾魂索从头顶扫下来，李浩然巍然不动，勾魂索正中脑门的一瞬间，李浩然的身体突然平移出几米远，堪堪躲过了攻击，带着微笑，冲叶少阳平出一掌，中指弯曲，捏成一个法诀，看上去波澜不惊，但叶少阳却觉察到了危险，左手也捏了一个法诀，运用了八成罡气，迎击过去。
<br/>　　两只手掌击打在一起，却没有产生反震之力，而是紧紧贴在一起。
<br/>　　叶少阳立刻感到一股吸力从对方手掌传递过来，似乎要把自己的神识吸走，急忙作法顶住，但吸力太强，只能苦苦支撑。
<br/>　　“当初灭我宗门，你也有份，我也该好好谢谢你。”
<br/>　　“不用谢，我做好事从来不留名，叫我雷锋。”叶少阳咬紧牙关说道。
<br/>　　李浩然微笑着，手上的吸力越来越强。
<br/>　　“放开他！”小九飞身赶来，凌空拍出一掌，在身前幻化出一只巨大的狐爪，朝李浩然头顶打来。
<br/>　　李浩然抬起右手，将金刚琢掷了出去，口中念咒，金刚琢立刻幻化出数倍大小，在头顶旋转，黑光四溢，将狐爪顶开。
<br/>　　小九人已飞来，正要施展第二轮攻击，突然一声惨叫传来，三人一起转头看去，只见苏沫身体倒飞出去，在跌落的瞬间，将手中的铜铃朝师弟张云扔了过去。
<br/>　　她是被道风一击所伤。
<br/>　　道风立刻撇开他，朝张云飞过去。
<br/>　　铜铃需要用灌入法力，才能激活灵气，被摇动发声，因此在空中的时候是不会响的。张云接住铜铃，刚要摇动，突然一道青光从天而降，急忙摸出一把长剑，凌空划了几下，布置成一道法力结界，想要护住身体。
<br/>　　道风根本没有客气，挥动打神鞭，直接砸了下去，结界只抵挡了几秒钟不到，便破碎了，打神鞭重重地击打在他脑门上，直接打碎了脑壳，脑浆迸裂。
<br/>　　张云的魂魄，也直接被打碎了，精魄从脸上七窍中飞出，魂飞魄散……
<br/>　　在被击中的一瞬间，张云近乎本能地把铜铃扔了出去。
<br/>　　道风立刻起身去追，却被一道远处飞来的不定符拦了一下，灵力充盈，一看就是出自至少地仙境界的人之后。
<br/>　　道风打碎不定符，再去抓铜铃，毕竟已经晚了一步——
<br/>　　李浩然松开了叶少阳，猛然后退，已经牢牢抓住了铜铃，抽身急退，黎山老母等人也缓过手来，也不主动进攻，立刻挡在李浩然身前，合作布置了一道强大的结界。
<br/>　　“现身吧。”李浩然冲道风笑了一下，作法摇晃铜铃。
<br/>　　再没有办法阻拦了，大伙感到已经听见了铃声，然后就在铃声响起的瞬间，突然想起一声石破天惊的道：“虚无有法！”
<br/>　　声音如同炸雷一般，震耳欲聋，也盖住了铃声。
<br/>　　众人转头看去，道号却是广宗天师发出的。
<br/>　　广宗天师这一嗓子虽然破了铃声，但自己似乎也非常累，用力吹着胡子，对李浩然道；“祖师何必呢。”
