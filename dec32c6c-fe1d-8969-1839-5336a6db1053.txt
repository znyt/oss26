　　Santiago移动到那名受伤的“黑水”队员身边的时候，那名大腿股动脉受创的“黑水”队员已经没有了呼吸。
<br/>　　这名股动脉受创的“黑水”队员靠在一辆轿车上，他缠在左大腿上的绷带已经完全被血浸透，他身底下流着一大片的血，他的脸很白，是严重失血后所造成的惨白。
<br/>　　Santiago伸手搭在那名“黑水”队员的颈动脉上，确认他已经没有脉搏后，按住耳麦对David说道：“队长，Joe挂了。”
<br/>　　“什么！？”
<br/>　　“我说Joe挂了，他死了！”Santiago拔高声音，对着耳麦说道：“我们已经失去两个人了，难道还要把这个该死的任务继续下去吗！？”
<br/>　　“Santiago，我知道你的想法。”David停顿了一下，说道：“我们没有选择，只能服从！这是洛菲克通过公司高层直接向我们下达的命令，我们必须执行，不然我们会被降级，甚至会被放弃，洛菲克的能量超过我们的想象！”
<br/>　　“Shit！”
<br/>　　Santiago怒骂一声，拉掉挂在自己耳朵上的导管耳麦，伸手将Joe失去生命光彩的眼睛合上，抱着m4a1战术步枪，进行一个战术移动，闪身藏到一辆SUV左侧车轮旁，寻找着进攻的机会。
<br/>　　此时在这个停车场，火光冲天，爆炸声、自动武器的点射声、扫射声，汽车警报声响做一片。因为火光，夜视仪和热成像仪全都成了摆设，失去了作用，双方都只能依靠自己的眼睛搜索藏在停车场中的敌人。
<br/>　　Santiago瞅准机会，从汽车背后闪身而出，用m4a1战术步枪打出一个两连发点射。
<br/>　　哒哒！
<br/>　　噗噗！
<br/>　　子弹打在罗昊的胸口，罗昊觉得自己胸口就像是挨了两记重拳，身体不受控制的倒飞出去，摔倒在地上，向后滑出一道距离。
<br/>　　大意了！
<br/>　　罗昊心中暗呼“不妙”，抬起MPX对着站在自己不远处的Santiago扣动扳机。
<br/>　　咔！
<br/>　　MPX并没有如罗昊意料中的那样打出节奏感的点射，而是发出“咔”的一声轻响。
<br/>　　这不是因为弹匣中子弹打空了，罗昊一直在留意着弹匣中剩余的子弹数量，子弹卡壳了！
<br/>　　在战场上，生死之间，子弹卡壳无疑是致命的，尽管现在各大轻武器制造商对武器使用故障率降低非常重视，可是这并不能彻底消除武器使用故障率。用墨菲定律的角度来说，只要有可能会发生，就一定会发生。
<br/>　　所以，战场上能活下去的人都是好运的，因为他们的武器在关键时刻没有发生故障；因为他们隐蔽的掩体没有被敌人的炮火波及到；因为……
<br/>　　很显然，罗昊这次的运气并不算好，他的武器发生故障了。
<br/>　　原本准备闪避的Santiago此时也是明白发生了什么情况，脸上露出狰狞的微笑，m4a1黑洞洞的枪口指向躺在地上的罗昊。刚才那个点射因为没有精确瞄准，所以被罗昊身上的防弹背心给阻挡了，但是这次Santiago不会再让罗昊从自己枪口下活下来。
<br/>　　罗昊顺势往旁边滚开，右手顺势拔出快枪套中的手枪，对着Santiago连开两枪，令得Santiago动作滞缓了一下后，左手从刀鞘中拔出CQB型格斗刀，甩手扔了出去。罗昊没有指望着这一记飞刀可以直接要了Santiago的命，如果是王小虎或者司徒空的话，或许有可能做到，但是他不行，而且如Santiago这样有着丰富实战经验的雇佣兵，想要用一把飞刀直接要了他的命，实在太难了，所以罗昊这一刀飞向了Santiago的脚。
<br/>　　一阵尖锐的刺痛传来，Santigao脚下一个踉跄，低头看到扎在自己大腿上的CQB格斗刀，Santiago顿觉不妙，但是所幸CQB格斗刀的刀刃比较窄，造成的创口也比较细小，这一刀也只是皮肉伤。
<br/>　　Santiago握着m4a1战术步枪，忍着从大腿上传来的痛楚，脚下保持着战术步伐，向前移动，可是地上已经看不到罗昊的身影了。
<br/>　　“该死！”
<br/>　　Santiago正准备后撤一道人影从他身旁的汽车后面飞扑而出，一把抱住他，撞向他左手边那辆轿车。
<br/>　　砰！
<br/>　　轿车车门被撞得凹陷下去，车窗玻璃也是出现了蛛网状的裂纹，轿车警报系统发出尖锐的警报声。
<br/>　　罗昊提膝一脚顶在Santiago命根子，瞬间让Santiago瞪大了眼睛，身体弓起；罗昊左右手连出数拳击打在Santiago腹部，右手一个肘击打在Santiago脸颊上，将他嘴里两颗牙齿打落，紧接着右手食指勾住了扎在他大腿上的CQB格斗刀刀柄上的指环，反手一刀，扎在Santiago脖子上。
<br/>　　噗嗤！
<br/>　　随着罗昊拔出CQB格斗刀，一道血箭从Santiago脖子上飙射而出。Santiago捂着自己的脖子，踉跄着脚步倒在地上，可即便是受了这致命的创伤，Santiago还想着能在自己生命的最后时刻拉上一个垫背，他的手伸向腰间的手雷包。
<br/>　　Santiago的动作并没有瞒过罗昊的眼睛，罗昊也根本不会给他这样一个机会，将右手的CQB格斗刀甩向Santiago左肩，顺手拔出手枪对着Santiago的眉心就是一枪，血花飞溅而起，一颗手雷从Santiago左手滚落了下来。
<br/>　　那颗从Santiago左手滚落下来的手雷保险销已经被拔出，只差最后将手雷的保险夹拆掉。
<br/>　　“妈的，差点就炸了。”
<br/>　　罗昊收起手枪把那颗手雷捡了起来，重新插上保险销，并且将Santiago左肩上的CQB格斗刀给拔了出来，收回到刀鞘中。
<br/>　　罗昊找回自己那把MPX冲锋枪，拆下弹匣，拉动枪栓将卡在枪膛中那颗子弹给退了出来，嘴里嘀咕道：“这外国人的东西，也并不是万无一失嘛，老子差点就没命了。”
<br/>　　在狙击手Henry、突击手Joe之后，Santiago已经是第三个阵亡的“黑水”队员了。
<br/>　　David的小队是一个标准的战术小队人员编制，加上David总共十二个人，而现在他们的人员损失已经达到了四分之一，不仅如此，他们的活动范围也是在被进一步压缩。
<br/>　　“队长，Santiago也挂了！”
<br/>　　David的耳麦中再次传来一个他最不愿意听到的消息，David明白自己这边不能再这么耗下去了，他必须跟公司取得联系，让公司方面联系洛菲克取消这次行动！
