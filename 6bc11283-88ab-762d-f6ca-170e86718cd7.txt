　　“竟然真的成了。”
<br/>　　萧逸看着指尖那抹乳白色力量，脸色惊骇。
<br/>　　要修炼出剑力，其本身，说白了便是沟通天地之力，寻到其中的剑力意味。
<br/>　　剑力法诀的存在，便是等同一种帮助与指引。
<br/>　　越是高品阶的剑力法诀，便容易沟通天地之力，堪破其中的剑力意味。
<br/>　　就如同修炼功法，越是强大的功法，便越容易让武者感悟天地武道，从而突破境界。
<br/>　　而刚才，萧逸清晰地看到，在自己催动武魂，尝试沟通天地之力找寻其中剑力意味时，竟是瞬间成功了。
<br/>　　这也是他瞬间凝聚出剑力的原因。
<br/>　　“难道是我更适合修炼这一种剑力？”萧逸皱了皱眉。
<br/>　　事实上，据他在风刹殿的查阅卷宗所知，剑域的剑者，无论修炼哪一种剑力都行。
<br/>　　所谓的适合修炼哪一种剑力，不过是难度大小的问题罢了。
<br/>　　对于外来武者而言，萧逸同样不认为修炼不了。
<br/>　　既然它是一门修炼体系，剑域剑者修得，外来武者也必然修得，其区别，不过同样是难度大小罢了。
<br/>　　而自己，或许是元力和肉体方面，修炼其剑力来，比较困难。
<br/>　　反倒是武魂修炼剑力，轻松简单。
<br/>　　萧逸也只得这般认为。
<br/>　　想罢，萧逸再次闭目修炼。
<br/>　　再次催动武魂，沟通天地之力。
<br/>　　又是这一瞬间，轰…天地间，一阵轰鸣。
<br/>　　这一次，萧逸没有因为惊骇而停下修炼，而是一边沟通天地之力，一边感知着剑力的存在。
<br/>　　这般细细感知下，萧逸再次大吃一惊。
<br/>　　此时，武魂已与天地中的剑力意味沟通连接。
<br/>　　而他，也清晰感知到了存在于天地中的剑力。
<br/>　　但，感知中所观景象，却让他脸色大惊。
<br/>　　在他的感知中，一缕缕白色光芒，清晰充斥于天地中。
<br/>　　这些白色光芒，并非无规律，而是极其有规律地存在着，跃动着。
<br/>　　这些白色光芒，便是剑力。
<br/>　　剑力，充斥了整个剑域范围。
<br/>　　但剑力，却也并非静止着，而是不断游走。
<br/>　　四面八方的剑力，此时正朝着剑域的中心涌去。
<br/>　　而从剑域的中心范围，又不断反涌着剑力分散于四面八方。
<br/>　　剑力的游走，这一缕缕白色光芒，仿佛成一半椭圆形状，覆盖了整个剑域。
<br/>　　“嘶。”萧逸看了眼天地间的剑力，倒吸一口凉气。
<br/>　　他终于明白为何剑力如此奇特强大。
<br/>　　也终于明白，为何他之前强行感悟，竟遭受反噬吐血。
<br/>　　难怪他感悟时，会感觉到剑力的意味若有若无，或几乎不存，或充斥天地。
<br/>　　这剑力，根本就是充斥了整个剑域。
<br/>　　但同时，剑力，根本算不得上天地武道之力。
<br/>　　它，不过是拥有着一丝天地意味罢了。
<br/>　　简单来说，剑力，是只存在于剑域之中的特殊力量。
<br/>　　出了剑域，天地间将再无剑力这种特殊力量。
<br/>　　只是，到底是什么东西，竟然能将力量覆盖了整个剑域？
<br/>　　“是万剑碑林。”萧逸双眼一眯，直视远方。
<br/>　　剑域之内，四面八方的剑力涌动方向，悉数朝着的，正是剑域的中心范围。
<br/>　　那里，也正是万剑碑林的所在地。
<br/>　　“万剑碑林，果真不简单。”
<br/>　　“看来，此番前往万剑碑林，会有惊喜也不一定。”萧逸笑笑。
<br/>　　想罢，萧逸再次摒除杂念。
<br/>　　趁着还有数天时间，他自是多修炼些剑力。
<br/>　　剑力这等特殊力量，在剑域中的威力如此强大，他怎可能放弃。
<br/>　　“凝。”萧逸心头低喝一声。
<br/>　　指尖，再次凝聚出一抹白色光芒。
<br/>　　只不过，这丝白色光芒，渐渐地，又化作了火红色光芒。
<br/>　　萧逸内视体内。
<br/>　　他此刻，本是催动体内两大武魂，沟通天地间的剑力意味。
<br/>　　而很显然的，两大武魂，都轻松地便沟通成功，也修炼出了剑力。
<br/>　　不过，冰鸾剑修炼出的剑力，还未来得及凝聚至指尖，便已被冰鸾剑吸收殆尽。
<br/>　　唯独控火兽，虽然面容呆呆傻傻，但却仿佛‘任劳任怨’，不断修炼出剑力，随后被萧逸体内吸收。
<br/>　　这些剑力，由武魂修炼而出。
<br/>　　既不属于气剑力一脉，亦不属于器剑力一脉。
<br/>　　真正要算的话，便是魂剑力吧。
<br/>　　萧逸试了试。
<br/>　　指尖一抹剑力涌出，同时，星光剑气凝聚。
<br/>　　剑力，融合入剑气之内。
<br/>　　嗤…
<br/>　　指尖一动，一道剑气破空而出。
<br/>　　空气中，仅仅出现了一声‘嗤’声。
<br/>　　剑气，一闪而过，横跨数十里。
<br/>　　沿途所过，无物再存，已是一片虚无。
<br/>　　树木、山石、地面，一瞬间在剑气之下湮灭成齑粉。
<br/>　　“咕。”萧逸不禁咽了口口水。
<br/>　　单单是这一道剑气，其威力就不亚于一剑出三十剑齐至的威力，甚至稍强一些。
<br/>　　萧逸霎时脸色大喜，连忙盘膝坐下，继续修炼。
<br/>　　不过，才堪堪闭目。
<br/>　　远方，已是数道身影破空而来。
<br/>　　“嗯？”萧逸看了眼远方，双眼一眯。
<br/>　　“六个武道大能？”萧逸眉头一皱。
<br/>　　他接触的武道大能层次武者，已然不少了。
<br/>　　自然的，单凭袭来的气息，便能知晓来人修为。
<br/>　　嗖…嗖…嗖…
<br/>　　几乎是数个呼吸间。
<br/>　　六道身影便横跨万里，凭空而现。
<br/>　　来人，是六个老者。
<br/>　　“气宗六剑侍。”萧逸脸庞一冷。
<br/>　　这六个老者，他是第一次见，但却知晓他们的身份。
<br/>　　之前，在寒光府上空，遇到的六个老者，乃是器宗六剑侍，器宗之内的六位剑道大能。
<br/>　　而现今出现的这六位老者，则是气宗的剑道大能。
<br/>　　这六人，他虽第一次见，但却在风刹殿的卷宗中看过他们的画像，自是认了出来。
<br/>　　“小子，有些眼力。”六个老者，冷笑一声。
<br/>　　“诸位有事？”萧逸淡漠问道。
<br/>　　“你说呢？”六个老者面容冰冷。
<br/>　　“堂堂萧逸小贼，果真闻名不如见面。”
<br/>　　“只是，你太爱管闲事了些。”
<br/>　　“我气宗的事，你也敢管？”
<br/>　　话音落下的瞬间，六个老者，陡然杀意凛然。
<br/>　　萧逸见状，同样面露杀意，“看来是来报复了。”
<br/>　　“话，我只说一遍，要么滚，要么死。”
<br/>　　“猖狂。”六剑侍暴喝一声，瞬间出手。
