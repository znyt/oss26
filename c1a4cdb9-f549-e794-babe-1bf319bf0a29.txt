        &lt;h3&gt;第三卷第一百一十三章  原谅过去&lt;/h3&gt;
<br/>
<br/>        “就是我们当初的学霸------陈子昂！欢迎陈子昂同学回归！”
<br/>
<br/>        夏颜和呼延临庆感觉自己的脸上火辣辣的“啪”了一声。他们几十个同学就为了等陈子昂的到来？
<br/>
<br/>        “陈子昂？”大厅里瞬间安静了下来，人们都齐齐的站起来，看向门口。
<br/>
<br/>        党桂芳已经朝门口跑过去了，穿着高跟鞋因为跑动，差点摔倒。
<br/>
<br/>        陈子昂穿着黑色宽松v领毛衣，白色直筒裤，黑色的低跟鞋，外搭浅灰色紫貂短款大衣。
<br/>
<br/>        提着黑色限量款的lv、扎着马尾，不施粉黛的出现的门口，扎着的马尾随着她的走动在阳光下摇曳。
<br/>
<br/>        高中毕业到现在除了傅海生，他们都没有见过她。
<br/>
<br/>        “这是哪个穷屌丝陈子昂吗？”党沐悄悄的问呼延临庆和夏颜。
<br/>
<br/>        “哼，说不定身上的衣服都是劣质货、冒牌货。”看着陈子昂那身名牌刺疼了党沐的眼。
<br/>
<br/>        夏颜是真知道陈子昂有公司的，但是她不会给呼延他们说，她就想看他们羞辱陈子昂。
<br/>
<br/>        “就她那样能穿得起名牌？肯定都是假的，故意在我们面前穷扎势呢。”夏颜说。
<br/>
<br/>        “我最讨厌那种没钱还扎势的人了。”
<br/>
<br/>        陈子昂伸手扶住差点摔倒的党桂芳。“还是那么毛毛躁躁的。”
<br/>
<br/>        “哇，陈子昂你这个坏女人，你这几年跑哪里去了，你说、你说啊。”党桂芳直接就哭出声。
<br/>
<br/>        “我在呢，一直都在呢。”俩个人拥抱在一起，陈子昂也热泪盈眶。
<br/>
<br/>        党桂芳比陈子昂本来就低了，现在陈子昂穿上高跟鞋，整个人比党桂芳高了很多，党桂芳感觉自己失态，她抬起头看着陈子昂。
<br/>
<br/>        皮肤还是那么好，眉眼也没变，还是有小酒窝，就是高了很多，她得昂着头看她。
<br/>
<br/>        “你现在怎么这么高了？”党桂芳拉着陈子昂的手。
<br/>
<br/>        “这么多人站着呢。”陈子昂笑着说。
<br/>
<br/>        “我不管，周宏生叫的人，让他招呼去。”党桂芳霸道的说，因为他们是同学，也会聚会，彼此都很熟悉了，所以党桂芳才能这样说。
<br/>
<br/>        “欢迎我们的学霸回归。”掌声一片，啪啪的打在夏颜的脸上，夏颜紧了紧拳头。
<br/>
<br/>        同学们一股脑的都涌过来，围在陈子昂的周围。
<br/>
<br/>        大家都一样很好奇陈子昂这么多年去了哪里，现在他们有的工作，有的做个小生意，有的回家种地，各行各业的都有。
<br/>
<br/>        “让一让，大家都很好奇陈学霸这几年去哪里了，来，我来采访一下。”一个男同学直接拿过话筒。
<br/>
<br/>        “陈子昂，这么多年，你去了哪里？为什么不和我们联系？”
<br/>
<br/>        “我从高中毕业，在一个地方当了三年的老师，然后去北京了，这几年一直都在北京。之前因为通讯不方便，后来是没有各位的联系电话。”陈子昂微笑着说。
<br/>
<br/>        “原谅你了，还有没有人问问题？”这似乎是一场采访。
<br/>
<br/>        “我来，老陈，你说你当初学习那么好，怎么就没有考上大学？你是不是考试的时候睡着了？”又来一个同学，语气里是惋惜还有一点点责怪，陈子昂看了一会。
<br/>
<br/>        “我说，你真成圆的啦？”陈子昂看了半天突然笑了，他们经常在一起打球的小胖子。
<br/>
<br/>        “还不是你这个乌鸦嘴，快点告诉我原因，要不然罚你和我一样成圆的。”
<br/>
<br/>        陈子昂的心里微微有些感动，时光再怎么流逝，那曾经的美好依然存在。
<br/>
<br/>        “真要听？”
<br/>
<br/>        “真要听。”
<br/>
<br/>        “当时我榜上无名，因为我的资料被人改了，某人现在的人生应该就是我的。”
<br/>
<br/>        “啊！”大家一片惊呼。
<br/>
<br/>        “同学们，事情已经过去了很多年，你们不问我也不会说，无论以前怎么样，我都已经不在乎了，希望大家都幸福，更希望取代我的人要幸福，要不然白白浪费了我另一种人生。”陈子昂既不看傅海生也不看夏颜。
<br/>
<br/>        呼延临庆和党沐听见陈子昂的话吃惊的看了一眼夏颜，不自主的朝后退了一步，但是呼延临庆又朝前走了一步，和夏颜傅海生站在一起。
<br/>
<br/>        陈子昂不想再和傅海生、夏颜有纠葛，陈子霖告诉了傅海生的一切，她从傅海生的眼神也能看出来傅海生对她那种炽热的情感，傅海生已经结婚生子，无论当初对她是什么样的情感，现在他都不应该对她存在任何一种幻想。
<br/>
<br/>        无论他的家庭是怎么样的摇摆不定，和她一点点的关系也没有，她，现在就是他们高中时期的同学，你们抢了我的人生，我原谅了你们，但是人还是有底线的，请不要再来触碰我的底线。
<br/>
<br/>        “子昂，你现在在北京啊，你在做什么工作呢？”另外一个同学问。
<br/>
<br/>        “我做的是信息化。”
<br/>
<br/>        “陈子昂，你就是那个发表了很多论文的陈子昂？”
<br/>
<br/>        “是我没错。”
<br/>
<br/>        “夏颜，你看我就说是我们子昂吧，你还说子昂没有那个本事。”那个同学兴奋的冲夏颜喊。
<br/>
<br/>        “子昂，欢迎你回来。”夏颜笑着说。
<br/>
<br/>        “谢谢。”
<br/>
<br/>        “海生，子昂回来了，你都不会说话了吗？”夏颜拉着傅海生。
<br/>
<br/>        “你们当初可是好了三年的啊。信也写了几百封了吧？子昂你怎么能说你和傅海生没有联系呢。”
<br/>
<br/>        “是吗？那怎么我现在没有和傅海生在一起，而是你和他在一起呢？你什么都喜欢抢啊？抢去的用着舒服吗？”陈子昂一句话将夏颜打得无处遁形，成功的变成了小三。
<br/>
<br/>        “你.....”
<br/>
<br/>        “海生，好久不见，来，叫一声爸爸。”陈子昂看着傅海生笑着说。
<br/>
<br/>        “哈哈哈，粑粑！”大家被陈子昂逗笑了，齐声喊着“粑粑”。
<br/>
<br/>        大家认识的还是哪个霸气的学霸陈子昂，她只是比以前更自信，更有魅力了。
<br/>
<br/>        呼延临庆听说陈子昂在北京之后直接就闭了嘴，他和陈子昂的公司有过接触，第一个项目托关系拿下来，几年都没有搞定，后来一直被陈子昂的公司压制着。
<br/>
<br/>        今天他才知道此陈子昂就是彼陈子昂，他再也不敢说那是个“穷屌丝”了。
<br/>
<br/>        “好了，好了，大家不觉得饿嘛？各就各位，老婆，带着你的闺蜜入席吧。”周宏生邀请大家入席，既然是同学聚会，当然是少不了吃吃喝喝的。
<br/>
<br/>        夏颜本来还想去撩一撩陈子昂的，傅海生说：“夏颜，好歹给你自己留点情面吧，现在的陈子昂你以为还是原来的陈子昂了吗？你以为周宏生是傻子，会花费这么大的气力去讨好党桂芳？
<br/>
<br/>        陈子昂刚刚说的话也很清楚了，她什么都知道，只是她不愿意再去计较了，所以，算我求你不要再去作了，陈子昂并不欠你和我什么，而我也和陈子昂没有你说的好了三年，一切都是你我的自以为是！”傅海生冷冷的说完，直接就去找陈子昂了。
<br/>
<br/>        陈子昂和党桂芳坐一起，俩个人正聊得热火朝天，傅海生拍拍党桂芳的肩膀，党桂芳让人给他加了一个凳子，三个高中时期的死、党又坐在了一起。
<br/>
<br/>        周宏生也坐过来了，四个人就时光的流逝唏嘘不已。
<br/>
<br/>        “子昂，你和寒总、郝总、江总是啥关系？”周宏生终于是逮着机会问陈子昂。
<br/>
<br/>        “朋友啊，还是我公司的大股东。”
<br/>
<br/>        “哎呀，子昂，你太厉害了。认识的人都是人中龙凤。”周宏生笑哈哈的，他们几个的关系果然不一般。
<br/>
<br/>        “别给你脸上贴金了，你不是。”党桂芳笑着说。
<br/>
<br/>        “只要老婆知道我是龙就行了。”狗粮毫无征兆的就撒了一地。
<br/>
<br/>        “桂芳，你不知道啊，那个我们公司的三个大老总啊，那个帅啊，没有办法形容，真的是帅的掉渣！哎呀，帅得掉渣，可惜我不是女人。”周宏生摇头晃脑。
<br/>
<br/>        “那可不，你也不看看我们的子昂有多美！”党桂芳比周宏生还嘚瑟。
<br/>
<br/>        大家都喝了一些酒，但是因为陈子昂提前和周宏生沟通过，限酒，大家都是喝得恰到好处。
<br/>
<br/>        同学聚会也是圆满而快乐，只是陈子昂没有想到，此次见周宏生是最后一面。
<br/>
<br/>        而党桂芳和傅海生此后的人生也发生了改变。
