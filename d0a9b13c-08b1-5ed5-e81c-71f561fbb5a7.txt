　　第654章
<br/>　　目前李行哉大概有一万兵力，这等兵力在帝国各路诸侯中，不算多不过也不算少。
<br/>　　兵员多是由漕帮召集而来，大多是码头上搬运货物的苦力。日子过得不如何，但江南的好风光却给了他们结实的身板。而在李行哉搜刮海盗后，也获得了充足的财物，靠着这些财物，他购置了大批精良兵器。
<br/>　　文有楚云生，武有戚继光。戚继光其实也能按半个文官算，他本身武艺并不如何，但练兵的能力实在有些恐怖。在他的训练或者说摧残下，本来没什么战斗意志的江南兵，竟有几分言听计从，视死如归的气质。
<br/>　　有李行哉这样身先士卒的指挥官，有楚云生这样算来算去的谋士，有充足财物带来的精良兵器。虽一万人但万众一心，如一杆长矛狠狠捅穿了刘顶天的防御。
<br/>　　他们没有形成有效的阻拦，便被李行哉突破沙滩，继而突破城防，由登陆战转变成巷战，战场由海滩转移到城中。
<br/>　　在刘顶天的一番祸害下，琅琊城已无多少活人。富家还能靠财宝保命，穷人也就只能用天灵盖硬抗。李行哉带人闯进来，入目是一片疮痍。
<br/>　　他已杀红了眼，手提着霸王枪一路厮杀，莫看跛了一只脚，竟然无人能挡他的路。关键也是他身后跟着上百亲卫队，而现在刘顶天已经无法组织起上百人的战斗力。
<br/>　　一路所向披靡，李行哉已占领了琅琊城的主要街道，剩下一波波小股匪贼已经不足为患。
<br/>　　“王上，王上……”
<br/>　　楚云生也一瘸一瘸在兵丁的护送下过来，见到李行哉无事，他方才长松口气。
<br/>　　“王上无事就好，无事就好。”
<br/>　　李行哉明显已经杀红了眼，手提霸王枪口中吼道：“刘顶天在哪里，他在哪里？”
<br/>　　戚继光也着急忙慌赶过来，道：“禀王上，敌军大部已经被歼灭，刘顶天带着残部藏在城主府。”
<br/>　　“走，去城主府，我非亲手宰了此贼。”
<br/>　　这场胜利来得太过顺利，但却理所应当。刘顶天攻城时是饿极的狼，但这几天在琅琊城胡作非为，他们已经变成了吃饱的猪。战斗力急转直下，在整场战斗中，几乎没对李行哉造成威胁。
<br/>　　李行哉已经率领队伍来到城主府前，命令士兵将城主府团团围住。无论在任何一座城池，城主府都是最坚固的存在。前厅后院，一城之主居住在这里处理政务。当然，如今刘顶天已经占领这里，前任城主被剥皮填草挂在城头。
<br/>　　士兵将城主府围住，正要寻一破绽攻进去。李行哉喘着粗气，于他而言，这种亲上战场杀敌的机会并不多，今日一场血战，也将他骨子里的血性释放出来。
<br/>　　“王上，你看……”戚继光指着城主府大门。
<br/>　　李行哉看清楚了，大门一点点洞开，在门后跪满了人。大门刚刚打开，一群匪贼便挥刀斩下，滚滚人头而落。
<br/>　　李行哉红了眼，手中握紧枪杆，恨不得立刻冲进去，将刘顶天宰了。
<br/>　　当然，刘顶天连面都没有露，说不得此刻在何处窝着。这幅场面，李行哉也并非没有遇见过。当初，跟着程大雷混得时候，敌人以平民掩护攻城是敌人经常做的事。
<br/>　　当初程大雷倒也狠心，管你平民不平民，一杀了之。不过，当初跟着程大雷混是一回事，自己主持一场战斗是另外一回事。一个念头，便是无数平民的人头落地，李行哉却下不去这个狠心。
<br/>　　倒也不知道，程大雷是怎样磨练出来的铁石心肠。
<br/>　　“王上，下令进攻吧。”戚继光意气风发，一军之将的存在总是喜欢胜利的。
<br/>　　李行哉面沉似水，却闭口不言。
<br/>　　“王上……”戚继光再次催促。
<br/>　　李行哉终于做出决定，开口道：“围住这里，派命清除城中匪……”
<br/>　　咚！
<br/>　　戚继光忽然一惊，见李行哉身子晃了三晃，一头栽倒在地。
<br/>　　楚云生拎着半截砖头出现在戚继光身后，道：“看我干什么，进攻呐。”
<br/>　　戚继光大手一挥，口中道：“进攻。”
<br/>　　大军从队伍中一拥而入，在城主府内大杀特杀，无论平民还是匪贼都成了刀下之鬼。
<br/>　　等李行哉醒过来时，发现自己躺在一张大床上，房屋打扫得很干净，但空气中还是有些血腥味。
<br/>　　李行哉坐起来，感觉脑袋还有些疼，他问：“谁动得手？”
<br/>　　楚云生讪讪端过来一杯热茶：“也是情急处理，王上，你没事吧？”
<br/>　　李行哉白了他一眼，也是无可奈何，道：“后来怎样了？”
<br/>　　后来的事情也没什么出奇，大军攻入城主府，刘顶天在翻墙逃离时被堵住，直接被乱刃分尸。城中的小股匪贼，也被逐个击破，没搞出怎样的事情。
<br/>　　到现在，轰轰烈烈的刘贼起事，已经被李行哉剿灭。楚云生已经代替李行哉发了安民令，后续的事情正在处理。
<br/>　　“王上，书院派人来了，求见王上。”
<br/>　　“他们竟然有脸来，不见。”李行哉冷哼一声，书院在这次琅琊劫中，表现实在摆不上台面。
<br/>　　刘顶天虽然占领琅琊城，但从始至终没能攻下书院的万卷山。当然，书院也没有出动人手对抗，他们明哲保身，看琅琊城成杀戮场做壁上关。
<br/>　　所以李行哉对他们十分不屑，满口忠孝礼义，一肚子男盗女娼。
<br/>　　“王上，还是见一见吧，毕竟是天下读书人的圣地，如今王上正在用人之际。”
<br/>　　李行哉无可奈何的叹口气，正要起身见一见来人，忽然一激灵，吓出一身冷汗。如今自己手下有了兵，文武都有人手，但同时，自己也在杀不想杀的人，做不想做的事，越来越如当年的傀儡。
<br/>　　如果都是做傀儡，何苦奔波折腾到现在，难道就是为了给自己打造一副更大的枷锁。
<br/>　　楚云生说得或许是对的，但并不是李行哉想要的。
<br/>　　他深吸一口气，将抬起的手放下，面上带出一抹冷笑。
<br/>　　“见还是要见的，通知他们，我将上万卷山，让书院子弟跪迎。”
<br/>(本章完)
