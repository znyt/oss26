<br/>　　这先天玉如意，便是一件不折不扣的天级灵器，甚至是在无数天级灵器中，都足以称得上是赫赫有名的。
<br/>　　如意君曾经纵横九域，所凭借的便是这一方先天玉如意。

<br/>　　看到夜天君拿出先天玉如意，楚风眠心中暗自想到。
<br/>　　夜天君得到如意子的洞府，这先天玉如意在他手中，也不奇怪。

<br/>　　如此一来，绝对是一个极为棘手的手段。
<br/>　　先天玉如意的攻击能力虽然不强，但是防御力，绝对是在天级灵器之中都排名前列的。
<br/>　　楚风眠想要打破，都并不容易。
<br/>　　在击败夜天君之前，必须要先对付这先天玉如意才行。

<br/>　　楚风眠一眼便是观察到，在这先天玉如意上面表现出来的力量，还是极为纯净的力量，没有任何一丝的混乱之力。
<br/>　　这夜天君身为混乱神体，一切被他炼化过的灵器，应该都会拥有一种混乱之力才对，但是这先天玉如意上面，却是丝毫没有。
<br/>　　楚风眠观察了一下，便是发现了其中的问题所在。
<br/>　　先天玉如意，虽然强大，但是却是一件至纯至清的灵器。
<br/>　　而夜天君的混乱之力，正好是跟这先天玉如意力量相撞。

<br/>　　与其是打破先天玉如意的防御，倒不如说直接将这先天玉如意夺走，最为简单。

<br/>　　先天玉如意在手，夜天君脸上的气焰便是更加嚣张起来。
<br/>　　“破不开？”
<br/>　　楚风眠嘴角露出几分讥讽笑容道。
