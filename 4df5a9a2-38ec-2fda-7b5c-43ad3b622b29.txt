　　第700章
<br/>　　“大当家……”高飞虎长身而起，双手抱拳。
<br/>　　“坐，坐。”程大雷摆摆手打断他的话：“今日不谈尊卑，只论兄弟。”
<br/>　　程大雷边饮边说，渐渐酒意上涌，嘴越来越碎：“今日在场的都是自家弟兄，也没有外人。记得我在青牛山刚起事时，只不过五个人……”
<br/>　　程大雷想起林少羽，心情郁郁，又咕咚咚罐进去一碗酒：“那时候真是吃了上顿没下顿，除了担忧官兵打过来，还得担心能不能填饱肚子……从幽州到长安，从长安到凉州，一开始也不容易，兄弟们都是将脑袋拴在裤腰带上过活，说不得那天死在别人刀下，这辈子就算交待了。”
<br/>　　“是好不容易熬到今天，今时不同往日，在座的诸位，少说手下也有几千号人，杀个人连眼皮都不用眨一下。咱们再也不用担心吃不饱，不仅吃得饱，换要吃得好。”
<br/>　　“说是把个人的官职好好分一下会更好些，但我总是想着，大家都是兄弟，一个锅里捞饭吃，没必要分得那么清楚。”程大雷说着说着突然骂了一句：“可兄弟这两个字往往是最靠不住的。”
<br/>　　在场诸多人同时噤若寒蝉，抬起头望着程大雷。
<br/>　　程大雷从椅子上站起来，手扶着椅背，道：“家大业大，兄弟们都有了自己的小算盘，我这个人没什么大本事，真真就不是个做大事的人。说真的，我已经很累了，这蛤蟆寨的第一把交椅，我是实在不愿坐了，诸位有谁想坐的没有？”
<br/>　　在场几人同时站起，几乎异口同声道：“大当家，不可。”
<br/>　　程大雷摇摇头，满脸绯红，身体踉跄，但眼神却是澄清。
<br/>　　“今日我不是要玩什么杯酒释兵权，也并非要试探兄弟们的心意，用不着，我也不屑玩弄这些把戏。我是真心想问一句，这第一把交椅有没有人能做？”
<br/>　　他目光一张脸一张脸扫过，首先看向刘悲：“刘悲……”
<br/>　　紧跟着程大雷又摇摇头：“你不行，你志不在此，秦蛮等人也不会服你。”
<br/>　　刘悲端起一碗酒一饮而尽。
<br/>　　“关将军……你虽有帅才，却也不是个能服众的人物，这第一把交椅你做不了。”
<br/>　　关鱼低头抿酒，却也没有表示不服气。
<br/>　　“和总管……你心思通透，八面玲珑，是只老狐狸，但做不了山大王。”
<br/>　　程大雷目光一一扫过，一一点评诸多人。在场诸多人中，人人都是出奇之处，可若说谁能稳坐军帐中，压得住八方，却也只是程大雷了。
<br/>　　“徐军师，你……”
<br/>　　最后，程大雷的目光落在徐神机身上，徐神机脸泛红晕，当场就要站起来。
<br/>　　“你……”程大雷顿了顿：“好好喝酒，今天饭菜的味道还是不错的。”
<br/>　　抬起头，目光望着面前众人，除了徐神机外，其他几人都已站起，抱拳拱手，异口同声道：“大当家，属下知错。”
<br/>　　“错，又有什么错。”程大雷重新歪坐在椅子上，目眺远方：“若你们有能耐坐稳这个位置，我便放心去了。”
<br/>　　无论如何，诸人已经被绑在一起，自家有自家的利益，各自有各自的算盘。但程大雷的话至少点明白一件事，大家都在一艘船上，首先要保证蛤蟆寨这艘船不翻。而在现在，程大雷的确是唯一的掌舵手，目前没有任何人可以取代他。
<br/>　　程大雷似醉非醉，睁着双眼道：“明正赏罚，一断以律，从今日起，若是谁有律不准，一概严惩。”
<br/>　　“属下遵命。”
<br/>　　程大雷端起酒碗，缓缓饮尽，然后将碗底亮给诸人看。
<br/>　　“诸位，我不希望来日，我与诸位在刑场上饮端头酒。”
<br/>　　“是。”
<br/>　　程大雷无力的挥挥手，道：“退下吧，退下吧。”
<br/>　　刘悲还欲在说什么，却见程大雷歪在椅子上，已经响起轻轻鼾声。
<br/>　　诸人散去，一一走出城主府的大门，却是谁都没有牵马或者坐轿，只不过三五成群。
<br/>　　看个人离去时的样子，便可以看到谁与谁更亲近些。刘关张赵走在一起，秦蛮徐神机高飞虎高飞豹同行，吴用在武将那里讨不到好处，随着和珅一起走。
<br/>　　城主府的高楼上，程大雷外靠在栏杆上，静静看着这一幕。
<br/>　　苏樱端着浓茶在其身后，轻轻敲了敲柱子，道：“你喝杯茶醒醒酒吧。”
<br/>　　程大雷接过茶碗，将浮沫吹去，静静道：“你说，我这算不算也是耍心眼？”
<br/>　　“装神弄鬼，荒诞可笑，却句句说得是实话。你说他们能听进去么？”
<br/>　　程大雷摇摇头，又点点头：“听不听得进去不重要，只是蛤蟆寨当真没到分家的时候，分就是散，散了就什么都没了。各人有各人的小心思，也不能要求所有人与我心思相同，那其实也是一件很可怕的事情。只不过，大家有共同做事的想法，不要搞得太过分，便也就可以了。”
<br/>　　“明定赏罚，该赏则赏，该罚则罚，我今天也就是给他们敲敲边鼓，真的把事情搞起来，我可不想在死几个孙小毛了。”
<br/>　　苏樱安静听着，道：“我现在多少知道，你有多累了。”
<br/>　　程大雷揉揉她的脑袋，道：“走，下去吧，风怪大的，别吹坏了身子。”
<br/>　　苏樱白了他一眼，道：“楼上的风再大，也终究没你见的风大。”
<br/>　　程大雷将手伸到栏杆外，凉风从指间绕过，他点点头：“秋天快来了，风的确蛮大的。”
<br/>　　苏樱搀住程大雷，二人相扶往楼下走。
<br/>　　徐神机一行人走在一起，高飞豹摸摸脑袋，道：“徐军师，秦大哥，大当家今夜究竟什么意思？”
<br/>　　徐神机没有说话，反倒是秦蛮冷哼一声，道：“定是你贪得无厌，城破之后捞得太狠，才令大当家如此恼火。”
<br/>　　高飞豹吐吐舌头，感觉脖颈后面有些发亮。高飞虎脸上有些发烧：“也没有捞得太狠么，都是大当家赏给我们的，大当家不给的，我们可不敢要。”
<br/>　　“你自己不要，你手下的山贼呢，他们是闲得住么。”
<br/>　　“姓秦的你什么意思，大家都是一样出身，你看不起谁。”
<br/>(本章完)
