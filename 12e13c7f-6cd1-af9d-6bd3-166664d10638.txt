<br/><br/>　　    在今天午赵凯召开了新闻发布会，将悬赏的金额提高到一千万后不久，赵凯在悬赏公告所公布的电话终于接到了一个有价值的电话，之所以说这个电话有价值，是因为这个电话直接显示的是用傅华的手机号码。<br/>　　<br/>　　    打来电话的人是一个声音沙哑的男人，他只是简单的说了一句话：“把一千万在下午四点钟前准备好，要现金，不连号的旧钞，四点钟我会通知你们到什么地方来换傅华的。这段时间我会监视着你们的，如果被我知道你们报警了的话，傅华死定了。”<br/>　　<br/>　　    男人说完，挂了电话。赵凯看到这个人用的是傅华的手机号码，傅华的手机在傅华被绑架之后去向不明，现在看来很可能是被绑匪带走了。于是赵凯知道这个打电话的人肯定是与傅华被绑架有着某种联系的，连犹豫都没犹豫马动用了他在银行系统一个多年的关系，紧急从从银行调集了一千万元的不连号旧钞备用。<br/>　　<br/>　　    四点钟，悬赏公告公布的电话再次响起，这一次那个男人用的是另外一部电话，他在电话里让赵凯带人来房山区的刘庄村，他会在村边的公路等着赵凯他们。<br/>　　<br/>　　    赵凯放下电话之后，马要带着人去房山区的刘庄村，本来他是不想让赵婷跟着的，但是说什么也要跟着过来，于是赵凯只好带着她一起去了房山区。赵凯他们到了在房山区刘庄村村边的公路时候，一个男人开着一部皮卡已经等在那里了。<br/>　　<br/>　　    男人戴着墨镜和口罩，赵凯他们根本看不到他的真实的面目，他看到赵凯和他的人来了之后，按了汽车喇叭示意赵凯他们过去，让赵凯的人把一千万搬了皮卡，然后跟赵凯说：“我先走，十分钟之后会有短信通知你傅华在哪里了。”<br/>　　<br/>　　    在这个时候，赵凯、赵婷为了傅华的安危，是宁舍一千万，也不想看到傅华出事的，因此他们虽然心有疑虑，但却也不敢对这个男人怎么样的，于是让男人带着这一千万离开了。好在十分钟之后，短信真的过来了，短信指示赵凯说，傅华在这座山的一个防空洞里。<br/>　　<br/>　　    于是赵凯按照短信的指示来到了这个防空洞前，北京的防空洞是当年珍宝岛战役发生之后，当时的政府为了防止国外敌对势力的空袭，而在北京各地所挖的一座地下城，北京的很多地方都是有这种防空洞，一些繁华地带的防空洞现在大多被利用了起来，成为了地下商场之类的，而一些较偏僻的地方则是无人问津，被废弃了。<br/>　　<br/>　　    绑匪可能是考虑到防空洞隐蔽，一般人很少会来，所以才会利用这里作为拘禁傅华和罗茜男的场所。赵婷听赵凯说这是那个短信指示的山洞，二话没说，打开车门跳下车往山洞里面冲，赵凯和通汇集团的保安紧随其后。<br/>　　<br/>　　    在山洞，一行人很快找到了一个被紧锁的铁门，随行的保安们用带进来防身的铁棍将门撬开了，在山洞，他们发现了已经倒在地的的傅华和罗茜男，两人已经陷入深度昏迷，叫也叫不醒。于是，保安将两人抬了车，将两人送进了最近的医院当。<br/>　　<br/>　　    经过医生诊断，罗茜男和傅华是因为极度脱水造成电解质紊乱从而导致的深度昏迷，首先需要进行大量的补液以纠正电解质紊乱。<br/>　　<br/>　　    赵婷看傅华和罗茜男被医生采取了一些治疗措施之后，依然没有马醒过来的意思，追问医生说：“医生，为什么他们还不醒啊？”<br/>　　<br/>　　    医生说：“他们是深度昏迷，不可能一采取救治措施能清醒过来的。”<br/>　　<br/>　　    赵婷说：“那他们什么时候究竟能够清醒过来啊？”<br/>　　<br/>　　    医生说：“这个不好说，这样看病人的身体状况和究竟昏迷了多长时间。”<br/>　　<br/>　　    赵婷有些急了，问道：“什么叫做不好说啊，难道说他们还有可能醒不过来吗？”<br/>　　<br/>　　    医生说：“是有这种可能的，如果病人昏迷的时间太长，还是真的有可能醒不过来的。男患者的症状还好一点，他只是严重脱水，那位女患者的身体状况很糟糕了，从她的症状看，她除了严重脱水之外，应该还有过一定程度的失血，身体状况极差，能不能从昏迷当醒过来更不好说了。”<br/>　　<br/>　　    赵婷着急的看了看一旁的赵凯，说：“爸，你说怎么办啊？傅华他有可能醒不过来了。”<br/>　　<br/>　　    赵凯安慰赵婷说：“小婷，你先别急，一定有什么办法能让傅华醒过来的。”<br/>　　<br/>　　    赵凯说完转身对医生说道：“医生，有没有什么针对这种情况的特效药啊，如果有的话不论多少钱我们都可以付的。”<br/>　　<br/>　　    医生苦笑了一下说：“这不是钱的问题了，目前来说他们这种状况只能是采用大量的补液这一针对性的治疗措施了，没有什么特效药能让这两个人马从昏迷醒过来的。”<br/>　　<br/>　　    赵凯看了看这个医院相对较简陋的设备，有心将傅华和罗茜男转院，他说：“那他们这个情况可不可以转院啊？转到市区里面的大医院去治疗会不会好一点？”<br/>　　<br/>　　    医生说：“他们两人的状态基本稳定了下来，转院倒是可以的。大医院的治疗条件也更好一些，对两人的治疗也更有利。”<br/>　　<br/>　　    于是赵凯和赵婷给傅华和罗茜男办了转院手续，把两人转入了市区一家著名的大医院当治疗。在赵凯赵婷的强烈要求下，这家大医院的医生马对傅华和罗茜男进行了专家会诊。<br/>　　<br/>　　    专家会诊的结果基本跟前面那家医院的诊断是一致的，不过他们对傅华和罗茜男的能不能够苏醒相对较乐观一点。专家们说：“傅华和罗茜男较幸运的是被关在防空洞里的。”<br/>　　<br/>　　    按照常规来说，一个人在没有食物和水的环境下是很难存活五天那么长时间的，专家们认为防空洞相对低温潮湿的环境一定程度降低了两人身体对水分的需求，这才让两人存活了那么长的时间。同时低温环境也在一定程度减少了电解质紊乱给两人身体造成的损害。”<br/>　　<br/>　　    专家们的这个诊断让赵凯赵婷多少宽心了一点。赵凯将傅华和罗茜男被救回来的情况通知了警方。万博和专案小组的钱组长马带着人赶了过来。<br/>　　<br/>　　    在看过了傅华和罗茜男的身体状况之后，万博和钱组长把赵凯从房间叫了出去，万博看着赵凯说：“赵董啊，您能跟我们警方说说解救傅华和罗茜男的过程吗？我们需要根据这个情况找到绑匪的下落。”<br/>　　<br/>　　    赵凯看了一眼万博，笑了笑说：“万队，钱组长，这段时间你们警方为了侦破傅华被绑架的这个案子做出了极大的努力，都很辛苦，这个我们赵家是十分感激的，以后你们如果有什么地方用到我赵凯和通汇集团的，我们一定会尽力的。不过解救傅华和罗茜男的经过，你们还是不要问了。我想要的是傅华回来，现在傅华回来了，这够了。”<br/>　　<br/>　　    万博说：“赵董，我们警方为了侦破案件所做的那些工作都是应该做的，这个倒不需要您给我们什么回报，不过您如果不提供解救傅华和罗茜男具体情况的话，我们警方没什么依据来破案了，所以您看……”<br/>　　<br/>　　    赵凯说：“万队，你不要再说了，我对那个提供线索的人是有承诺的，承诺过不会去追查他的。作为一个商人，无信不立，做出的承诺必须要遵守，所以抱歉了，我不能跟你们讲他的情况。”<br/>　　<br/>　　    钱组长说：“赵董啊，这个不好吧？根据我们警方的监控，这个提供线索的人第一次跟你们联系，使用的是傅华的手机，因此我们怀疑他很可能是绑匪其的一个。”<br/>　　<br/>　　    万博说：“当时我们也考虑过要监控你们这一次的解救行动的，但是又怕打草惊蛇，影响到对傅华的解救，所以没有采取进一步的行动。现在傅华已经救回来了，我们警方对此也没什么顾忌了，因此赵董啊，请您把解救的情形跟我们详细讲一下，我们也好侦破此案，把绑匪绳之于法。”<br/>　　<br/>　　    赵凯看了看万博，笑了笑说：“万队，对你们警方能够这么体谅我们这些家人的处境，让我们能够顺利地解救了傅华，对此我再次表示感谢。不过，你们既然监控到了绑匪的电话，那你们可根据这个电话查到了他们的下落的啊。”<br/>　　<br/>　　    钱组长摇了摇头，说：“不行的，这个绑匪很狡猾，他跟你们通话的时间都很短，我们的侦讯设备还来不及查到它们的具体位置。”<br/>　　<br/>　　    赵凯说：“钱组长，我倒不是对你们警方工作有什么意见啊，假设说我把解救傅华的整个过程都提供给你们警方了，你能够确保一定会将这一伙绑匪都抓起来，没有一个漏之鱼吗？”<br/>　　最快更新，无弹窗阅读请。<br/><br/><br/>
