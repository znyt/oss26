　　仔细研读《太极玄清道》第一层，苏重很快沉浸其中。
<br/>　　和宋大仁教导内容对照，他对功法内容了解越发透彻。
<br/>　　“不愧是修仙功法，第一步就是引起入体，利用天地力量增强自身。相比之下内功修炼还需要水谷之精，效率质量都差了很多。”苏重对比之前接触内功，明白其中差异。
<br/>　　把第一层彻底搞明白后，苏重收起秘籍，起身离开房间。
<br/>　　越是修炼高深功法，就越需要心平气和。他骤然得到高深秘法，心情激荡实在不是修炼的好时候。
<br/>　　苏重走进厨房，找出食材准备给自己做顿饭庆祝一下。
<br/>　　他切菜的时候，全神贯注，不管是太极玄清道还是其他杂七杂八的事情，都被他通通摒弃，眼中只有手中菜刀和案板上的食材。
<br/>　　仔细感受食材被切开的状态，仔细分辨炒制时食材散发的味道。将自己精神全部集中，很快他就做出一桌美味饭菜。
<br/>　　抱出桌椅放在厨房外天井内，苏重慢条斯理开始吃饭。
<br/>　　刚才一通忙活，他的心绪已经完全平静下来。
<br/>　　“二蛋师弟！你竟然在偷吃！”一个古灵精怪丫头冲进厨房，看到苏重在吃东西顿时大呼小叫起来。
<br/>　　他身后还有一个小跟班，正是张小凡。
<br/>　　“想吃就一块吃吧。”苏重笑道，起身进屋拿了两双碗筷。
<br/>　　田灵儿和张小凡早就被香气勾出馋虫，拿过筷子也不客气，直接开吃。
<br/>　　苏重反而吃的不多，只是夹了几下菜就停住不吃。笑呵呵看着张小凡和田灵儿两人抢饭吃。
<br/>　　“二蛋师弟，你做的饭太好吃啦！以后你就来给大家做法吧！”田灵儿眼中闪烁光芒。
<br/>　　想到以后每天都能吃到这么美味的食物，他就兴奋的不能自已。
<br/>　　“好啊，到时候小凡你来帮忙，我把这手厨艺交给你。”苏重欣然答应。
<br/>　　大竹峰有个不成文习俗，最小的弟子负责做饭。苏重对此并不排斥，做饭而已，花不了他多长时间。
<br/>　　而且还有张小凡，等教会他做饭，苏重就会更清闲。
<br/>　　打发走两人，苏重慢悠悠回到房间。到这会儿，他已经完全平静起来。
<br/>　　再次拿出太极玄清道看了一遍，苏重将需要冥想的几个符文牢牢记住。
<br/>　　然后闭上眼睛开始修炼。
<br/>　　他穿越多个世界，修炼过无数功法。太极玄清道固然神奇，但苏重明白起本质后并不难上手。
<br/>　　调整呼吸，观想符文，整个人进入一种特殊状态。好似化作一个空洞，周围天地内特殊活性能量开始像他聚集。
<br/>　　很快，一缕头发丝状灵气，顺着皮肤毛孔钻如体内。
<br/>　　一股温凉质感刺激身心：“就是它！当初田不易探查身体伤势，用的就是这种能量！”
<br/>　　知道练对了路子，苏重压下心绪，继续维持在特殊状态，牵引灵气进入身体。
<br/>　　但是很快，苏重就睁开眼睛，眉头仅仅皱起。
<br/>　　“灵气在体内循环一圈，很快就溢出散开，这和秘籍记载的不一样。”
<br/>　　按照秘籍讲解，吸入体内的灵气虽会消散，但总会留下一部分。
<br/>　　把这部分灵气存储在丹田，然后慢慢搬运炼化。
<br/>　　在这个过程中，灵气、身体、精神三者相互熏染。最终灵气变成自己的一部分，身体和精神得到某种强化，这就是太极玄清道。
<br/>　　经过不断强化，精神逐渐发生质变，然后就能够离开身体控制外物。这时候就可以修炼驾驭飞剑法宝的御物术。
<br/>　　“身体留不住灵气，那还如何修炼？”苏重有些傻眼。
<br/>　　“人家张小凡是因为修炼大梵般若，所以才会灵气流失，自己根本就存不住灵气，那还怎么修炼？”苏重眉头紧紧周期。
<br/>　　苏重不信邪，再次修炼了一遍。
<br/>　　努力了整整一个下午，苏重一无所获。不过他也找到了原因所在。
<br/>　　“这具身体天资本来就不好，现在又被我夺舍。无法完全控制全身。行走坐卧还没什么，精准控制全身毛孔就不太行了。”
<br/>　　太极玄清道里面特有法门，只要观想特定符文，就能闭合全身毛孔留住灵气。可惜，不管苏重如何尝试，他都无法关闭全身毛孔。
<br/>　　灵气自然就跑没影了。
<br/>　　第二天，苏重找到宋大仁，将自己情况说出。宋大仁也没了主意。这么多年修炼下来，顶多有人修炼的慢一些，还从来没出现过这种无法修炼的情况。
<br/>　　“走，你跟我去见师傅。”宋大仁抓住苏重，一个跳跃就飞上天。
<br/>　　这还是苏重第一次被人带着飞天，感受着宋大仁体内磅礴灵气，苏重着实羡慕的很。
<br/>　　“师傅，小师弟修行除了岔子，他无法截流灵气，您赶紧给看看。”宋大仁焦急道。
<br/>　　田不易脸色一变，立刻开始查看苏重身体。
<br/>　　几次探查，田不易脸色难看。
<br/>　　“可能是因为上次受伤太重，破坏了根基……”他有点儿说不下去。
<br/>　　苏重身体确实无法截流灵气，没法截流灵气就不能修炼玄清道。
<br/>　　明明已经拜入山门，却不能修炼。这在田不易看来太残酷了。
<br/>　　“一点儿希望都没有吗。”苏重皱着眉头，虽然不太高兴，却也没怎么沮丧。比这还倒霉的穿越他都经历过。
<br/>　　“尔旦，稍后我去通天峰问问掌门师兄，看他有没有办法，不要着急。”田不易温声安慰。
<br/>　　苏重心里感激，田不易作为师傅还是很靠谱的。自己不过是个新进弟子，他都能这么上心。
<br/>　　不过苏重有自己的打算：“师傅，我这种情况这么罕见，掌门那里又不一定有什么办法。您就别去了。”
<br/>　　别人不知道，他自己清楚。根本不是什么伤势过重导致。
<br/>　　原因在于他是个外来户，和身体无法百分百契合。哪里有什么方法能治疗。这可是穿越后遗症。
<br/>　　而且即便治好了又能怎样。昨天修炼他已经有了感觉。自己这身体的天赋实在不怎么样。
<br/>　　“师傅，能让我去藏书楼看看吗，既然不能修炼玄清道，修炼些寻常武功也好，起码能强身健体不是。”苏重笑道。
<br/>　　田不易见苏重这么快就调整好状态，心里满是诧异，仔细观察，发现苏重是真的调整好了心态，心里欣慰至于更加惋惜。
<br/>　　这种心性最适合修炼，只要稳扎稳打厚积薄发，将来一定能大有所为。
<br/>　　可惜啊！
<br/>　　“你能想的开我很高兴。稍后就让大人带你藏书楼。你也不要气馁，我去通天峰看看。掌门师兄见多识广，不一定就没有解决办法。”田不易道。
<br/>　　“多谢师傅！”苏重道谢。
<br/>　　不过他的心思已经跑到了藏书楼。
<br/>　　玄清道不能修炼，那就练别的吗。他又不是第一次遇到这种情况，大不了就自己创造功法。
<br/>(本章完)
